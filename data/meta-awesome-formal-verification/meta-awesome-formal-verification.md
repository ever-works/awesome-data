## Description
meta-awesome: Formal Verification is a curated "meta-awesome" entry that aggregates multiple awesome lists focused on:
- Formal verification
- Theorem proving
- Correctness-by-construction tools and workflows

It serves as a central index to help developers, researchers, and engineers discover specialized awesome lists and resources in the formal methods ecosystem.

## Features
- **Meta-aggregation of awesome lists**
  - Collects and links to multiple awesome lists related to formal verification and theorem proving.
  - Acts as a directory-of-directories for formal methods resources.

- **Coverage of formal verification topics**
  - Formal verification techniques and tools.
  - Theorem provers and proof assistants.
  - Correctness-by-construction and related methodologies.

- **Programming-focused curation**
  - Targets resources relevant to programming, software correctness, and verification tooling.

- **GitHub-hosted index**
  - Accessible as a Markdown document within the `meta-awesome` GitHub repository.
  - Easy to browse, fork, and contribute to via standard GitHub workflows.
