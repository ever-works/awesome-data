## Overview

Awesome shadcn/ui is the largest collection of resources for shadcn/ui, the popular React component library built with Radix UI and Tailwind CSS.

## What is shadcn/ui?

A collection of beautifully designed, accessible components that you can copy and paste into your apps. Built with Radix UI primitives and Tailwind CSS.

## Resource Categories

- **Templates** - Starter templates and boilerplates
- **Themes** - Pre-built color themes and design systems
- **Components** - Additional components and extensions
- **Blocks** - Pre-built component compositions
- **Tools** - Development and design tools
- **Integrations** - Framework and library integrations

## Popular Use Cases

- **Admin Dashboards** - Full-featured admin templates
- **Landing Pages** - Marketing and product pages
- **SaaS Applications** - B2B/B2C web applications
- **Documentation Sites** - Technical documentation
- **E-commerce** - Online stores and marketplaces

## Key Features

- Fully accessible components
- Customizable with Tailwind
- TypeScript support
- Dark mode built-in
- Copy-paste workflow
- No package dependencies
