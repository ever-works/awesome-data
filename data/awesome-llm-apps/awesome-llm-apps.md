## Overview

A curated collection of awesome LLM apps built with RAG (Retrieval-Augmented Generation), AI Agents, Multi-agent Teams, MCP (Model Context Protocol), Voice Agents, and more, exploring applications that combine LLMs from OpenAI, Anthropic, Gemini, and open-source alternatives.

## Features

### AI Agent Applications

- Single-agent systems for focused tasks
- Multi-agent teams for complex workflows
- Autonomous agent frameworks
- Agent orchestration patterns

### RAG Applications

- Document Q&A systems
- Knowledge base integrations
- Semantic search implementations
- Hybrid search approaches
- Multimodal RAG (text, images, audio, video)

### MCP Integration

- Model Context Protocol implementations
- Tool-calling and function execution
- External API integrations
- Data source connections

### Voice and Multimodal

- Voice-enabled assistants
- Speech-to-text integration
- Text-to-speech applications
- Image understanding and generation

## Supported Models

- OpenAI GPT-4, GPT-4o, o1
- Anthropic Claude 3.5 Sonnet, Opus
- Google Gemini Pro, Ultra
- xAI Grok
- Open-source: Llama, Qwen, Mistral

## Application Categories

- Customer support chatbots
- Code assistants
- Research tools
- Content generation
- Data analysis
- Personal productivity
