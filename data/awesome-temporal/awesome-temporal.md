## Overview

Awesome Temporal provides resources for Temporal, the open-source platform for orchestrating durable workflows.

## What is Temporal?

Temporal is a scalable and reliable runtime for durable workflow execution, ensuring that workflows continue to completion even if failures occur.

## Key Features

- **Durable Execution** - Workflows survive failures
- **Reliable** - Built-in retries and error handling
- **Scalable** - Handles millions of workflows
- **Version Safe** - Deploy code without breaking running workflows
- **Observable** - Built-in visibility and monitoring
