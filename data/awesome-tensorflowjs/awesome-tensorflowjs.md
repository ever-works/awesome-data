# Awesome TensorFlow.js

**URL:** https://github.com/aaronhma/awesome-tensorflow-js#readme  
**Category:** Themed directories  
**Tags:** JavaScript, Machine Learning, Web

## Overview
Awesome TensorFlow.js is a curated directory of learning materials, tools, and community resources for working with TensorFlow.js, the JavaScript library for training and running machine learning models in the browser or in Node.js.

## Features

### Learning Resources
- **Documentation**
  - Official TensorFlow.js guide
  - TensorFlow.js API reference

- **Tutorials & Courses**
  - TensorFlow.js gallery of projects, tutorials, videos, and more
  - Google Machine Learning Crash Course
  - Google Codelabs hands-on tutorials
  - “Beginning Machine Learning with TensorFlow.js” online course for web developers
  - “Effective TensorFlow.js” tutorials and best practices

- **Models & Example Projects**
  - Official TensorFlow.js models repository with pretrained models
  - TensorFlow.js models catalog on tensorflow.org
  - `@tensorflow-models` packages on npm
  - Official TensorFlow.js examples repository
  - Official TensorFlow.js demos site with live examples

- **Research Papers**
  - *TensorFlow.js: Machine Learning for the Web and Beyond* (design, API, implementation, and use cases)
  - *JSDoop and TensorFlow.js: Volunteer Distributed Web Browser-Based Neural Network Training* (distributed browser-based training feasibility and scalability)

- **Videos**
  - Official TensorFlow.js YouTube video series focused on TensorFlow ecosystem updates

### Tools
- Links to libraries and tools/utilities related to TensorFlow.js (as organized in the GitHub repository’s Tools section)

### Community
- Pointers to online communities and social media channels relevant to TensorFlow.js (as organized in the GitHub repository’s Community section)
