## Overview
meta-awesome: Games is a curated meta-list that aggregates other “awesome” lists focused on games. It emphasizes collections about games themselves, including open-source titles, catalogs of games, and playful or experimental game-related projects.

## Features
- Aggregates multiple “awesome” lists related specifically to games.
- Focuses on lists about:
  - Open-source games.
  - Game catalogs and directories.
  - Playful or experimental game projects.
- Serves as a central reference point to discover game-related awesome lists.
- Hosted on GitHub as part of the broader `meta-awesome` collection.
