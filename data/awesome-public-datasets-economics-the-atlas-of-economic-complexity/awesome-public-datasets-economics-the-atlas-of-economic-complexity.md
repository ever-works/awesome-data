# Awesome Public Datasets – Economics: The Atlas of Economic Complexity

**Category:** Themed Directories  
**Brand:** awesome-public-datasets  
**Directory Type:** Curated dataset listing (economics / trade)

## Overview
Awesome Public Datasets – Economics: The Atlas of Economic Complexity is a curated catalog entry in the Awesome Public Datasets collection that points to Harvard’s Atlas of Economic Complexity. The referenced resource is an interactive platform and data repository for analyzing international trade flows and economic complexity indicators.

## Features
- Links to Harvard’s Atlas of Economic Complexity dataset and tools.
- Focus on **international trade flows** data between countries.
- Access to **economic complexity indicators**, such as measures of how advanced or diversified an economy’s exports are.
- Part of the **Economics** section of the Awesome Public Datasets collection, making it easier to discover alongside related economic data resources.
- Provides a structured YAML metadata file within the `apd-core` repository for programmatic inclusion in other tools or directory UIs.

## Data / Content Focus
- Macroeconomic and trade-related datasets.
- Country-level export and import structures.
- Indicators used to assess the complexity and sophistication of national economies.

## Access
- Primary reference: GitHub Awesome Public Datasets entry.  
- Underlying data and interactive visualizations are provided by Harvard’s Atlas of Economic Complexity (accessed via the linked resource in the catalog entry).
