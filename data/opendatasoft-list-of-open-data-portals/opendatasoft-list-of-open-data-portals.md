# OpenDataSoft list of open data portals

## Overview
The OpenDataSoft list of open data portals is a large, curated meta-directory of around 1,600 open data portals from around the world. It focuses on government and institutional open data sites, helping users discover datasets across countries, regions, and cities.

- **Type:** Meta-directory / directory-of-directories
- **Category:** Meta-directories
- **Provider / Brand:** OpenDataSoft
- **URL:** https://www.opendatasoft.com/blog/2015/11/02/how-we-put-together-a-list-of-1600-open-data-portals-around-the-world-to-help-open-data-community

## Features
- Curated list of ~1,600 open data portals worldwide
- Focus on government and institutional open data sites
- Global coverage across multiple countries and cities
- Functions as an “awesome list” style resource for open data discovery
- Useful starting point for researchers, developers, and practitioners looking for open datasets
