## Overview

Awesome Knowledge Graph provides comprehensive resources for building, managing, and querying knowledge graphs, covering both the theoretical foundations and practical implementations.

## Graph Databases

### Property Graph Databases
- **Neo4j**: Leading native graph database with Cypher query language
- **Dgraph**: Fast, distributed graph database designed for real-time query execution
- **JanusGraph**: Scalable graph database optimized for storing and querying large graphs
- **ArcadeDB**: Multi-model database supporting graph, document, key-value, and search
- **Memgraph**: In-memory graph database for real-time analytics
- **TigerGraph**: Analytics platform for deep link analytics

### RDF Triple Stores
- **Stardog**: Enterprise knowledge graph platform
- **Apache Jena**: Framework for building semantic web and linked data applications
- **Virtuoso**: High-performance object-relational SQL database
- **GraphDB**: RDF database with reasoning and analytics capabilities

### Multi-Model Databases
- **OrientDB**: Multi-model database supporting graph, document, and object models
- **ArangoDB**: Native multi-model database with graph, document, and key-value stores
