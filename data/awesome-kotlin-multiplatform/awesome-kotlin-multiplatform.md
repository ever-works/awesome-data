## Overview

Awesome Kotlin Multiplatform (KMP) is a comprehensive collection of libraries and tools for building cross-platform applications with shared Kotlin code.

## Features

- **UI Frameworks**: Compose Multiplatform for shared UI across platforms
- **Networking**: Ktor client for cross-platform HTTP
- **Database**: SQLDelight, Realm for multiplatform data persistence
- **Dependency Injection**: Koin, Kodein for KMP
- **State Management**: MVI and Redux patterns for KMP
- **Serialization**: kotlinx.serialization for cross-platform data
- **Coroutines**: Async programming across all platforms
- **Testing**: Multiplatform testing frameworks

## Supported Platforms

- Android (native)
- iOS (native via Kotlin/Native)
- Web (via Kotlin/JS)
- Desktop (JVM - Windows, macOS, Linux)
- Backend (JVM servers)
