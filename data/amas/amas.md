# AMAs

**Website:** https://github.com/sindresorhus/amas#readme  
**Category:** Themed Directories  
**Tags:** community, developer-tools, resources

## Description
AMAs (Awesome & Marvelous AMAs) is an awesome-list-style directory of developer and tech-related “Ask Me Anything” (AMA) sessions. It curates links to individuals and organizations who host AMAs, letting the community easily discover people and groups they can ask questions to.

## Features
- **Curated AMA directory**
  - Collection of developer- and tech-focused AMAs
  - Organized in an “awesome list” style repository
- **People-focused listings**
  - Section for individuals you can “ask anything”
- **Organization-focused listings**
  - Section for organizations and teams that host AMAs
- **Create-your-own AMA guide**
  - `create-ama.md` with instructions on how to set up and list your own AMA
- **Community-driven contributions**
  - `contributing.md` describing how to propose additions or changes
- **Open GitHub repository structure**
  - Public folder and file layout for transparency and collaboration
- **Awesome List compliance**
  - Labeled and formatted as an [Awesome](https://awesome.re) list
- **Open licensing**
  - Released under CC0 (public domain dedication), allowing reuse and adaptation
