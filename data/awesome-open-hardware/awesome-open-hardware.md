# Awesome Open Hardware

**URL:** https://github.com/delftopenhardware/awesome-open-hardware#readme  
**Category:** Themed directories  
**Tags:** hardware, open-source, awesome-lists

## Description
Awesome Open Hardware is a curated “awesome list” of resources for making and learning about open‑source hardware. It aggregates links to projects, educational materials, tools, and reference content relevant to open hardware development.

## Features
- Curated directory focused on **open-source hardware**
- Emphasis on **helpful items for making and learning about open hardware projects**
- Organized sections (as indicated in the README structure):
  - **Projects** – example open hardware projects
  - **Talks** – presentations and recorded talks about open hardware
  - **Papers** – academic or technical papers related to open hardware
  - **Conferences** – events and conferences covering open hardware topics
  - **Platforms** – platforms or ecosystems used to develop or share open hardware
  - **Podcasts** – audio resources on hardware and open-source topics
  - **Books** – books for deeper learning
  - **Training programs** – educational or training offerings related to hardware
  - **Further readings** – additional reference material beyond the main categories
  - **Related awesome** – links to other relevant awesome lists
- Open contribution model:
  - Public **contributing guidelines** (`contributing.md`)
  - Suggestions and additions from the community are welcomed via issues and pull requests
- Openly licensed (LICENSE file in repository)
- Maintainers listed can request removal via an issue

## Pricing
- **Free** – Public GitHub repository and resource list, available at no cost.

## Additional Notes
- Maintained by the Delft Open Hardware community (referenced at https://delftopenhardware.nl).
- Suitable as a starting point for anyone exploring or building open-source hardware projects.
