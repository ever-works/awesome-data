# FCP-INDI

The Functional Connectomes Project and International Neuroimaging Data-sharing Initiative (FCP-INDI) provides large-scale, open-access neuroimaging datasets, primarily resting-state and task-based fMRI, along with associated tools and documentation for the neuroscience community.

---

## Overview

- **Name:** FCP-INDI (Functional Connectomes Project & International Neuroimaging Data-sharing Initiative)
- **Category:** Datasets
- **Domain:** Neuroscience, medical imaging (fMRI)
- **Primary Focus:** Large-scale open-access neuroimaging datasets (resting-state and task-based fMRI)
- **Access Model:** Open data for non-commercial use (registration required for INDI datasets)
- **Website:** <http://fcon_1000.projects.nitrc.org/index.html>

---

## Features

### 1. Data & Resources

- **1000 Functional Connectomes Project**  
  Large-scale collection of resting-state fMRI datasets from multiple sites and populations.  
  - Access: [Dataset table](http://fcon_1000.projects.nitrc.org/fcpClassic/FcpTable.html)

- **INDI-Prospective**  
  Prospectively collected neuroimaging datasets shared via the International Neuroimaging Data-sharing Initiative.  
  - Access: <http://fcon_1000.projects.nitrc.org/indi/IndiPro.html>

- **INDI-Retrospective**  
  Retrospectively assembled datasets aggregated from previously collected studies.  
  - Access: <http://fcon_1000.projects.nitrc.org/indi/IndiRetro.html>

- **Preprocessed Connectome Project**  
  Preprocessed versions of FCP/INDI datasets to lower the barrier for analysis and reproducibility.  
  - Access: <http://preprocessed-connectomes-project.org/>

- **AWS S3 Resources**  
  Hosting and distribution of datasets via Amazon Web Services S3 for scalable, programmatic access and cloud-based analysis.  
  - Access: <http://fcon_1000.projects.nitrc.org/indi/s3/index.html>

- **Addiction Connectome Preprocessed Initiative (ACPI)**  
  Preprocessed neuroimaging datasets focused on addiction-related research.  
  - Access: <http://fcon_1000.projects.nitrc.org/indi/ACPI/html/index.html>

- **FCP Scripts**  
  Scripts and related resources to support data handling, preprocessing, or analysis of FCP-INDI datasets.  
  - Access: <http://www.nitrc.org/frs/?group_id=296>

- **How-Toâ€™s / Documentation**  
  Wiki-style documentation for using the datasets and associated tools.  
  - Access: <http://www.nitrc.org/plugins/mwiki/index.php/fcon_1000:MainPage>

- **Forums**  
  Community discussion boards hosted via NITRC for questions, announcements, and user interaction.  
  - Access: <http://www.nitrc.org/forum/?group_id=296>

### 2. Access & Licensing

- **Registration Requirement**  
  - Users must register with the 1000 Functional Connectomes Project website on NITRC to access INDI datasets.  
  - Registration grants full access to the datasets.

- **Usage Rights**  
  - Datasets may be used **unrestrictedly for non-commercial purposes**, subject to dataset-specific conditions.

- **Access Links**  
  - Request dataset access: <http://fcon_1000.projects.nitrc.org/indi/req_access.html>  
  - Additional information and acknowledgement details: <http://fcon_1000.projects.nitrc.org/indi/indi_ack.html>
