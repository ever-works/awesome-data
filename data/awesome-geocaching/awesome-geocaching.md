# Awesome Geocaching

## Overview
Awesome Geocaching is a curated "awesome list" of online resources related to geocaching, the GPS-based outdoor treasure-hunting activity. It aggregates tools, apps, guides, and other geocaching-related assets from around the web into a single, organized reference.

## Category
- Themed directory / Awesome list

## Key Focus
- Geocaching (GPS-based outdoor treasure hunting)
- Online resources and tools for geocachers

## Features
- **Curated resource list**: Collects geocaching-related links and assets from across the internet.
- **Geocaching scope**:
  - Focus on the recreational activity of hiding and seeking geocaches using GPS or mobile devices.
  - Emphasis on caches located worldwide and identified by coordinates.
- **Multiple resource types** (as indicated in the project description and structure):
  - Tools and utilities for geocaching.
  - Mobile and web apps related to geocaching.
  - Guides and how-to resources.
  - General reference resources about geocaching.
- **Open contribution model**:
  - Public contribution guidelines for suggesting new links and improvements.
  - Code of conduct for community participation.
- **Openly licensed**:
  - Includes a LICENSE file clarifying usage terms.
- **GitHub-based directory**:
  - Maintained as a README-based list for easy browsing and forking.

- Not applicable. This is a free, open GitHub-based resource list.
