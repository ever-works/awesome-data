### Latin America KLEMS

**Type:** Themed directory / structured dataset resource  
**Category:** Themed Directories  
**Tags:** datasets, productivity, economics

#### Description
Latin America KLEMS is a regional KLEMS‑style database project that provides detailed accounts of productivity, output, and factor inputs for Latin American countries. Listed under the economics section of Awesome Public Datasets, it serves as a structured data resource and directory of downloadable productivity statistics for the region.

#### Features
- KLEMS‑style database structure for productivity analysis.
- Coverage of multiple Latin American countries.
- Detailed measures of:
  - Productivity
  - Output
  - Factor inputs (e.g., labor, capital, and related inputs where available)
- Organized as a regional resource focused specifically on Latin America.
- Included in the Awesome Public Datasets economics section, indicating a curated and topic‑centric public data source.
- Functions as a directory of downloadable datasets, rather than a single file, enabling access to multiple related productivity series.
