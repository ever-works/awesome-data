## Overview

WebAssembly (Wasm) Components are a typed package format that packages core Wasm Instructions into typed libraries and binaries. This repository provides a curated collection of Component Model tooling and ready-to-use components.

## What are WASM Components?

- Typed package format for WebAssembly
- Fully typed interfaces with language bindings generators
- Portable across environments with WASI 0.2+
- Support for async operations (coming in WASI 0.3)

## Features

- Component Model tooling and frameworks
- Ready-to-use WebAssembly components
- Language bindings generators
- WASI implementation guides
- Server-side WASM production examples
- Component composition patterns

## Recent Developments (2026)

WASI 0.2 was released in early 2023, making components portable across environments. WASI 0.3 is planned for late 2025 with native async support. Server-side WASM is becoming a serious production option with growing adoption.
