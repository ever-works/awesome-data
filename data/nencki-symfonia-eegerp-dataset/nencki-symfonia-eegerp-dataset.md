# Nencki-Symfonia EEG/ERP Dataset

## Overview
The Nencki-Symfonia EEG/ERP Dataset is a high-density electrophysiological dataset hosted on GigaDB. It contains detailed EEG and event-related potential (ERP) recordings designed to support research on brain responses and cognitive processes.

- **Type:** Research dataset
- **Domain:** Neuroscience / Cognitive neuroscience
- **Host platform:** GigaDB
- **Provider / Brand:** Nencki Institute (Nencki-Symfonia)
- **Access URL:** http://gigadb.org/dataset/100990

## Features
- High-density EEG recordings
- ERP (event-related potential) data suitable for time-locked analysis of brain responses
- Detailed electrophysiological measurements for studying cognitive processes
- Structured as a research-oriented dataset (usable for analysis, method development, and benchmarking)
- Hosted and distributed via the GigaDB repository for scientific data

## Use Cases
- Analysis of brain responses to experimental stimuli
- Cognitive process investigation (e.g., perception, attention, memory)
- Method development and validation in EEG/ERP signal processing
- Training and benchmarking of algorithms in neuroscience and neuroimaging
