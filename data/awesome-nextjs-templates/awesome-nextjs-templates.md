## Overview

Awesome Next.js Templates provides curated starter templates and boilerplates for Next.js applications.

## Template Categories

- **SaaS Starters** - B2B/B2C application templates
- **E-commerce** - Online store templates
- **Blogs** - Content management systems
- **Portfolios** - Personal website templates
- **Admin Dashboards** - Backend management interfaces
- **Landing Pages** - Marketing page templates

## Popular Starters

- Next.js + TypeScript + Tailwind
- Next.js + Prisma + tRPC
- Next.js + Supabase + Auth
- T3 Stack (Next.js, TypeScript, Tailwind, tRPC)
- create-t3-app

## Features Included

- **Authentication** - NextAuth.js, Clerk, Supabase Auth
- **Databases** - Prisma, Drizzle, TypeORM
- **Styling** - Tailwind CSS, styled-components
- **UI Components** - shadcn/ui, Chakra UI, MUI
- **State Management** - Zustand, Redux Toolkit

## Tech Stacks

- Serverless (Vercel, Netlify)
- Full-stack (Node.js, PostgreSQL)
- JAMstack (Static generation)
- Headless CMS integration

## Use Cases

- Rapid prototyping
- Production applications
- Learning Next.js
- Client projects
