## Overview

Comprehensive collection of low-code and no-code platforms for building applications without traditional coding.

## Open Source Platforms

- Budibase - Build apps quickly
- ToolJet - Internal tools builder
- n8n - Workflow automation
- NocoBase - AI-powered platform
- Appsmith - Low-code framework

## Commercial Platforms

- Retool - Internal tools
- OutSystems - Enterprise platform
- Mendix - Application development
- Bubble - Web app builder
- Webflow - Website builder

## Use Cases

- Internal tools
- Business applications
- Workflow automation
- Admin panels
- CRUD apps

## Features

- Visual drag-and-drop
- Database integrations
- API connections
- Custom code support
- Deployment options

Ranges from free open-source to enterprise commercial.
