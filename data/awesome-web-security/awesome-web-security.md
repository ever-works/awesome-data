## Overview
Awesome Web Security is a curated, GitHub-hosted “awesome list” that aggregates learning materials, tools, and references focused on web security. It functions as a directory for developers, security engineers, and learners who want to study, practice, and deepen their understanding of web application security.

## Features
- **Curated web security resources**  
  A hand-picked collection of links and references related specifically to web application security.

- **Awesome-style directory structure**  
  Organized as a classic "awesome list" repository on GitHub, making it easy to browse and contribute.

- **Learning and practice focus**  
  Emphasis on materials that help users both learn concepts and practice real-world web security skills (e.g., labs, training resources, tutorials).

- **Multi-language documentation**  
  - `README.md` (main list)
  - `README-jp.md` (Japanese)
  - `README-zh.md` (Chinese)

- **Contribution guidelines**  
  - `CONTRIBUTING.md` describing how to propose new links or improvements to the list.

- **Community standards**  
  - `code-of-conduct.md` outlining expected behavior for contributors and community members.

- **Developer tooling & metadata**  
  - `.github` directory for repository configuration (issues/pr templates, workflows, etc.).
  - `.vscode` directory for editor-related configuration.
  - `.gitignore` for repository housekeeping.

- **Public, community-driven project**  
  - Public GitHub repository that can be starred, forked, and extended by the community.
