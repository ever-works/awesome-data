# meta-awesome: Get Apps

**Category:** Meta Directories  
**Tags:** meta-directory, apps, awesome-lists  
**Source:** https://github.com/jepemo/meta-awesome/blob/master/doc/get-apps.md

## Overview
meta-awesome: Get Apps is a curated meta-directory that aggregates multiple "awesome" lists focused on application discovery. It provides a central index of app catalogs, install guides, and related resources to help users find and install software across platforms.

## Features
- **Meta-aggregation of awesome lists**
  - Collects and links to various GitHub "awesome" lists related to applications.
  - Acts as an index rather than hosting the lists directly.

- **Application discovery focus**
  - Targets lists that help users discover new apps and tools.
  - Emphasis on catalogs of software rather than general programming resources.

- **Installation and setup resources**
  - Includes lists and documents that cover app installation, package managers, and deployment resources.

- **Organized documentation page**
  - Central documentation located in `doc/get-apps.md` within the meta-awesome repository.

- **GitHub-based curation**
  - Built on top of GitHub, leveraging public repositories and community-maintained awesome lists.
