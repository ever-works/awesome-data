# Awesome WP-CLI

**Website:** <https://github.com/schlessera/awesome-wp-cli#readme>

A curated directory of commands, packages, and resources for WP-CLI, the command-line interface used to manage WordPress sites.

---

## Overview

Awesome WP-CLI is an open-source, community-maintained list that organizes the WP-CLI ecosystem. It focuses on third-party commands, Composer packages, and related resources that extend or improve working with WP-CLI when managing WordPress installations via the command line.

---

## Features

- **Curated ecosystem overview**  
  - Focused specifically on WP-CLI and its extensions.  
  - Highlights useful third-party commands and packages.

- **Packages and commands**  
  - Lists Composer packages that provide new or enhanced WP-CLI commands.  
  - Includes commands bundled inside WordPress plugins or themes.

- **Resources for working with WP-CLI**  
  - Collects learning and reference resources relevant to WP-CLI usage.  
  - Oriented toward command-line management of WordPress instead of the admin dashboard.

- **Open-source and community-driven**  
  - Hosted on GitHub as a public repository.  
  - Includes contribution guidelines (`contributing.md`).  
  - Includes a Code of Conduct (`code-of-conduct.md`).

- **Project metadata & tooling**  
  - Uses `awesome-lint` to enforce Awesome List conventions.  
  - Repository includes standard project files (`.editorconfig`, `.gitattributes`, `.gitignore`, `package.json`, `package-lock.json`).  
  - Licensed under an open-source license (`license` file in the repo).
