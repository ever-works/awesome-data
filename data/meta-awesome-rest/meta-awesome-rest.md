# Meta Awesome – REST

**Category:** Meta Directories  
**Website:** <https://github.com/jepemo/meta-awesome/blob/master/doc/rest.md>  
**Platform:** GitHub

## Overview
Meta Awesome – REST is an Awesome-style meta directory that aggregates curated lists and Awesome repositories focused on the REST architectural style, its principles, and best practices. It serves as a central index pointing to other high-quality REST-related collections.

## Features
- **Meta directory of REST resources** – Acts as a hub that links out to multiple curated REST and API-related lists.
- **Aggregation of Awesome lists** – Specifically targets "Awesome"-branded repositories and collections in the REST/API ecosystem.
- **Focus on REST architectural style** – Emphasizes resources around REST design principles, constraints, and architectural best practices.
- **Coverage of best practices** – Includes links to lists that address API design quality, consistency, and robustness in RESTful systems.
- **API-centric scope** – Oriented toward RESTful APIs, including design, consumption, and associated tooling.
- **Continuously extensible** – As a GitHub-based document, it can be updated and expanded by maintainers or contributors over time.
