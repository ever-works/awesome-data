# Awesome Astrophotography

**Website:** https://github.com/lunohodov/awesome-astrophotography#readme  
**Category:** Themed Directories  
**Tags:** photography, science, resources

## Overview
Awesome Astrophotography is a curated, open-source directory of tools, techniques, software, and learning resources for astrophotography (astronomical imaging of celestial objects, events, and night-sky regions).

## Features
- **Curated resource list** focused specifically on astrophotography and astronomical imaging.
- **Structured contents** for easy discovery of materials:
  - Articles
  - Books
  - Tutorials
  - Presentations
  - Software
    - Acquisition
    - Guiding
    - Image Processing
    - Planetarium
    - Planning
    - Other
  - YouTube Channels
  - Reference materials
  - Astronomy as a Service (AaaS)
  - Community resources
- **Educational focus** covering tools, techniques, and learning materials for all levels.
- **Open-source GitHub repository** with:
  - README-based index of resources
  - License file
  - Contribution guidelines
  - Code of conduct
- **Community contributions** encouraged via pull requests following the contributing guide.
