## Overview

Lists by jnv provides a comprehensive catalog of curated lists from GitHub and other platforms, going beyond traditional awesome lists to include all types of resource collections.

## Features

- Meta-list covering awesome lists and other list formats
- Includes non-GitHub curated collections
- Organized by topic and platform
- Links to lists in different languages
- Academic and research list compilations
- Community-curated collections
- Industry-specific resource lists
- Educational resource directories
- Tool comparison matrices
- Historical list archives
