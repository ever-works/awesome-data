## Overview

This awesome list provides comprehensive resources for feature flagging and management, covering both commercial and open-source solutions for progressive feature delivery and experimentation.

## Commercial Solutions

- **LaunchDarkly** - A feature management platform that allows software development teams to deliver to their customers with feature flags, targeted rollouts, and experimentation
- **Split** - A feature delivery platform that powers feature flag management, software experimentation, and continuous delivery
- **Statsig** - A software product experimentation tool designed to make A/B testing more useful for companies
- **Optimizely** - Feature experimentation and A/B testing platform

## Open Source Solutions

- **Unleash** - A powerful open-source solution for feature management with over 10,000 GitHub stars and 20+ million downloads. Best for engineering teams needing customizable, scalable feature rollouts
- **Growthbook** - An open-source platform to help companies make data-driven product decisions with feature flags and A/B tests
- **Flipt** - An open-source, self-hosted feature flag solution
- **Flagsmith** - An open-source feature flag and remote config service

## Use Cases

- Progressive feature rollouts and canary deployments
- A/B testing and experimentation
- Kill switches for emergency feature disabling
- User segmentation and targeting
- Configuration management without code deployment

## Pricing

- Open source solutions: Free and self-hosted
- Commercial platforms: Vary by provider, typically starting with free tiers for small teams
