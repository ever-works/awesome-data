## Overview

Awesome Python Books is an open, curated directory of Python books. It organizes learning resources for Python developers into structured categories, following the "awesome list" style.

## Features

- **Curated book directory** specifically focused on Python programming.
- **Topic-based organization** to help find books by subject area.
- **Multiple skill levels for grammar/fundamentals**:
  - Zero Basis (absolute beginners)
  - Primary (basic level)
  - Intermediate
  - Advanced
  - References
- **Domain-specific sections**, including:
  - Data Structures and Algorithms
  - Complexity Science
  - Game Development
  - Hardware
  - Machine Learning
  - Deep Learning
  - Natural Language Processing
  - Network
  - Science and Data Analysis
  - Security
  - SQL and ORMs
  - System Administration
  - Testing
  - Web Development
  - Web Scraping
- **Other Awesome Lists** section linking to related curated lists.
- **Multilingual documentation** with a Chinese README (`README-ZH_CN.md`).
- **Open source repository** hosted on GitHub.
- **Contribution guidelines** via `CONTRIBUTING.md` for community contributions.
- **Clearly defined license** via `LICENSE` file.
