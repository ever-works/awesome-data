## Overview

Awesome Electron Alternatives provides a curated list of frameworks for building cross-platform desktop applications as alternatives to Electron, often with smaller bundle sizes and better performance.

## Featured Frameworks

- **Tauri** - Build smaller, faster, and more secure desktop applications using Rust with web frontend (uses WebView2/WebKit)
- **Wails** - Create desktop apps using Go and web technologies
- **NW.js** - Call Node.js modules directly from the DOM
- **Neutralinojs** - Portable and lightweight cross-platform application development framework

## Comparison

**Size Comparison:**
- Electron apps: ~120MB (includes Chromium)
- Tauri apps: ~5-10MB (uses system webview)
- Wails apps: ~10-15MB (uses system webview)

**Performance:**
- System webview-based alternatives use less memory
- Faster startup times
- Smaller distribution sizes

## Key Differences from Electron

- Use system webview instead of bundled Chromium
- Smaller binary sizes
- Lower memory consumption
- Different backend languages (Rust, Go vs Node.js)
- Better security in some cases

## Use Cases

- Building lightweight desktop applications
- Creating performance-critical apps
- Reducing distribution size
- Cross-platform development
- Secure desktop applications
