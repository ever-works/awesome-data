## Overview

Awesome Decentralized is a comprehensive collection of distributed, decentralized, and peer-to-peer applications and tools. The list focuses on projects that enable users to maintain privacy and avoid centralized control.

## Key Categories

### Applications
- **Communication**: Ricochet (anonymous instant messaging via Tor), Matrix (decentralized communication protocol)
- **File Storage**: IPFS, Dat, Storj, Sia
- **Social Networks**: Mastodon, Diaspora, Scuttlebutt
- **Cryptocurrencies**: Bitcoin, Ethereum, Monero

### Infrastructure
- **Networking**: Tor, I2P, ZeroNet
- **Databases**: OrbitDB, GunDB
- **Platforms**: Holochain, Solid

## Use Cases

- Building censorship-resistant applications
- Creating privacy-preserving communication tools
- Developing distributed file storage systems
- Implementing peer-to-peer networks

Free and open-source. Most projects are community-driven and available under various open-source licenses.
