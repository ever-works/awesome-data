## Overview

Awesome 3D provides comprehensive resources for 3D graphics development, from web-based visualization to professional rendering engines.

## 3D Web Technologies

- **Three.js** - Popular JavaScript 3D library
- **Babylon.js** - Powerful 3D engine for the web
- **WebGL** - Low-level 3D graphics API for browsers
- **WebGPU** - Next-generation graphics API
- **A-Frame** - VR framework for the web
- **PlayCanvas** - WebGL game engine

## Rendering Engines

- **Blender** - Open-source 3D creation suite
- **Unity** - Real-time 3D development platform
- **Unreal Engine** - High-fidelity game engine
- **Godot** - Open-source game engine

## 3D Libraries & Frameworks

- **Open3D** - Modern library for 3D data processing (Python/C++)
- **VTK** - Visualization Toolkit for scientific visualization
- **PCL** - Point Cloud Library
- **OpenGL** - Cross-platform graphics API
- **Vulkan** - High-performance 3D graphics API

## Modeling & CAD

- **OpenSCAD** - Programmable CAD
- **FreeCAD** - Parametric 3D modeler
- **Blender** - Modeling, animation, rendering
- **MeshLab** - 3D mesh processing

## 3D File Formats

- glTF - GL Transmission Format
- OBJ - Wavefront format
- FBX - Autodesk format
- STL - 3D printing format
- COLLADA - Interchange format

## Use Cases

- Web-based 3D visualization
- Game development
- Scientific visualization
- CAD and engineering
- 3D printing preparation
- VR/AR applications
