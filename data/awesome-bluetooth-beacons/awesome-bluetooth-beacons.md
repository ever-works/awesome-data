# Awesome Bluetooth Beacons

A curated, open-source directory of tools, libraries, and resources for working with Bluetooth beacons (iBeacon, Eddystone, and related BLE technologies).

**Website:** https://github.com/rabschi/awesome-beacon#readme  
**Category:** Themed Directories  
**Tags:** IoT, Bluetooth, Hardware

---

## Overview

Awesome Bluetooth Beacons (awesome-beacon) is a community-maintained list of software, tools, libraries, and educational resources for developing with Bluetooth beacon technologies, including Google Eddystone, Apple iBeacon, and general Bluetooth Low Energy (BLE). It organizes links to projects and documentation across major platforms and use cases.

---

## Features

### Core Focus Areas
- **iBeacon and Eddystone coverage** – Resources and tools for both major beacon formats.
- **Bluetooth Low Energy (BLE)** – Links to Bluetooth Smart / BLE tooling and documentation.
- **Cross-platform scope** – Sections for iOS, Android, OS X, Linux, Windows, Node.js, and hybrid frameworks (Cordova, PhoneGap, Xamarin, Titanium).
- **Open source orientation** – Emphasis on open-source projects, libraries, and code examples.

### Major Sections & Resource Types
- **Eddystone by Google**  
  - Information and resources around Eddystone as a platform for marking up the physical world with contextual data.

- **Physical Web**  
  - Resources describing the Physical Web concept: interaction with nearby smart devices (e.g., vending machines, posters, bus stops) without requiring dedicated app installs.

- **Proximity Beacon API by Google**  
  - Links related to Google’s Proximity Beacon API for managing and interacting with beacons.

- **iBeacon Resources by Apple**  
  - Official Apple documentation and materials about iBeacon and its use in proximity and location-aware applications.

- **iBeacon for Developers**  
  - Developer-focused resources illustrating iBeacon use cases (e.g., venue check-ins, museum exhibits, context-aware content) and implementation details.

### Platform- and Topic-Specific Collections
- **Hacks & Cool Apps**  
  - Examples of experimental projects, demos, and creative applications built with beacons.

- **Installation & Radio Planning**  
  - Guidance and tools related to physical deployment, coverage, and radio planning for beacon installations.

- **Beacon Discovery & Configuration Tools**  
  - Utilities for scanning, discovering, configuring, and managing beacon devices.

- **iOS**  
  - **Beacon Scanner Apps** – iOS apps for scanning and testing beacons.  
  - **Swift** – Swift libraries and code examples for beacon integration.  
  - **Objective-C** – Objective-C libraries and sample code.  
  - **Stack Overflow Q&A** – Curated questions and answers relevant to beacon development on iOS.

- **Virtual Beacons**  
  - Tools and projects that simulate beacons in software for testing and development.

- **Android**  
  - **Beacon Development** – Android SDKs, libraries, and examples for beacon-enabled apps.  
  - **Beacon Scanner Apps** – Android apps for scanning and debugging beacons.  
  - **Beacon Advertiser Apps** – Apps that turn Android devices into beacon transmitters.  
  - **Stack Overflow Q&A** – Curated Q&A for Android beacon development.

- **Cordova, PhoneGap, Xamarin, Titanium**  
  - Plugins and frameworks for integrating beacons into hybrid and cross-platform mobile apps.

- **OS X / macOS**  
  - Libraries and tools for working with beacons on Apple desktop platforms.

- **Linux**  
  - Linux-based tools and code for beacon scanning, advertising, and BLE experimentation.

- **Node.js**  
  - Node.js libraries and utilities for interacting with BLE devices and beacons.

- **Windows**  
  - Windows tooling and libraries for beacon development and BLE interactions.

### Bluetooth Low Energy & Web
- **Bluetooth Smart & BLE Tools**  
  - General-purpose BLE debugging, analysis, and development tools.

- **Web Bluetooth API**  
  - Resources for using Web Bluetooth to interact with nearby BLE / beacon devices directly from web browsers.

### Ecosystem & Industry
- **Beacon Developer Kits & BLE Chips**  
  - References to hardware kits and BLE chipsets for building beacon devices.

- **Proximity Trends & Outlook**  
  - Articles and resources discussing trends, use cases, and the future of beacon-based proximity solutions.

- **Vendor-driven Beacon Standardization**  
  - Information around standardization efforts led by vendors in the beacon and proximity space.
