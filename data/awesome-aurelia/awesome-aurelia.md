# Awesome Aurelia

**Tags:** awesome-lists, javascript, framework  
**Brand:** Aurelia

## Overview
Awesome Aurelia is a curated, community-maintained directory of libraries, plugins, tools, and learning resources for the Aurelia front-end framework. It aggregates links to official documentation, community channels, tutorials, example projects, and integrations, helping developers discover Aurelia-related material in one place.

## Features

### Scope of the Directory
- Focused specifically on the **Aurelia** JavaScript client framework.
- Organizes content into structured sections for easier navigation.
- Includes links to both official and community-maintained resources.

### Content Sections

#### Aurelia Core Resources
- **Official Resources**
  - Official site
  - Official blog
  - Official documentation
  - API documentation
  - Getting started guide
  - Official Aurelia GitHub repository
  - Aurelia creator/lead (Rob Eisenberg) Twitter profile

  - `#aureliajs` Twitter hashtag
  - Aurelia forum on Discourse

  - Aurelia TypeScript Gitter channel

  - Aurelia AMA link (Hashnode)

- **Aurelia Tools** – various utilities and tools supporting Aurelia development.
- **Seed Projects**

  - TypeScript seed projects

#### Learning Resources
- **Aurelia Tutorials**
  - Step-by-step guides and how-tos for building with Aurelia.

- **Aurelia Videos** – video tutorials, talks, and walkthroughs.
- **Aurelia Books** – books dedicated to Aurelia development.
- **Aurelia Articles** – written articles on patterns, best practices, and use cases.

#### UI & Design
- **Material Design**

- **Aurelia Libraries** – general-purpose libraries compatible with or focused on Aurelia.

#### Examples & Sample Code

### Technology Coverage
- **JavaScript / Aurelia framework** – primary focus.
