## Overview

Awesome .NET MAUI is a comprehensive collection of resources for .NET MAUI, the evolution of Xamarin.Forms that extends mobile development to desktop platforms (Windows and macOS). MAUI enables developers to build apps that run natively on Android, iOS, macOS, and Windows from a single codebase.

## Features

- Official .NET MAUI documentation and samples
- MAUI Community Toolkit controls and behaviors
- Third-party UI control libraries
- MVVM frameworks (CommunityToolkit.Mvvm, Prism)
- Platform-specific integrations and native APIs
- Blazor Hybrid for web-based UIs in MAUI
- Data binding and state management patterns
- Navigation and routing strategies
- Dependency injection and services
- Local data storage (SQLite, Preferences)
- REST API and GraphQL integration
- Authentication flows (OAuth, MSAL)
- Maps and location services
- Camera and media access
- Performance optimization techniques
- Deployment and distribution guides
- Migration from Xamarin.Forms
- Community resources and forums
