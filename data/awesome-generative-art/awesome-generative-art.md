## Overview

Awesome Generative Art is a comprehensive collection of resources for artists and developers interested in creating art through algorithms and code.

## Features

- **Tutorials**: Step-by-step guides including "How to make your first generative art with P5.js"
- **Books**: Recommended reading on generative art theory and practice
- **Editors**: Processing, p5.js Editor, and other creative coding environments
- **Tools**: Software and libraries for generative art creation
- **Frameworks**: p5.js, Processing, openFrameworks, and more
- **Galleries**: Online platforms like OpenProcessing for sharing and discovering work
- **Learning Resources**: The Coding Train and other beginner-friendly tutorials
- **Communities**: Forums and groups for generative artists

## Platforms

- Processing (Java-based)
- p5.js (JavaScript)
- Three.js for 3D
- Canvas API
- WebGL and shaders
- TouchDesigner
- Max/MSP

## Topics Covered

- Algorithmic composition
- Particle systems
- Fractals and L-systems
- Noise and randomness
- Data visualization as art
- Interactive installations
- Audio-reactive visuals
- Generative typography

## Use Cases

- Interactive installations
