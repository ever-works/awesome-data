## Overview

Awesome Frontend Development aggregates awesome lists for all major frontend frameworks and tools in one comprehensive resource.

## Framework Lists Included

- **React** - Component-based UI library
- **Vue.js** - Progressive JavaScript framework
- **Angular** - Platform for building web applications
- **Svelte** - Compiled component framework
- **Flutter** - Cross-platform UI toolkit

## Mobile Frameworks

- **React Native** - Build mobile apps with React
- **Flutter** - Google's UI toolkit for mobile, web, desktop
- **Ionic** - Hybrid mobile app framework
- **Capacitor** - Native mobile runtime

## Build Tools

- **Webpack** - Module bundler
- **Vite** - Next-generation frontend tooling
- **Rollup** - Module bundler for libraries
- **Parcel** - Zero-config bundler
- **esbuild** - Extremely fast bundler

## Desktop Frameworks

- **Electron** - Build desktop apps with web technologies
- **Tauri** - Lightweight desktop framework
- **NW.js** - Node-WebKit

## Use Cases

- Finding framework-specific resources
- Choosing frontend technologies
- Learning frontend development
- Building modern web applications
