# Awesome-Hacker-Search-Engines

**Category:** Themed Directories  
**Tags:** security, search, hacking

## Description
Awesome-Hacker-Search-Engines is an Awesome-style curated directory of specialized search engines that are useful for:
- Penetration testing
- Vulnerability assessments
- Red/Blue team operations
- Bug bounty reconnaissance and research

## Features
- **Curated collection of search engines** focused on security and hacking use cases.
- **Themed directory format** (Awesome list) organizing tools for quick exploration.
- **Coverage of multiple security workflows**, including:
  - Penetration testing discovery and enumeration
  - Vulnerability research and assessment support
  - Red team reconnaissance and attack surface mapping
  - Blue team investigation and defensive research
  - Bug bounty recon and target profiling
- **Repository structure** for easy browsing and contribution:
  - `Awesome.md` containing the main curated list
  - `Doc/` folder for additional documentation
  - `index.html`, `style.css`, `script.js` for a web view of the directory
  - GitHub Actions workflows under `.github/workflows`
- **Open licensing** via included `LICENSE` file (MIT badge shown in README).
- **Compatible with the broader Awesome ecosystem** (uses the Awesome badge and style conventions).
