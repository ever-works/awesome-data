## Overview

Awesome visionOS provides comprehensive resources for developers building applications for Apple's spatial computing platform, covering RealityKit, SwiftUI, and ARKit integration.

## Features

- visionOS SDK documentation and tutorials
- RealityKit for 3D content and immersive experiences
- SwiftUI for spatial interfaces and windows
- SharePlay for collaborative experiences
- Spatial audio and EyeSight features
- Hand tracking and gesture recognition
- Window management in 3D space
- Passthrough and immersion levels
- App architecture patterns for spatial computing
- Sample projects and open-source apps
