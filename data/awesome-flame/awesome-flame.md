# Awesome Flame

**Category:** Themed Directories  
**Tags:** awesome-lists, games, Dart

An open, community-maintained curated list of the best resources around the Flame game engine for Flutter, including games, libraries, tools, and learning material.

## Overview

Awesome Flame is a GitHub-hosted "awesome list" dedicated to the Flame game engine ecosystem. It aggregates and organizes links to notable Flame-based projects and supporting resources, helping developers discover examples, tooling, and learning content for building games with Flutter and Flame.

- **Project type:** Open curated directory / awesome list
- **Ecosystem:** Flame game engine (for Flutter / Dart)
- **Source:** [GitHub Repository](https://github.com/flame-engine/awesome-flame#readme)
- **License:** CC0 (public domain dedication, per repository badge)

## Features

- **Curated Flame ecosystem list**  
  - Focused specifically on the Flame game engine and its surrounding community.
  - Collects “best of” items rather than all possible links.

- **Games and example projects**  
  - Links to completed and in-progress games built with Flame.  
  - Example codebases that demonstrate gameplay mechanics and architectural patterns in Flame.

- **Libraries and extensions**  
  - Third-party packages and libraries that extend or complement Flame.  
  - Utilities for common game-development tasks within the Flutter/Flame stack.

- **Tools**  
  - Tools that assist with development workflows for Flame games (e.g., build, debug, assets, level design) where available in the list.

- **Tutorials and learning resources**  
  - Tutorials for getting started with Flame and building specific types of games.  
  - Articles, blog posts, and educational content explaining Flame concepts and game development patterns in Dart/Flutter.

- **Articles and reference material**  
  - Curated reading list of posts and write-ups relevant to Flame and Flutter game development.

- **Multilingual content files**  
  - Additional content files such as `CONTENT_ES.md` (Spanish) and `CONTENT_ZH.md` (Chinese) to support non-English audiences.

- **Contribution guidelines**  
  - `CONTRIBUTING.md` in the repository describes how to add new projects or resources to the list.

- **Open license**  
  - Licensed under CC0, allowing broad reuse of the list’s contents.
