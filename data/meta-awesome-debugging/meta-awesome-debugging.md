# Meta Awesome – Debugging

**Website:** <https://github.com/jepemo/meta-awesome/blob/master/doc/debugging.md>

## Overview
Meta Awesome – Debugging is a curated, “awesome-style” meta index that aggregates multiple debugging-related awesome lists and resources into a single, organized reference. It focuses on tools, techniques, and best practices for debugging software across different languages and platforms.

## Category
- Meta-directories
- Developer tools

## Features
- **Meta index of debugging lists**: Central hub that links out to numerous “awesome” collections focused specifically on debugging.
- **Coverage of debugging tools**: Collects references to debuggers, profilers, tracers, and related utilities from various awesome lists.
- **Practices and methodologies**: Includes links to resources on debugging strategies, workflows, and best practices.
- **Language-agnostic scope**: Targets debugging content across multiple programming languages and ecosystems (where available from the underlying lists).
- **Ecosystem-wide aggregation**: Pulls in disparate awesome lists so users can discover debugging resources from multiple communities in one place.
- **Open GitHub-based index**: Maintained as a Markdown document on GitHub, making it easy to browse, fork, and contribute.
- **Part of the Meta Awesome collection**: Integrates into a broader meta-directory that indexes awesome lists for many different topics, with this document dedicated to debugging.
