# Awesome IPFS

**Website:** https://github.com/ipfs/awesome-ipfs#readme  
**Category:** Themed Directories  
**Tags:** awesome-lists, blockchain, curated-lists

## Overview
Awesome IPFS is a community-maintained, curated directory of projects, applications, tools, services, and learning resources built on or for IPFS, the peer‑to‑peer hypermedia protocol.

## Features
- **Curated project directory**
  - Community list of notable projects, apps, tools, and services related to IPFS.
  - Focuses on software and resources that use or support the IPFS protocol.

- **Multiple content sections** (as listed in the repository)
  - **Implementations**
    - Highlights different open-source implementations of the IPFS protocol.
    - Links to the official IPFS docs page for popular or actively maintained implementations.
  - **Apps**
    - Directory of applications built on top of IPFS.
  - **Browsers**
    - List of web browsers or browser integrations that support IPFS (e.g., `ipfs://` handling or similar integrations).
  - **Tools**
    - Collection of tools that assist in working with IPFS (e.g., development tooling, integrations).
  - **Debugging Tools & Learn**
    - Resources and tools focused on debugging IPFS-based systems and learning how IPFS works.

- **Community contributions**
  - Open contribution model via GitHub pull requests.
  - Defined **Content Policy** for submissions to ensure quality and relevance.

- **Repository resources**
  - CONTRIBUTING guidelines for how to add or update entries.
  - CODEOWNERS and maintenance tooling (e.g., stale issue management) to keep the list up to date.

## Content Type
- Non-commercial, informational GitHub repository.
- Functions as an **awesome-style list** rather than a hosted web application or SaaS product.
