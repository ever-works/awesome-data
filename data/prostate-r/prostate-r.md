# Prostate-R

A classic prostate cancer dataset distributed with the R package ElemStatLearn and included in the Awesome Public Datasets directory. It contains clinical and biomarker variables frequently used for statistics and machine learning examples.

[https://web.stanford.edu/~hastie/ElemStatLearn/datasets/prostate.data](https://web.stanford.edu/~hastie/ElemStatLearn/datasets/prostate.data)
