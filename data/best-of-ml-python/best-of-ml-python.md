## Overview

Best of ML Python is a ranked, curated list of awesome machine learning Python libraries that is updated weekly with quality metrics.

## Featured Libraries

- **graph-nets** - DeepMind's library for building graph networks in TensorFlow
- **jraph** - JAX library for graph neural networks
- **PyTorch Geometric** - Geometric deep learning with PyTorch
- **DGL** - Deep Graph Library for graph neural networks
