## Overview

Awesome TypeSafe is a curated list of TypeScript libraries that provide strong type safety guarantees. These libraries help developers catch errors at compile-time and ensure type correctness between different layers of an application, from frontend to backend.

## Features

- Schema validation libraries (Zod, Yup, io-ts, Valibot)
- End-to-end type-safe API solutions (tRPC, Zodios, ts-rest)
- Type-safe database clients (Prisma, Drizzle, Kysely)
- Type-safe form libraries (React Hook Form with Zod, Formik)
- Runtime type checking and validation
- Type-safe routing solutions
- Type-safe state management libraries
- Type-safe testing utilities
- Type inference helpers and utilities
- TypeScript configuration best practices
- Type-safe GraphQL clients and servers
- Type-safe environment variable parsers
- Monorepo type-sharing strategies
- Code generation tools for type safety
- Static type checkers and linters
- Type-safe utility libraries
