## Overview

favorite-link is an open source, awesome-style curated collection that automatically gathers and organizes favorite open source projects and links on a daily basis. It focuses on building a structured directory of notable GitHub repositories and related resources, with supporting RSS/feed tooling.

## Features

- **Daily collection of links**: Automatically collects new favorite open source projects and links every day.
- **Awesome-list style directory**: Organizes items in the style of GitHub "awesome-lists" / curated lists.
- **Focus on open source repositories**: Emphasis on notable GitHub projects and developer resources.
- **Tagging and categorization**: Items are organized with tags and categories to form a navigable directory.
- **RSS / feed support**: Includes tooling to expose the curated links via RSS/feeds for subscription and automation.
- **Web frontend**: Public website at `https://www.guanguans.cn/favorite-link/` presenting the directory.
- **GitHub-based workflow**: Managed as a GitHub repository, enabling version control, forking, and collaboration.
- **High-activity project**: Long-running repo with many commits, indicating ongoing curation and updates.
