# Meta Awesome - Packages

**Website:** https://github.com/jepemo/meta-awesome/blob/master/doc/packages.md  
**Category:** Meta Directories  
**Tags:** awesome-lists, meta-directory, packages

## Overview
Meta Awesome – Packages is an Awesome-style meta collection that indexes other curated lists focused on software packages and libraries. It acts as a higher-level directory, helping users discover and navigate many different package-related Awesome lists in one place.

## Features
- **Meta index of package-related Awesome lists** – aggregates links to multiple curated lists of software packages and libraries.
- **Awesome-style organization** – follows the familiar “Awesome list” convention, making it easy to browse and contribute.
- **Higher-level directory** – serves as a top-level map to specialized Awesome lists, rather than listing individual packages directly.
- **Focus on software packages and libraries** – targets collections that catalog language-specific, framework-specific, or ecosystem-specific packages.
- **GitHub-based collection** – hosted on GitHub, allowing easy access, forking, and contribution via pull requests.
