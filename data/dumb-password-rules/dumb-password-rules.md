## Overview

Dumb Password Rules is a compilation of sites with problematic password requirements that actually reduce security rather than enhance it. The repository includes a bot that periodically posts random examples on social media platforms.

## Features

- **Comprehensive Documentation**: Database of sites with overly restrictive or nonsensical password rules
- **Public Shaming**: Highlights poor security practices to raise awareness
- **Community Contributions**: Open for submissions via pull requests
- **Automated Sharing**: Bot posts examples on Mastodon and Blue Sky
- **Website**: Compiled rules available at https://dumbpasswordrules.com

## Submission Guidelines

Contributions should include:
- Site name
- Link to the page with the offending rule
- Screenshot of the rule
- Written description of the problematic requirement

## Purpose

The repository aims to educate developers and businesses about password security best practices by showcasing examples of what NOT to do, promoting better authentication implementations that balance security with usability.
