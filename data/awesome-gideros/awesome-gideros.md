# Awesome Gideros

**Category:** Themed Directories  
**Tags:** awesome-lists, games, framework  
**Source:** https://github.com/stetso/awesome-gideros#readme

## Overview
Awesome Gideros is a curated, community-driven collection of resources for the Gideros game engine. It aggregates libraries, plugins, tools, and learning materials relevant to developing games and applications with Gideros.

## Features
- **Curated resource list** focused specifically on the Gideros game engine.
- **Libraries and plugins** for extending Gideros functionality (e.g., game logic, graphics, utilities, integrations).
- **Learning materials** such as tutorials, guides, and educational content for Gideros developers.
- **Open-source GitHub repository** allowing community contributions via pull requests.
- **Contributing guidelines** provided in a dedicated `contributing.md` file to standardize submissions and updates.
- **Brand-aligned assets** including a Gideros logo image (`gideros_logo.png`) for consistent visual identification of the list.
- **Versioned history** via Git commit log, allowing tracking of additions and changes to the resource list.
