## Overview

Best-of Lists takes a data-driven approach to curating open-source projects, featuring ranked lists based on metrics like GitHub stars, contributor activity, and project health scores.

## Features

- Ranked lists based on quantitative project metrics
- Automatic weekly or monthly updates
- Project health indicators and trending data
- Best-of-ml-python with 23K+ stars for machine learning
- Best-of-jupyter for notebook ecosystem tools
- Best-of-mcp-servers for Model Context Protocol
- Category-based organization (Web Dev, DevOps, AI, Security)
- Comparison across similar projects
- Historical trend analysis
- Search and filter capabilities
