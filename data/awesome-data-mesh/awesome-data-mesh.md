## Overview

Awesome Data Mesh compiles resources about Data Mesh, a modern data architecture paradigm that decentralizes data management across organizational domains, emphasizing domain ownership and data as a product.

## Four Core Principles

1. **Domain-oriented data and compute**: Data ownership aligned with business domains
2. **Data as a product**: Treating data with product thinking
3. **Self-serve data platform**: Infrastructure as code for data teams
4. **Federated computational governance**: Distributed decision-making with global standards

## Features

- **Educational Materials**: Published book *Data Management at Scale* (O'Reilly), training workshops by Zhamak Dehghani
- **Video Content**: Multiple video collections including "Data Mesh Paradigm Shift in Data Platform Architecture"
- **Community Engagement**: Active Slack community focused on Data Mesh and Scaled Architecture
- **Written Content**: Foundational articles by Zhamak Dehghani on Martin Fowler's platform, blog posts exploring implementation approaches
- **Conference Resources**: Domain-Driven Design Europe and other conference materials
- **Implementation Guides**: Explorations of different types of data meshes and ownership patterns
