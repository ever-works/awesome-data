# Awesome Public Datasets – Healthcare Category

**Brand:** awesome-public-datasets

## Overview

Awesome Public Datasets – Healthcare Category is a curated, open directory of publicly available healthcare and medical datasets. It is part of the broader Awesome Public Datasets project and focuses specifically on health‑related data resources suitable for research, analysis, machine learning, and education.

## Features

- **Curated Healthcare Dataset Collection**  
  Aggregates public datasets that are specifically focused on healthcare, medicine, and health sciences.

- **Open and Public Data Sources**  
  Only includes datasets that are openly accessible for research, analysis, and educational use (subject to each dataset’s own license and terms).

- **Part of Awesome Public Datasets Ecosystem**  
  Integrated into the main Awesome Public Datasets project (`apd-core`), ensuring consistent structure, quality standards, and community-driven maintenance.

- **Categorized and Cataloged Entries**  
  Datasets are organized within a dedicated `Healthcare` section, making it easier to discover health‑related data among the broader collection.

- **Research and Education Focus**  
  Targets use cases like academic research, data science projects, methodological demonstrations, and classroom teaching materials.

- **GitHub-based Directory**  
  Hosted as files in a GitHub repository, allowing easy browsing, forking, and integration into workflows and tools that use GitHub.

- **Community-Contributed**  
  The list can be extended and improved by the community via pull requests and issues on the GitHub repository (following the main project’s contribution guidelines).

- **Stable Referencing via Slug**  
  Identified in the ecosystem as `awesome-public-datasets-healthcare-category`, making it easy to reference or link to this specific themed subdirectory.

## Pricing

- **Free** – The directory itself is freely accessible as an open GitHub repository. Individual datasets may have their own licenses and usage conditions, which should be checked per dataset.
