# Awesome Supabase

**Description**  
A curated, community-driven collection of resources, tools, example projects, libraries, and official starters for building with Supabase, an open‑source Firebase alternative.

**Website**  
https://github.com/lyqht/awesome-supabase#readme

**Category**

**Tags**

## Features

- **Curated Awesome List**  
  - Central index of high‑quality Supabase-related resources.  
  - Follows the general "awesome" list conventions and standards.

- **Official Starters Index**  
  - Lists official Supabase starter projects.  
  - Starters are compatible with `@supabase/supabase-js` v2.  
  - Helps developers quickly bootstrap Supabase-powered applications.

- **Covers Core Supabase Capabilities (via linked resources)**  
  - Resources around using Supabase Postgres databases.  
  - Authentication integrations and examples.  
  - Instant APIs usage patterns.  
  - Edge Functions examples.  
  - Realtime subscriptions implementations.  
  - Storage features and usage.

- **Community & Contribution Friendly**  
  - Public contribution guidelines via `CONTRIBUTING.md`.  
  - Code of Conduct for community interactions.  
  - Open to new resources, tools, and examples from the community.

- **Repository Structure & Governance**  
  - Versioned via Git with visible commit history.  
  - Licensed via a dedicated `LICENSE` file, clarifying reuse.  
  - Automation/workflows present under `.github/workflows` (e.g., for linting or checks on contributions).
