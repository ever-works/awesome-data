# Awesome ComputerCraft

**Category:** Themed Directories  
**Tags:** games, mods, awesome-lists  
**Website:** https://github.com/tomodachi94/awesome-computercraft#readme

## Overview
Awesome ComputerCraft is a curated “awesome list” of libraries, programs, literature, and mods for **ComputerCraft** and its forks, a Minecraft mod that adds programmable computers and turtles. The repository aggregates community resources to help players and developers find tools and references for building with ComputerCraft.

## Features
- Curated list format focused on ComputerCraft and related forks.
- Includes:
  - **Libraries** for extending ComputerCraft functionality.
  - **Programs** and scripts that run on in‑game computers and turtles.
  - **Literature / documentation** resources relevant to ComputerCraft.
  - **Mods** that integrate with or extend ComputerCraft.
- Organized as a GitHub repository for easy browsing and contribution.
- Supporting files for project maintenance and contribution:
  - `CODE_OF_CONDUCT.md` for community standards.
  - `CONTRIBUTING.md` with contribution guidelines.
  - `DISCUSSION.md` and `FAQ.md` for project context and common questions.
  - Licensing information in `LICENSE.md` and `LICENSES/`.
  - Configuration for tooling (`.gitignore`, `.vale*`, `flake.nix`, `REUSE.toml`, etc.).

## Pricing
