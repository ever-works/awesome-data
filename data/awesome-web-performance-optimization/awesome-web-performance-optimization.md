## Overview
Awesome Web Performance Optimization (Awesome WPO) is a curated, community-driven list of tools, articles, and other resources dedicated to improving website speed and efficiency.

## Features
- Curated "awesome list" format focused specifically on web performance optimization (WPO)
- Includes a wide range of resources such as:
  - Tools for measuring, analyzing, and improving web performance
  - Articles and guides on best practices for website speed and optimization
  - Additional reference materials collected in the `content` directory
  - Separate `COURSES.md` file listing courses related to web performance
- Open, community-contributed repository (GitHub-based)
- Organized project structure with:
  - `content/` folder for categorized resources
  - `website/` folder for a browsable site version of the list
  - Configuration files for automation and CI (`.circleci`, `.github`)
- Governed by a published Code of Conduct (`CODE_OF_CONDUCT.md`)
- Licensed under the MIT license, allowing broad reuse and adaptation
