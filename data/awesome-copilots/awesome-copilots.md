## Overview

Awesome Copilots catalogs AI-powered assistants and copilots across different domains, helping users find the right AI tools to enhance their workflows.

## Coding Assistants

### GitHub Copilot
AI pair programmer from GitHub/OpenAI integrated into various IDEs.

### Amazon CodeWhisperer
AWS's code completion service with security scanning.

### Tabnine
AI code completion supporting multiple languages and private models.

### Codeium
Free AI-powered code acceleration with chat functionality.

### Cursor
AI-first code editor built on VS Code.
