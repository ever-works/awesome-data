# Meta Awesome - PHP

**Category:** Meta Directories  
**Slug:** `meta-awesome-php`  
**Brand:** GitHub

Meta Awesome - PHP is a curated meta-index of "Awesome"-style lists and directories focused on PHP. It aggregates and organizes Awesome PHP resources so developers can quickly find specialized, community-maintained PHP lists and materials.

---

## Features

- **Meta-index of Awesome PHP resources**  
  Collects links to multiple Awesome-style PHP lists and directories in one place.

- **PHP-focused scope**  
  All indexed lists and resources are related to PHP, its ecosystem, and tooling.

- **Curated structure**  
  Resources are organized as a sublist within the broader Meta Awesome project, making it easier to navigate among many Awesome lists.

- **Central entry point for PHP Awesome lists**  
  Serves as a starting hub to discover more specific Awesome collections (e.g., frameworks, libraries, tools, learning resources) within the PHP ecosystem.

- **Open GitHub-hosted document**  
  Available as a Markdown document in a public GitHub repository (`meta-awesome/doc/php.md`), making it easy to browse, copy, or contribute via pull requests.

- **Part of the larger Meta Awesome project**  
  Integrates into a wider catalog of meta-lists for different languages and topics, allowing cross-navigation from PHP to other technology-specific Awesome indices.

## Pricing

- Access to Meta Awesome - PHP is **free** as it is a public GitHub-hosted resource.
