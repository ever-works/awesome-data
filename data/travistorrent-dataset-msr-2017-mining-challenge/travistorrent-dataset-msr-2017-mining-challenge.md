### TravisTorrent Dataset – MSR 2017 Mining Challenge

**Category:** Datasets  
**Tags:** datasets, software-engineering, GitHub  
**Source:** https://project-awesome.org/awesomedata/awesome-public-datasets

#### Overview
TravisTorrent is a research dataset created for the MSR 2017 Mining Challenge. It combines information from GitHub repositories with build and test data from Travis CI, supporting empirical software engineering and mining software repositories research.

#### Features
- Combined data from **GitHub** and **Travis CI**.
- Focused on **software engineering** and **continuous integration** research.
- Designed for **empirical studies** and **mining software repositories**.
- Suitable for analyzing relationships between code changes, builds, and test outcomes.

#### Use Cases
- Studying build and test behavior in open-source projects.
- Analyzing the impact of commits on CI outcomes.
- Mining software repositories for quality, reliability, and process metrics.
