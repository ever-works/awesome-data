## Overview

Awesome Data Science Viz provides curated tools for data science, machine learning, and visualization with focus on Python and web technologies.

## Python Tools

- **Analysis** - pandas, NumPy, SciPy
- **Machine Learning** - scikit-learn, XGBoost, LightGBM
- **Deep Learning** - TensorFlow, PyTorch, Keras
- **Visualization** - Matplotlib, Seaborn, Plotly

## Web Visualization

- **D3.js** - Data-driven documents for interactive viz
- **Plotly.js** - JavaScript graphing library
- **Chart.js** - Simple yet flexible charting
- **Vega** - Declarative visualization grammar
- **Observable** - Reactive notebook platform

## Data Science Workflow

- **Data Collection** - Web scraping, APIs
- **Data Cleaning** - pandas, OpenRefine
- **Exploration** - Jupyter notebooks
- **Modeling** - ML/DL frameworks
- **Visualization** - Charts and dashboards
- **Deployment** - Streamlit, Dash, Gradio

## Web Applications

- Streamlit for rapid prototyping
- Dash for analytical web apps
- Voilà for notebook dashboards
- Panel for custom apps

## Use Cases

- Exploratory data analysis
- Interactive dashboards
- Machine learning pipelines
- Report generation
- Web-based analytics
