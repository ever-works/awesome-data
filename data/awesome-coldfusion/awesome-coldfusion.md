# Awesome ColdFusion

**Website:** https://github.com/seancoyne/awesome-coldfusion#readme  
**Category:** Themed Directories  
**Tags:** programming-languages, web, developer-tools

## Description
Awesome ColdFusion is a curated directory of frameworks, libraries, tools, and resources for ColdFusion (CFML) development. It aggregates community-maintained links to software and references across the ColdFusion ecosystem.

## Features
- **Curated ecosystem directory**
  - Collects notable ColdFusion frameworks, libraries, and software projects.
  - Inspired by the broader *Awesome* lists pattern (e.g., awesome-javascript).
- **Organized by topic** (sections in the list include):
  - **Engines** – ColdFusion and CFML runtime engines and servers.
  - **Application Frameworks** – Web and application frameworks built for ColdFusion.
  - **Testing** – Tools, libraries, and frameworks for testing ColdFusion code.
  - **Dependency Injection** – DI/IoC libraries for ColdFusion applications.
  - **CMS** – Content Management Systems implemented in or for ColdFusion.
  - **NoSQL** – Libraries and tools for working with NoSQL databases from ColdFusion.
  - **REST** – Frameworks and utilities for building REST APIs with ColdFusion.
  - **Other Libraries** – Miscellaneous ColdFusion libraries that don’t fit other categories.
  - **Editors** – Editor/IDE integrations and tools oriented toward ColdFusion development.
  - **Tools** – Development, build, and utility tools supporting the ColdFusion stack.
  - **References** – Reference material such as language references or engine docs.
  - **Resources** – Tutorials, guides, articles, and learning materials.
  - **Documentation** – Official or community documentation resources.
- **Community-driven contributions**
  - Open to pull requests for adding or updating entries.
  - Maintained publicly on GitHub for transparency and collaboration.
- **License-free reuse**
  - Released under the Creative Commons CC0 Public Domain Dedication, allowing broad reuse and redistribution.
