# Awesome CSV

**Description**  
Awesome CSV is a carefully curated, open-source list of tools, libraries, and resources for working with CSV (Comma-Separated Values), covering validation, conversion, schema generation, data generation, standards, and more.

**Website**  
https://github.com/secretGeek/awesomeCSV#readme

## Features

- **Curated CSV Tooling Directory**  
  - Focused specifically on CSV, a text-based tabular data format using comma-separated values.  
  - Organizes links to tools and resources into clear sections.

- **Repair or Validate CSV**  
  - Collection of tools that help detect, validate, and repair malformed or non-standard CSV files.

- **Generate Table Schema**  
  - Tools that can infer or generate table/column schemas from CSV data.

- **Treat CSV as SQL**  
  - Utilities for querying CSV files using SQL-like syntax, enabling ad‑hoc analysis without importing into a database.

- **Convert To or From CSV**  
  - Tools for converting between CSV and other formats (e.g., import/export utilities, format transformers).

- **CSV ↔ JSON**  
  - Dedicated converters and libraries for transforming CSV data to JSON and back.

- **Generate Data**  
  - Resources and tools for generating sample or synthetic data sets in CSV format.

- **Data Collections**  
  - Links to CSV-based datasets for testing, experimentation, or demonstration.

- **Essays & Reading**  
  - Collected essays and commentary on data serialization formats and the role of CSV.

- **Standards**  
  - References to CSV-related standards and specifications.

- **Conferences & Community**  
  - Mentions conferences and events relevant to CSV, data formats, or related tooling.

- **Meta: Other Lists**  
  - Links to other similar or complementary “awesome” lists in the data/tooling ecosystem.

- **Repository Extras**  
  - `awesomecsv.csv` file enumerating the list entries in CSV form.  
  - `README.md.generated` and `README.md.ntp` used to generate the primary README.  
  - `code-of-conduct.md` defining contribution behavior expectations.  
  - `contributing.md` with guidelines for proposing additions or changes.

- **Open Source and Community-Driven**  
  - Public GitHub repository accepting contributions via pull requests.  
  - Includes a formal Code of Conduct and contribution guidelines.
