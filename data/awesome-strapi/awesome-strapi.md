## Overview

Awesome Strapi provides resources for Strapi, the most popular Node.js headless CMS. Strapi enables developers to build powerful APIs quickly with a customizable admin panel, while giving content teams a user-friendly interface.

## Features

- Official Strapi documentation and tutorials
- Strapi plugins ecosystem (GraphQL, i18n, users-permissions)
- Content Types Builder and custom fields
- REST and GraphQL API generation
- Authentication and authorization (JWT, providers)
- Media library and upload handling
- Role-based access control (RBAC)
- Webhooks and lifecycle hooks
- Frontend integrations (Next.js, Gatsby, Nuxt)
- Database support (PostgreSQL, MySQL, SQLite, MongoDB)
- Deployment guides (Heroku, AWS, Vercel)
- Custom plugin development
- Email providers integration
- Internationalization (i18n)
- Performance optimization
- Migration between Strapi versions
- Community starters and templates
- Video tutorials and courses
