# Awesome DevOps

A curated GitHub collection of DevOps platforms, tools, practices, and learning resources for building and improving DevOps and SRE capabilities in an organization.

- **Website:** http://awesome-devops.xyz
- **Repository:** https://github.com/wmariuss/awesome-devops
- **Category:** Themed directories
- **Tags:** devops, awesome-lists
- **License:** CC0-1.0

## Description
Awesome DevOps is a community-maintained, GitHub-hosted "awesome list" that aggregates links to DevOps-related tools, platforms, practices, and educational resources. It is designed as a reference directory to help teams discover and explore technologies and approaches used in modern DevOps and Site Reliability Engineering (SRE).

## Features

- **Curated DevOps directory**
  - Organized list of DevOps platforms and tools.
  - Focus on resources that support building and improving DevOps culture and SRE teams.

- **Covers multiple DevOps dimensions**
  - Tools and technologies (e.g., platforms, services, utilities) used in DevOps workflows.
  - Practices and methodologies relevant to DevOps and SRE.
  - Learning and educational resources for individuals and teams.

- **GitHub-based collection**
  - Hosted as a public GitHub repository (`wmariuss/awesome-devops`).
  - Standard project files such as `README.md`, `CONTRIBUTING.md`, and `LICENSE` for transparency and collaboration.

- **Documentation site**
  - Backed by a documentation configuration (`mkdocs.yml`) and `docs` directory for a browsable website at awesome-devops.xyz.

- **Community contributions**
  - Guidelines for contributions available via `CONTRIBUTING.md`.
  - Uses `CODEOWNERS` for maintainership and review structure.

- **Open licensing**
  - Published under the CC0-1.0 license, allowing broad reuse of the curated list.
