# Awesome Pinned Gists

**Brand:** GitHub  
**Category:** Themed Directories  
**Tags:** github, developer-tools, awesome-lists

## Description
Awesome Pinned Gists is a curated GitHub directory of open source tools and resources for creating dynamic, auto-updating pinned gists on GitHub profiles. It focuses on showcasing projects, activity, and other data-driven content inside pinned gists.

**Source:** https://github.com/matchai/awesome-pinned-gists#readme

## Features
- **Curated awesome list**
  - Catalog of open source projects specifically for dynamic pinned gists.
  - Follows the "Awesome" list conventions and is listed on awesome.re.

- **Organized by data source / use case**
  - **External Services**
    - Tools to display data from external services in a pinned gist (e.g., third-party APIs, external activity feeds).
  - **GitHub**
    - Tools for displaying enriched GitHub-related data in pinned gists (e.g., stats, activity, repositories information).
  - **User Defined**
    - Tools that render user-defined content configured via GitHub Actions or input parameters.
    - Example entries:
      - **date-box** – Updates a pinned gist to show anniversaries and countdowns (days or hours).
      - **image2grid** – Generates grid-like images/GIFs suitable for embedding in a GitHub README (and, by extension, related profile visuals).

- **Resources for automation**
  - "Resources" section listing tools and references to keep pinned gists automatically updated (commonly via GitHub Actions or scheduled jobs).

- **Open collaboration**
  - Clear contribution guidelines for adding new tools and improving the list (via `contributing.md`).
  - Code of Conduct defined in `code-of-conduct.md`.

- **License**
  - Published under CC0 (public domain dedication), allowing broad reuse and adaptation of the list content.
