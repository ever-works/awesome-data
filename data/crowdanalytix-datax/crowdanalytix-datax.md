# CrowdANALYTIX dataX

**Category:** Data Competitions  
**Website/Source:** https://github.com/kyrcha/datasets-directory

## Description
CrowdANALYTIX dataX is an online competition platform that hosts data science and analytics challenges. It provides real-world datasets that can be used by practitioners to develop models and by enterprises to crowdsource analytics solutions.

## Features
- Data science and analytics competition platform
- Hosts challenges based on real-world datasets
- Targets both individual practitioners and enterprise users
- Focus on applied machine learning and analytics tasks
