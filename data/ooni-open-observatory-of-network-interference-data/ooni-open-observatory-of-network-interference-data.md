## OONI Open Observatory of Network Interference Data

**URL:** https://ooni.torproject.org/data/
**Category:** Themed Directories  
**Tags:** networking, censorship, open-data

### Overview
The OONI Open Observatory of Network Interference Data is an open data portal from OONI (part of the Tor Project) providing one of the world’s largest open datasets on internet censorship and network interference. It aggregates measurements collected globally to document how access to information is blocked or manipulated.

### Features
- **OONI Explorer**  
  - Web interface to explore OONI’s open dataset on internet censorship.  
  - Access to measurement data from around the world.  
  - Accompanied by a user guide explaining how to search, filter, and interpret data.

- **OONI API**  
  - Programmatic access to OONI measurement data.  
  - Supports custom and large-scale analysis workflows.  
  - Suitable for integrating censorship measurement data into external tools and research pipelines.

- **Batch Data Access & Documentation**  
  - Documentation for batch consumption of OONI data.  
  - Guidance on available data formats, endpoints, and recommended access methods.

- **Analysis Tutorials**  
  - Example Jupyter notebook demonstrating analysis of OONI website measurement data (e.g., a Russia-focused case study).  
  - Downloadable notebook file for local use and adaptation.

- **Open License**  
  - Content and data made available under a Creative Commons license, enabling reuse with attribution.
