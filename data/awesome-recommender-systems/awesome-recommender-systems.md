## Overview

Awesome Recommender Systems provides comprehensive resources on recommendation algorithms, from classical collaborative filtering to modern deep learning approaches.

## Classical Methods

### Collaborative Filtering
- User-based CF: Find similar users
- Item-based CF: Amazon's approach, find similar items
- Model-based CF: Matrix factorization, SVD

### Content-Based Filtering
- Feature extraction from items
- User preference modeling
- Similarity computation

## Matrix Factorization

- SVD (Singular Value Decomposition)
- NMF (Non-negative Matrix Factorization)
- ALS (Alternating Least Squares)
- Implicit feedback methods

## Deep Learning Methods

### Neural Collaborative Filtering
- Multi-layer perceptron for user-item interaction
- Generalized matrix factorization
- Neural matrix factorization

### Sequential Models
- RNN/LSTM for session-based recommendations
- Transformers for sequences
- Self-attention mechanisms

### Graph Neural Networks
- User-item graph modeling
- Social network information
- Knowledge graph integration

## Key Papers

- Amazon's item-to-item collaborative filtering
- Netflix Prize algorithms
- YouTube recommendation system
- Pinterest visual recommendations

## Evaluation Metrics

- Precision, Recall, F1
- NDCG (Normalized Discounted Cumulative Gain)
- MAP (Mean Average Precision)
- Hit Rate
- Coverage, Diversity, Novelty
