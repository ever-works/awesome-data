## Overview

Awesome Distributed System Projects showcases real-world distributed systems implementations across various programming languages. Learn from production-grade projects how to build scalable, resilient distributed applications.

## Featured Projects

### Dapr
Portable, event-driven runtime for building microservices across cloud and edge with built-in state management, pub/sub, and service invocation.

### Service Weaver
Framework from Google for writing applications as modular binaries and deploying them as microservices.

### Temporal
Durable execution platform for building reliable distributed applications with workflow orchestration.

### Apache Kafka
Distributed streaming platform for building real-time data pipelines and applications.

### etcd
Distributed key-value store for distributed systems coordination from CoreOS/Kubernetes.

## Key Concepts

- **Consensus**: Raft, Paxos implementations
- **Consistency**: CAP theorem, eventual consistency
- **Fault Tolerance**: Replication, redundancy
- **Scalability**: Horizontal scaling, sharding
- **Communication**: RPC, message queues, event streaming
