# Awesome Analytics

**URL:** https://github.com/0xnr/awesome-analytics#readme  
**Category:** Themed Directories  
**Tags:** analytics, data-science

## Overview
Awesome Analytics is a curated, open-source directory of analytics platforms, frameworks, software, and related resources. It aggregates tools and services for tracking, measuring, and analyzing data across different use cases and technology stacks.

Maintained on GitHub, it is part of the broader "awesome" list ecosystem and focuses specifically on analytics tooling and resources.

## Features
- **Curated directory of analytics tools**  
  A maintained list of analytics-related frameworks, platforms, software, and services.

- **Broad coverage of analytics domains**  
  Tools are organized into multiple categories, including (as visible from the content):
  - General analytics
  - Real-time analytics
  - Website analytics
  - API/endpoint analytics
  - SEO analytics
  - Privacy-focused analytics
  - Heatmap analytics
  - Analytics layers (customer data/analytics infrastructure layers)
  - Mobile-focused analytics
  - App store analytics
  - Attribution tracking
  - Social media analytics
  - Developer analytics

- **Mix of self-hosted and SaaS options**  
  The list includes both:
  - Self-hosted/open-source analytics solutions
  - Hosted SaaS analytics products

- **Multi-language and multi-stack coverage**  
  Tools span various languages and ecosystems, such as:
  - PHP, Python, Java, Go, Scala, Perl, JavaScript/Node.js, TypeScript, Elixir, Crystal, Angular, C, and more
  - Some entries indicate containerized or cloud-native options (e.g., Go/Docker, Docker/Node.js).

- **License visibility**  
  Many tools in the list are annotated with their software licenses, such as:
  - MIT
  - Apache-2.0
  - GPL-2.0 / GPL-3.0
  - AGPL-3.0
  - EUPL-1.2
  This helps users quickly understand open-source licensing constraints.

- **Focus on different analytic needs**  
  Includes specialized categories for:
  - Real-time data processing and analytics
  - Privacy-focused and self-hosted analytics (for data protection/compliance needs)
  - Heatmaps for user behavior tracking
  - SEO and marketing-focused analytics
  - Mobile and app store analytics for mobile apps and app marketplaces
  - Attribution tracking for marketing and campaign measurement
  - Social media analytics covering insights and performance metrics

- **Part of the "Awesome" ecosystem**  
  Inspired by `awesome-bigdata` and linked as an “Awesome” list itself, following the familiar structure of other community-maintained awesome lists.

- **Community-maintained**  
  Contributions and feedback from the community are welcomed via GitHub pull requests and issues.

## Use Cases
- Discovering analytics tools for web, mobile, or backend services.
- Comparing self-hosted vs. SaaS analytics options.
- Finding privacy-focused or compliant analytics alternatives.
- Exploring specialized analytics (SEO, mobile, heatmaps, attribution, social media).
- Researching open-source analytics projects by language or license.
