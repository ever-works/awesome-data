## Overview
Awesome Haxe Game Development is a curated, community-maintained list of libraries, engines, tools, and learning resources for building games with the Haxe 4 programming language. It focuses on cross‑platform game development, covering web, mobile, desktop, and consoles.

## Features

### Curated Game Engines
A categorized list of Haxe 4–compatible game engines, typically indicating their supported platforms, such as:
- Web
- Mobile
- Desktop
- Consoles
- Unity integration (where applicable)

### Low-level Engines & Frameworks
Resources for building custom or lower-level engines, with notes on platform coverage such as:
- Web
- Mobile
- Desktop
- Consoles
- Nintendo 3DS (where available)

### Physics Libraries
Collections of physics engines and libraries usable from Haxe for game development, intended for:
- Collision detection and response
- Rigid body dynamics
- 2D/3D physics simulation

### Architecture & Design Patterns
Haxe-focused resources and libraries illustrating common game architecture patterns, including:
- IoC (Inversion of Control)
- EC (Entity Component)
- ECS (Entity-Component-System)
- FSM (Finite State Machine)
- MVC (Model View Controller)

### Networking
Networking-related tools and libraries suitable for Haxe games, such as:
- Client–server communication
- Multiplayer syncing
- Protocol handling

### Serialization & Storage
Libraries and utilities to handle:
- Data serialization (save data, config, assets metadata)
- Persistent storage across platforms

### Example Games
A showcase of released games built with Haxe, often tagged by platform and illustrated with screenshots, for example:
- Desktop-only titles
- Desktop + Console releases

These real-world examples help developers see how Haxe is used in commercial and indie games.

### Community & Contribution

- Open to community contributions to keep the list current and comprehensive.
