# Awesome Meteor

**Website:** https://github.com/Urigo/awesome-meteor#readme  
**Category:** Themed Directories  
**Tags:** JavaScript, Awesome Lists, Meteor, Full‑stack

## Overview
Awesome Meteor is a curated, community-driven directory of packages, libraries, tools, tutorials, and other resources for building full‑stack JavaScript applications with the Meteor framework.

## Features
- **Curated Meteor ecosystem directory**
  - Community-maintained list focused on Meteor-specific tools and resources
  - Covers both client and server aspects of full‑stack Meteor apps

- **Getting Started resources**
  - Links and materials to help developers begin working with Meteor

- **Collections**
  - Helpers and extensions for working with Meteor collections

- **REST**
  - Packages and tools providing REST support in Meteor applications

- **Forms and Templates**
  - Helpers for Blaze templates and form handling
  - Includes schema-related utilities such as `simpl-schema`

- **Users and Authentication**
  - Tools for managing users, authentication flows, and related security concerns

- **Administration**
  - Tools for administrating and managing Meteor applications

- **Monitoring**
  - Solutions for monitoring app health, performance, and behavior

- **Performance**
  - Tools aimed at optimizing and speeding up Meteor applications

- **Deployment**
  - Resources and tooling for deploying and maintaining Meteor apps in production

- **Docker Images**
  - Docker images and related resources for containerizing Meteor apps

- **Routers**
  - Routing solutions for Blaze-based Meteor apps
  - Includes routers like `flow-router`

- **Offline Support**
  - Tools that add or improve offline capabilities in Meteor

- **Testing**
  - Testing frameworks, libraries, and utilities for Meteor projects

- **SEO (Search Engine Optimization)**
  - Tools to improve SEO for Meteor-based web applications

- **Files**
  - Libraries for file handling and file management within Meteor

- **Search, Sort, and Pagination**
  - Tools for implementing search, sorting, and pagination in Meteor apps

- **Mobile Development**
  - Resources for building and integrating mobile experiences with Meteor

- **Data Visualization**
  - Charts, maps, tables, and other visualization tools compatible with Meteor

- **Analytics**
  - Analytics integrations and tracking tools

- **Cron Jobs and Scheduling**
  - Tools for running scheduled jobs, including timer utilities like `setTimeout` and `setInterval` in a Meteor context

- **Debugging Tools**
  - Tooling to help debug and inspect Meteor applications

- **Editor Plugins**
  - Plugins and extensions for code editors to improve Meteor development workflow

- **Scaffolding**
  - Generators and scaffolding tools for bootstrapping Meteor projects

- **Tooling**
  - Auxiliary tools that support development, build, and maintenance tasks

- **Boilerplate**
  - Starter and boilerplate Meteor application templates

- **Open Source Apps**
  - Example and reference Meteor applications with source code

- **Internationalization**
  - i18n and localization tools for multilingual Meteor apps

- **Front End Frameworks**
  - Integrations with alternative front-end frameworks beyond Blaze

- **Alternative Databases**
  - Tools and adapters for using databases other than MongoDB with Meteor

- **Resource Discovery**
  - **Books:** Curated list of books related to Meteor
  - **Courses:** Free and paid Meteor courses
  - **Tutorials:** Step-by-step guides and how-tos
  - **Blogs:** Ongoing written resources about Meteor development
  - **Websites:** External sites aggregating or publishing Meteor content
  - **Q&A:** Question-and-answer resources relevant to Meteor
  - **Community Newsletters:** Periodic community updates and news
  - **Social:** Social channels where the Meteor community interacts
  - **Work Opportunities:** Job and freelance opportunities involving Meteor

- **Built With Meteor**
  - Examples of commercial-grade applications built using Meteor

- **Deprecated Resources Section**
  - Dedicated area for items no longer compatible with current Meteor versions, helping users avoid outdated tools

- **Contribution Guidelines**
  - Clear CONTRIBUTING documentation for community members to submit new resources or updates
