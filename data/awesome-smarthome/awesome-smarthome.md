## Overview

Awesome SmartHome is a curated list of awesome SmartHome/Home Automation things focusing on open technologies and solutions that leave users in control. This list emphasizes self-hosted, privacy-respecting solutions.

## Complete Self-Hosted Systems

- **Home Assistant** (39,187 stars) - Home automation platform running on Python 3, supports 3000+ services with localized automation
- **openHAB** (3,484 stars) - Modular platform with over 220 supported device integrations
- **Domoticz** (2,831 stars) - Lightweight home automation system
- **Gladys** - Privacy-first home automation platform
- **pimatic** - Home automation framework
- **FHEM** - Perl-based automation system

## Features

- Complete, self-hosted solutions with frontend interfaces
- Suitable for end users
- Focus on open-source and user control
- Support for thousands of devices and services
- Cross-platform compatibility

## Hardware Support

- ESP8266/ESP32 microcontrollers
- Raspberry Pi and similar single-board computers
- Z-Wave and Zigbee devices
- MQTT-enabled sensors and actuators

## Community

Home Assistant alone boasts more than 62,000 active contributors, demonstrating a vibrant and growing community around open home automation.
