## Overview

Awesome UI Frameworks provides a curated list of UI frameworks and component libraries for building beautiful, accessible, and responsive user interfaces. The collection covers frameworks for React, Vue, Svelte, Angular, and vanilla JavaScript.

## Features

- React UI libraries (Material UI, Chakra UI, Ant Design, Mantine)
- Vue component libraries (Vuetify, Element Plus, PrimeVue)
- Angular UI frameworks (Angular Material, PrimeNG, Clarity)
- Svelte component libraries (Carbon Components, Attractions)
- Framework-agnostic UI kits (Shoelace, Web Components)
- Headless UI libraries (Radix UI, Headless UI, Ark UI)
- Design system tools and documentation generators
- CSS-in-JS solutions and styling frameworks
- Accessibility-focused component libraries
- Dark mode and theming support
- Animation and transition libraries
- Form and data table components
- Chart and data visualization libraries
- Icon libraries and icon systems
- Mobile-first responsive design tools
- Component playground and preview tools
- Design tokens and style guides
- Migration guides between frameworks

## Pricing

Free and open-source community resource covering mostly MIT-licensed libraries with some commercial options.
