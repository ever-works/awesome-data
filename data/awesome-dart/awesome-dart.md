# Awesome Dart

**Website:** https://github.com/yissachar/awesome-dart#readme  
**Category:** Themed directories  
**Tags:** awesome-lists, dart, developer-tools

## Overview
Awesome Dart is a curated directory of Dart frameworks, libraries, tools, and related software. It focuses on projects that are actively maintained, well documented, and widely used within the Dart community. The list follows the general style and standards of the broader “awesome” lists ecosystem.

## Features
- **Curated Dart ecosystem directory**  
  - Focus on quality Dart packages, tools, and projects.  
  - Entries are selected based on maintenance status, documentation quality, and community popularity.

- **Organized by topic/category**  
  The list is structured into multiple sections to make discovery easier:
  - Client Web App Frameworks  
  - Server Frameworks  
  - Web Frameworks  
  - Other Frameworks  
  - Cross-platform development  
  - Game Development  
  - Animation  
  - Template  
  - Database  
  - Package Managers  
  - Utilities  
  - Dependency Injection  
  - Parsers  
  - Validation  
  - ORM  
  - Image  
  - Testing  
  - Unions  
  - Crash monitoring  
  - Tools  
  - Multithreading  
  - Tutorials  
  - Community  
  - IDEs, Editors, and Plugins  
  - Everything Else

- **Ecosystem discovery support**  
  - Encourages users to search additional packages on Pub (https://pub.dartlang.org/) when something is not found in the list.  
  - Suggests contributing newly discovered high-quality packages back to the list.

- **Contribution guidelines**  
  - Documented process for proposing additions, removals, or improvements via pull requests.  
  - Guidance on assessing whether a package is a good fit (e.g., active maintenance, quality).  
  - Public contributor list and contribution history managed via GitHub.

- **Open content licensing**  
  - Licensed under CC0 (Creative Commons Zero), allowing reuse, modification, and redistribution without restriction.
