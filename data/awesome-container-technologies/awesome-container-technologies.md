## Overview

Comprehensive list of container technologies, runtimes, and related tools for building and managing containerized applications.

## Container Runtimes

- Docker - Industry standard
- containerd - CNCF runtime
- CRI-O - Kubernetes runtime
- Podman - Daemonless containers
- LXC/LXD - System containers

## Orchestration

- Kubernetes - De facto standard
- Docker Swarm - Docker native
- Nomad - HashiCorp orchestrator
- Apache Mesos - Cluster manager

## Build Tools

- Buildah - OCI image builder
- Kaniko - In-cluster image building
- BuildKit - Modern build toolkit
- Jib - Java containerization

## Image Management

- Harbor - Enterprise registry
- Quay - Red Hat registry
- Nexus - Artifact repository
- JFrog Artifactory - Universal

## Security

- Trivy - Vulnerability scanner
- Clair - Static analysis
- Anchore - Policy enforcement
- Falco - Runtime security

## Networking

- CNI plugins
- Calico - Network policy
- Flannel - Simple overlay
- Weave - Container networking

## Pricing

Mostly free and open-source.
