## Description
Colorful is an Awesome-style curated collection of resources to help designers and developers choose and explore color schemes, palettes, and related color tools.

## Features
- Curated links to color scheme and palette generators
- Collections of tools for exploring and manipulating colors
- Resources focused on color usage in design and CSS
- Organized in the familiar Awesome-list style for easy browsing
