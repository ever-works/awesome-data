## Overview

A comprehensive curated list of awesome chatbot services and resources. Covers frameworks, platforms, and tools for building text and voice-based conversational interfaces.

## Open Source Bot Building Frameworks

### Multi-Platform Frameworks

- **Botkit** - Build chatbots for Slack, Facebook Messenger, Twilio, and Microsoft platforms
- **Rasa** - Open source machine learning framework for automated text and voice conversations
- **Botpress** - Open-source conversational AI platform
- **ChatterBot** - Machine-learning based conversational dialog engine

### JavaScript/Node.js

- **Claudia Bot Builder** - Create chatbots for FB, Slack, Skype and Telegram
- **BotUI** - JavaScript framework for creating conversational UIs
- **Bottender** - Modern framework for building conversational apps

### Python

- **python-telegram-bot** - Telegram Bot API wrapper
- **slackbot** - Simple Slack bot framework
- **discord.py** - Discord API wrapper
