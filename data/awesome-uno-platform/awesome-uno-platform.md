## Overview

Awesome Uno Platform is a curated, community-maintained list of resources for developers building cross‑platform .NET applications with the Uno Platform UI framework. It aggregates learning materials, sample projects, libraries, and tools related to Uno Platform development for Windows, iOS, Android, WebAssembly, macOS, and Linux.

## Features

- **Curated resource list for Uno Platform**  
  A structured, maintained collection of links and references specifically for the Uno Platform ecosystem.

- **Coverage of multiple resource types**
  - **General**: Introductory and high-level resources about Uno Platform and cross‑platform .NET UI development.
  - **Repositories**: GitHub and other code repositories relevant to Uno Platform (samples, components, utilities).
  - **Tools**: Development tools, extensions, and utilities that improve productivity when working with Uno Platform.
  - **Libraries**:
    - **Toolkits**: UI and component toolkits tailored for Uno Platform.
    - **MVVM**: Libraries and frameworks that support MVVM patterns in Uno applications.
    - **Compatibility**: Helpers and shims to enhance compatibility across target platforms.
  - **Blogs**: Articles and blog posts focused on Uno Platform features, patterns, and best practices.
  - **Live apps**: Examples of production or publicly available applications built with Uno Platform.
  - **Open‑source apps**: Open‑source Uno‑based applications that can be studied or adapted.
  - **Live coding**: Streams or recorded sessions showing live Uno Platform development.
  - **Talks**: Conference and meetup talks related to Uno Platform.
  - **Videos**: Video tutorials, walkthroughs, and presentations.
  - **Books**: Longer‑form learning resources (books, e‑books) on Uno Platform or closely related topics.

- **Cross‑platform focus**  
  All resources are oriented around building single‑codebase .NET apps that run on Windows, iOS, Android, WebAssembly, macOS, and Linux using Uno Platform.

- **Open source and community‑driven**
  - Hosted as a public GitHub repository.
  - Open contributions via pull requests.
  - Governed by a documented code of conduct.

- **Structured documentation & governance**
  - `readme.md` as the main curated list.
  - `contributing.md` providing detailed contribution guidelines.
  - `code-of-conduct.md` defining community behavior standards.
  - Explicit open‑source license (`LICENSE` file) for the repository content.
