# Awesome Directus

**Website:** https://github.com/directus-community/awesome-directus#readme  
**Category:** Themed Directories  
**Tags:** awesome-lists, apis, database

## Overview
Awesome Directus is a curated, community-maintained collection of extensions, tools, and ecosystem projects built for or around Directus, a real-time API and dashboard for SQL database content.

## Features
- Curated list of Directus-related resources and projects.
- Focus on extensions that enhance or extend Directus functionality.
- Includes tools that integrate with or support Directus-based workflows.
- Covers broader ecosystem projects built on top of the Directus platform.
- Organized as an "awesome list" for quick discovery and exploration of the Directus ecosystem.
