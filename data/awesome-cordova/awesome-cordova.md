# Awesome Cordova

**Website:** <https://github.com/busterc/awesome-cordova#readme>

## Overview
Awesome Cordova is a community‑maintained, curated list of Apache Cordova plugins, tools, libraries, and learning resources for building JavaScript‑based hybrid mobile applications.

## Features
- **Curated Cordova ecosystem list**
  - Focused on Apache Cordova and hybrid mobile app development
  - Highlights “amazingly awesome” libraries, resources, and tools
- **Organized by topic** (sections in the list include):
  - **FAQ** – common questions and answers about Cordova usage and ecosystem
  - **Official** – links to official Cordova resources and documentation
  - **Learning** – tutorials, articles, and educational materials for getting started and going deeper with Cordova
  - **Plugins** – notable Cordova plugins to extend app functionality
  - **UI Frameworks** – UI kits and frameworks suitable for Cordova apps
    - **Pay to Use** – commercial / paid UI framework options
    - **Outdated** – older UI frameworks identified as no longer current
  - **Icons** – resources for app icons and related assets
  - **Tools** – build tools, CLIs, utilities, and other supporting software
- **Cross‑linking with other awesome lists**
  - References the broader “awesome-awesomeness” index of curated lists
- **Community contributions**
  - Contributions are explicitly welcomed via pull requests on GitHub
  - Uses common awesome‑list conventions (README, contribution guidelines implied)
