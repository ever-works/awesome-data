# Awesome Games of Coding

**Website:** https://github.com/michelpereira/awesome-games-of-coding#readme  
**Category:** Themed Directories  
**Tags:** games, education, programming

## Description
Awesome Games of Coding is an open, curated directory of games and interactive resources designed to teach programming concepts and specific programming languages through gameplay.

## Features
- **Curated game list** focused on learning programming through games and interactive tools.
- **Language-based organization**, including sections for:
  - Language Independent resources
  - Python
  - Java
  - JavaScript
  - C# (C Sharp)
  - Ruby
  - Assembly
  - Scala
  - Miscellaneous (other or mixed-language resources)
- **Educational focus** on helping users learn how to program or improve coding skills via game mechanics.
- **Open-source repository** hosted on GitHub.
- **Contribution guidelines** documented in a `contributing.md` file for adding or improving entries.
- **Code of conduct** outlining expected behavior for contributors.
- **Clear licensing** via a `LICENSE` file for reuse and contributions.
