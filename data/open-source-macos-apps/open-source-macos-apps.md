## Overview
Open Source macOS Apps is a curated, GitHub-hosted directory of open source applications for macOS. It collects and organizes macOS apps that have publicly available source code, presenting them in an “awesome list” style for easy discovery.

## Features
- **Curated listing of macOS apps** – Focused specifically on applications that run on macOS.
- **Open source only** – Every listed application provides access to its source code.
- **Awesome-list format** – Structured similarly to popular “awesome” GitHub lists for familiarity and ease of browsing.
- **Community-driven** – Maintained and expanded through contributions from many GitHub users (hundreds of contributors).
- **Hosted on GitHub** – Uses a public repository, enabling cloning, forking, and issue tracking.
- **Telegram updates channel** – Linked channel (`opensourcemacosapps`) for announcements and updates about the list.
- **Contributions welcome** – Clear invitation for the community to suggest or add new apps via standard GitHub workflows.
