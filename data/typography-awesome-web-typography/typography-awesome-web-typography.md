# Typography (Awesome Web Typography)

**Website:** https://github.com/deanhume/typography#readme  
**Category:** Themed Directories  
**Tags:** typography, web, design

## Overview
Typography (Awesome Web Typography) is a curated collection of resources focused on improving typography on the web. It aggregates articles, tools, books, talks, and other materials related to web fonts, performance, and typographic design.

## Features

- **Curated Article Collection**  
  - Articles specifically focused on web typography practices and principles.

- **Web Fonts & Performance Resources**  
  - Materials focused on the interaction between web fonts and site performance.

- **Tools Directory**  
  - Links to tools that assist with web typography (e.g., font selection, implementation, and optimization tools).

- **Books**  
  - Recommended books covering typography and web typography.

- **Font Combinations**  
  - Resources and references on pairing and combining typefaces effectively for the web.

- **Talks**  
  - Recorded talks or presentations on typography and web font usage.

- **Featured Section**  
  - Highlighted typography-related examples or projects (e.g., visual showcases like the Google I/O Device Wall image).
