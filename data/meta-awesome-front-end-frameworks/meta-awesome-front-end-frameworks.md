# meta-awesome: Front End Frameworks

**Category:** Meta Directories  
**Slug:** meta-awesome-front-end-frameworks  
**Tags:** meta-directory, frontend

## Overview
meta-awesome: Front End Frameworks is a curated meta-list that aggregates “awesome” collections focused on front-end frameworks, UI libraries, and client-side technology stacks. It serves as an entry point to discover multiple specialized awesome lists related to front-end development.

## Features
- **Meta-directory of awesome lists**
  - Collects links to multiple “awesome” GitHub lists focused on front-end frameworks and related topics.
- **Front-end framework focus**
  - Concentrates on frameworks and stacks used on the client side (e.g., UI frameworks, SPA frameworks, component libraries).
- **UI libraries coverage**
  - Includes collections about UI component libraries, design systems, and presentation-layer tooling.
- **Client-side stack resources**
  - Points to lists that cover broader front-end stacks (tooling, build systems, etc.) alongside frameworks.
- **Part of the meta-awesome project**
  - Integrated into the wider `meta-awesome` structure, which organizes awesome lists by domain.
- **Single documentation entry**
  - Exposed as a focused markdown document at:
    - `doc/front-end-frameworks.md` in the `meta-awesome` GitHub repository.

## Usage
- Use as a starting index to discover specialized awesome lists about:
  - Specific front-end frameworks
  - UI component and design libraries
  - Client-side stacks and related tooling
