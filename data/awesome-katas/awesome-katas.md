## Overview
Awesome Katas is a curated directory of programming kata exercises and related resources. It aggregates practice problems ranging from general coding exercises to more complex algorithms and real‑world scenarios, suitable for deliberate practice in any programming language.

## Features
- **Curated kata collection**
  - Aggregates code kata exercises discovered across the internet and the GitHub community.
  - Includes links to well-known kata resources and repositories (e.g., CodeKata, Wonderland Clojure Katas, SensioLabs PoleDev Katas, Gaurav Arora’s TDD Katas Collection).

- **Focus on deliberate practice**
  - Emphasizes repeated practice to improve programming skills.
  - Encourages not just solving problems but aiming for high-quality, idiomatic solutions.
  - Promotes using best practices of your chosen programming language.

- **Broad difficulty and domain coverage**
  - Exercises range from general coding katas to more complex algorithmic problems.
  - Includes katas inspired by real-life scenarios.

- **Language-agnostic approach**
  - Designed to be solved in any programming language the user prefers.

- **Open contribution model**
  - Public contribution guidelines (`CONTRIBUTING.md`) describe how to propose new katas or resources.

- **Open content license**
  - Published under the CC0 (public domain) license, allowing free use, adaptation, and sharing.
