# Awesome Software Patreons

**Category:** Themed Directories  
**Tags:** open-source, funding, community

## Overview
Awesome Software Patreons is a curated, community-maintained directory of programmers and software projects that accept ongoing financial support (e.g., Patreon, GitHub Sponsors, OpenCollective, Liberapay, project-specific donation pages). It focuses on open source software and related creators, helping users discover projects and individuals they can fund.

## Features

- **Curated list of software-related creators and projects**  
  - Emphasis on individual programmers and open source projects.  
  - All entries have at least one linked funding/sponsorship option.

- **Organization by type of project or creator**  
  - **Open Source Projects** (all released under an open source license), further divided into:
    - **Libraries**
    - **Operating Systems**
    - **Games**
    - **Open Source Art**
    - **Hardware related**
  - **People Doing Open Source Work**
  - **Other Kinds of Content**, including:
    - Articles and tutorials
    - Videos
    - Podcasts

- **Multiple funding platforms per entry (where applicable)**  
  - Entries can list Patreon, GitHub Sponsors, OpenCollective, Liberapay, and other funding links.

- **Concrete examples of supported open source projects** (partial list from the directory):
  - **Desktop / Applications / Tools**
    - Akira – Linux application for UI and UX design.  
    - Albert launcher – Keyboard launcher written in C++/Qt.  
    - Calibre – E-books manager.  
    - Deluge – BitTorrent client.  
    - DokuWiki – Wiki application.  
    - Duplicati – Backup client.  
    - Freeter – Productivity app for developers, designers and other professionals.  
    - Homebrew – Package manager for macOS.  
    - iTerm2 – macOS terminal replacement.  
    - JabRef – Bibliography management software.  
    - JHipster – Application platform for creating Spring Boot + Angular/React/Vue projects.  
    - LogSeq – Knowledge management tool.  
    - Memcode – Flashcard platform.

  - **3D / Graphics / Design / Editing**
    - Blender – 3D creation suite.  
    - FreeCAD – Parametric 3D modeler for real-life objects.  
    - Godot Engine – Complete 3D engine.  
    - Laigter – Automatic map generator (normal/specular/occlusion/parallax) for 2D game sprites.  
    - Olive – Video editing and compositing software.  
    - MuseScore – Music notation software.

  - **Development tools & engines**
    - Compiler Explorer – Online decompiler and compiler explorer.  
    - HaxeFlixel – Cross-platform 2D game engine.  
    - NativeScript-Vue – Native mobile apps with NativeScript and Vue.  
    - Neovim – Vim fork focused on extensibility and usability.

  - **Operating systems / filesystems / networking**
    - bcachefs – Linux filesystem.  
    - Firezone – Self-hosted VPN server using WireGuard.

  - **Mobile / app catalogs**
    - F-Droid – Installable catalogue of FOSS applications for Android.  
    - NewPipe – FLOSS video player for services like YouTube, SoundCloud, etc.

  - **Hardware / networking related**
    - Kismet – Wireless network tool.

- **Focus on open source sustainability**  
  - Highlights sponsorship as a practical way to fund ongoing development of widely-used tools and libraries.

## Use Cases

- Discover open source projects and developers to sponsor.  
- Find Patreon and alternative funding pages for tools you already use.  
- Explore creators producing tutorials, videos, or podcasts related to software and open source.

## Pricing

- The directory itself is free to access.  
- Individual projects and creators have their own independent funding pages and pricing tiers (not standardized or listed centrally in this directory).
