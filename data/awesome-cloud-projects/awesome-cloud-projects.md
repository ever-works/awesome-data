## Overview

Awesome Cloud Projects provides over 1100 cloud recipes, projects, and tutorials to gain hands-on experience across AWS, Azure, and GCP.

## Cloud Providers Covered

- **Amazon Web Services (AWS)** - EC2, S3, Lambda, RDS, and more
- **Microsoft Azure** - VMs, Blob Storage, Functions, SQL Database
- **Google Cloud Platform (GCP)** - Compute Engine, Cloud Storage, Cloud Functions

## Infrastructure as Code Formats

- **Terraform** - Multi-cloud IaC tool
- **AWS CloudFormation** - AWS-native IaC
- **AWS CDK** - Cloud Development Kit for AWS
- **Azure Bicep** - Azure-native IaC language
- **Bash Shell Scripts** - Automation scripts
