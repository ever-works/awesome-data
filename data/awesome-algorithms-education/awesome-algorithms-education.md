## Overview
Awesome Algorithms Education is a curated "awesome list" focused on learning and practicing algorithms. It aggregates educational resources such as courses, tutorials, and problem sets to help learners at different levels build algorithmic thinking and skills.

## Features
- **Curated resource directory** for learning and practicing algorithms.
- **Focus on education**, including:
  - Courses
  - Tutorials
  - Problem sets
- **Level-based structure** (as indicated in the contents):
  - Introductory resources
  - Intermediate resources
  - Advanced resources
- **Multi-language documentation**:
  - English README (`readme-en.md`)
  - Korean README (`readme-ko.md`)
- **Clear contribution process** via a dedicated `contributing.md` file for community additions and improvements.
- **Part of the Awesome ecosystem**, following the Awesome list format and standards.
