## Overview

Awesome .NET Core is a community-driven, curated list of libraries, tools, frameworks, and learning resources for .NET Core and modern .NET development. It organizes well-known and widely used ecosystem resources into clear categories to help developers discover technologies and references quickly.

## Features

### Scope & Structure
- Curated list focused on **.NET Core and modern .NET** development.
- Community-driven contributions with documented contribution guidelines.
- Accepts both **open-source** and **proprietary/commercial** software entries.
- Organized into a detailed, categorized table of contents to simplify navigation.
