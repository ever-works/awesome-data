# Awesome Public Datasets – Government: U.S. Food and Drug Administration (FDA)

**Name:** Awesome Public Datasets - Government: U.S. Food and Drug Administration (FDA)  
**Source:** https://open.fda.gov/index.html  
**Category:** Themed directories  
**Tags:** datasets, government, APIs

## Description
An Awesome entry for **openFDA**, the U.S. Food and Drug Administration’s open data initiative. It provides programmatic access to FDA data on drugs, devices, food, and related regulatory information for research, analysis, and application development.

## Features
- **Open-source APIs**  
  - Public REST APIs exposing FDA datasets (e.g., drugs, devices, food, adverse events, and other regulatory data).  
  - Suitable for integration into research workflows, analytics, and applications.

- **API Usage Statistics**  
  - Overview of how API endpoints are being used by the community.  
  - Helps understand popular endpoints and monitor general usage trends.

- **Data Downloads**  
  - Endpoint data available as downloadable **zipped JSON** files.  
  - Facilitates bulk access and offline analysis of FDA datasets.

- **Community Apps & Resources**  
  - Open community space highlighting apps and tools built using openFDA.  
  - Includes shared open source code, examples, and ideas for working with the APIs.

- **Regular Dataset Updates**  
  - New and updated datasets announced via news/updates section.  
  - Recent additions include:
    - **Cosmetic Adverse Events** dataset.  
    - **Complete Response Letters (CRL)** for New Drug Applications (NDAs) and Biologics License Applications (BLAs), including CRLs issued as recently as **2025**.  
    - **Drug Shortages** dataset.
