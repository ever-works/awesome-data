## Overview

Awesome Astronomy provides a comprehensive collection of astronomy resources, including space agency projects, astronomical databases, observation software, datasets, and educational materials for exploring the cosmos.

## Key Categories

- **Space Agencies** - NASA, ESA, JAXA, SpaceX missions and data
- **Astronomical Databases** - Star catalogs, exoplanet databases
- **Observation Software** - Planetarium software, telescope control
- **Data Analysis** - Python libraries for astronomical data
- **Citizen Science** - Public participation astronomy projects
- **Educational Resources** - Courses, tutorials, documentaries
- **Astrophotography** - Capturing images of celestial objects

## Space Agency Resources

### NASA
- Hubble Space Telescope data
- James Webb Space Telescope observations
- Mars rover missions and imagery
- Kepler and TESS exoplanet data
- NASA Open Data Portal

### ESA (European Space Agency)
- Gaia mission star catalog
- Rosetta comet mission data
- Planck cosmic microwave background

### JAXA (Japan Aerospace Exploration Agency)
- Hayabusa asteroid missions
- Akatsuki Venus climate orbiter

## Software & Tools

### Planetarium Software
- **Stellarium** - Free open-source planetarium
- **KStars** - Desktop planetarium for Linux
- **Celestia** - Real-time 3D space simulation
- **Gaia Sky** - 3D astronomy visualization

### Python Libraries
- **Astropy** - Core astronomy library for Python
- **SunPy** - Solar physics data analysis
- **Lightkurve** - Kepler & TESS time series analysis
- **Skyfield** - Positions for stars, planets, satellites

### Data Analysis
- **TOPCAT** - Interactive table and catalog analysis
- **SAOImage DS9** - Astronomical imaging and visualization
- **Aladin** - Interactive sky atlas

## Databases & Catalogs
