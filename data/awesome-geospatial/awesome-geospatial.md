## Overview

Awesome Geospatial provides a comprehensive collection of tools and resources for working with geographic information systems (GIS), spatial data, and mapping applications, from desktop GIS to web mapping and spatial analysis.

## Key Categories

- **Desktop GIS** - QGIS, ArcGIS, GRASS GIS
- **Spatial Databases** - PostGIS, SpatiaLite, GeoPackage
- **Web Mapping** - Leaflet, Mapbox, OpenLayers, Cesium
- **Python Libraries** - GeoPandas, Shapely, Rasterio, Folium
- **Remote Sensing** - Satellite imagery processing and analysis
- **Geocoding** - Address to coordinates conversion
- **Routing** - Pathfinding and navigation

## Desktop GIS Software

### Open Source
- **QGIS** - Free and open-source GIS application
- **GRASS GIS** - Geographic Resources Analysis Support System
- **gvSIG** - Desktop GIS for geospatial data management
- **uDig** - User-friendly Desktop Internet GIS

### Commercial
- **ArcGIS Pro** - Esri's professional GIS software
- **MapInfo Pro** - Desktop mapping and GIS tool
- **Global Mapper** - Affordable GIS application

## Spatial Databases

### PostgreSQL Extensions
- **PostGIS** - Spatial database extender for PostgreSQL
- **pgRouting** - Routing functionality for PostGIS
- **TimescaleDB** - Time-series extension (with PostGIS support)

### File-based
- **GeoPackage** - Open standard for geospatial information
- **SpatiaLite** - SQLite with spatial capabilities
- **Shapefile** - Legacy ESRI format (still widely used)

## Python Geospatial Libraries

### Core Libraries
- **GeoPandas** - Pandas for geospatial data
- **Shapely** - Manipulation of planar geometric objects
- **Fiona** - Reading and writing spatial data files
- **Rasterio** - Raster data I/O and processing
- **Pyproj** - Cartographic projections and coordinate transformations

### Visualization
- **Folium** - Interactive maps with Leaflet
- **Cartopy** - Cartographic projections and matplotlib integration
- **GeoViews** - Geographic data visualization with HoloViews
- **Plotly** - Interactive plotting with map support

### Analysis
- **PySAL** - Spatial analysis library
- **Scikit-learn** - Machine learning on spatial data
- **GeoDjango** - Geographic web framework

## Web Mapping Libraries

### JavaScript Libraries
- **Leaflet** - Leading open-source JavaScript library
- **OpenLayers** - High-performance mapping library
- **Mapbox GL JS** - Vector tile rendering
- **Cesium** - 3D globe and map visualization
- **MapLibre** - Open-source fork of Mapbox GL JS

### Tile Servers
- **OpenMapTiles** - Open-source map tile server
- **TileServer GL** - Vector and raster tiles server
- **GeoServer** - Open-source server for sharing geospatial data
