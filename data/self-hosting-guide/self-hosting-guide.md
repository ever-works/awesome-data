## Overview

Self-Hosting Guide is a complete resource for learning to host and manage software on your own infrastructure, from basics to advanced configurations.

## Features

- **Cloud Services**: Self-hosted cloud storage and sync (Nextcloud, Seafile)
- **LLM Hosting**: Running large language models locally
- **VPN Setup**: WireGuard and OpenVPN configuration
- **Automation**: Home Assistant and automation platforms
- **Networking**: Network configuration and security
- **Media Servers**: Plex, Jellyfin, and streaming services
- **Security**: Firewalls, SSL certificates, authentication
- **Monitoring**: Self-hosted monitoring and alerting
