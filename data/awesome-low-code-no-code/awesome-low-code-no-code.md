## Overview

Awesome Low-Code No-Code is a comprehensive collection of platforms and tools that enable programmers and non-programmers to create application software through graphical user interfaces and configuration instead of traditional code programming.

## Business Application Platforms

- **Budibase**: Open-source no-code/low-code platform for rapidly building internal tools with data connections, automation, and custom UI design
- **Appian**: Enterprise low-code platform for building applications at scale
- **OutSystems**: Full-stack low-code platform for rapid application development
- **Mendix**: Low-code application development platform for enterprises
- **Betty Blocks**: No-code platform for building enterprise applications
- **Lowdefy**: Build internal tools, web apps, admin panels, BI dashboards, and CRUD apps using YAML or JSON

## Workflow Automation

- **n8n**: Fair-code workflow automation tool with 350+ integrations
- **Activepieces**: Open-source business automation tool
- **Make (formerly Integromat)**: Visual platform for workflow automation
- **IFTTT**: Connect different apps and devices with simple conditional statements
- **Zapier**: Connect apps and automate workflows
- **Node-RED**: Flow-based programming tool for wiring together hardware devices, APIs, and online services

## Database and Backend

- **NocoDB**: Open-source Airtable alternative
- **Airtable**: Spreadsheet-database hybrid for organizing work
- **Supabase**: Open-source Firebase alternative
- **Hasura**: Instant GraphQL APIs on databases
- **Directus**: Open-source headless CMS and data platform

## Website and Landing Page Builders

- **Webflow**: Visual web development platform
- **Carrd**: Simple, responsive one-page sites
- **Framer**: Interactive design and prototyping tool
- **Wix**: Website builder with drag-and-drop functionality
- **Squarespace**: All-in-one website building platform

## Mobile App Development

- **Adalo**: Build custom mobile and web apps without code
- **Glide**: Turn spreadsheets into mobile apps
- **FlutterFlow**: Visual builder for Flutter apps
- **Thunkable**: Drag-and-drop mobile app builder

## AI and ML Platforms

- **PraisonAI**: Low-code Multi-AI Agents framework with YAML configuration, 100+ LLM support, MCP integration
- **Obviously AI**: No-code AI tool for building predictive models
- **Lobe**: Free tool to train machine learning models

## Analytics and Dashboards

- **Google Analytics**: Web analytics service
- **Posthog**: Open-source product analytics
- **Metabase**: Open-source business intelligence
- **Retool**: Build internal tools quickly

## Chatbot Builders

- **Botnation**: Create chatbots without coding
- **Chatfuel**: Build Facebook Messenger bots
- **ManyChat**: Chat marketing platform

## Forms and Surveys

- **Typeform**: Interactive forms and surveys
- **Jotform**: Online form builder
- **Google Forms**: Free survey tool

## Benefits of Low-Code/No-Code

- Faster time to market
- Lower development costs
- Empowers non-technical users
- Reduced maintenance overhead
- Easy prototyping and iteration
- Integration with existing systems

## Use Cases

- Internal business tools
- Customer portals
- Workflow automation
- Data management systems
- Landing pages and websites
- Mobile applications
- Chatbots and conversational interfaces
