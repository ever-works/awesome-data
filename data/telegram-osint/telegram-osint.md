# Telegram OSINT

**Brand:** arcx1

## Overview
Telegram OSINT is an open-source, Awesome-style curated directory of open-source intelligence resources specifically focused on Telegram. It is intended for investigators, security researchers, and analysts who need a structured starting point for gathering and exploring Telegram-related OSINT tools and references.

- **Type:** GitHub repository / static website
- **Scope:** Telegram-focused OSINT resources
- **Format:** Curated list presented as an "Awesome" directory
- **License:** WTFPL v2 (per `WTFPLV2.txt`)

## Features
- **Curated OSINT Resource List**  
  Focused collection of Telegram OSINT resources, organized to help users quickly find relevant tools, guides, or references.

- **Awesome-Style Directory**  
  Follows the familiar "Awesome list" convention, making the structure predictable and easy to browse.

- **Telegram-Specific Focus**  
  All items in the list relate to Telegram: investigation, intelligence gathering, analysis, or security research tied to the Telegram platform.

- **Static Website Version**  
  Includes `index.html` and `stylesheet.css`, enabling the curated list to be viewed as a standalone website, not just as a GitHub README.

- **Redistributable Content**  
  Explicitly encourages copying and distributing the website content ("FEEL FREE TO COPY & DISTRIBUTE THIS WEBSITE"), aligning with its permissive licensing.

- **Open Source Repository Structure**  
  - `README.md` containing the main curated list and project description.  
  - `index.html` for web presentation.  
  - `stylesheet.css` for styling.  
  - `WTFPLV2.txt` documenting the license.

- **Intended Audience**  
  Designed for investigators, security researchers, OSINT practitioners, and analysts looking to perform or learn about Telegram-focused OSINT.
