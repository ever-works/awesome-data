# Newspaper Navigator Dataset

Dataset containing extracted visual content from 16,358,041 historic newspaper pages in the Library of Congress’s Chronicling America collection.

## Overview
- **Provider:** Library of Congress Labs (Newspaper Navigator experiment)
- **Type:** Computer vision / image dataset
- **Source collection:** Chronicling America digitized historic newspapers
- **Scale:** Visual content extracted from 16,358,041 newspaper pages
- **Status of related app:** Original Newspaper Navigator web application (visual similarity search) hosted 2020–2025, now retired

## Features
- Extracted visual content from historic newspapers in Chronicling America
- Over 16 million newspaper pages processed
- Images originally used in an experimental web application for:
  - Visual similarity search
  - Browsing historic newspaper images
- Backed by a machine learning pipeline developed as part of the experiment
- Part of the Library of Congress Innovator in Residence program (2020)
- Associated research and resources:
  - Recognized as *Best Digital Humanities Dataset* at the 2020 DH Awards
  - *Best Resource Paper Runner-up* at CIKM 2020

## Use Cases
- Computer vision research on historical imagery
- Digital humanities and historical scholarship
- Visual culture, media, and press history studies
- Machine learning experiments on noisy, real-world historical print data
