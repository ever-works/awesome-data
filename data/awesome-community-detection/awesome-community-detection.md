# Awesome Community Detection

**Website:** https://github.com/benedekrozemberczki/awesome-community-detection  
**Category:** Themed Directories  
**Tags:** graph, research, awesome-lists

## Overview
Awesome Community Detection is a curated directory of research papers on community detection in graphs, along with links to available code implementations. It is organized as an “awesome list” to help researchers, practitioners, and students find methods, references, and tools related to community detection.

## Features
- Curated collection of community detection research papers
- Focus on graph and network analysis
- Links to available implementations of listed methods (where provided by authors or community)
- Organized as an “awesome list” for easy browsing
- Useful for research, experimentation, and comparative study of community detection algorithms
- Open, GitHub-hosted directory that can be extended and contributed to by the community
