# Awesome Game Talks

**Website:** https://github.com/hzoo/awesome-gametalks#readme  
**Category:** Themed Directories  
**Tags:** games, education, media

## Overview
Awesome Game Talks is a curated, “awesome-style” directory of recorded game development talks, presentations, and lectures from conferences and online events. It focuses on game design, development, and related topics, collecting links to videos and resources from multiple sources.

## Features
- **Curated collection of game development talks**  
  - Focus on game development, game design, and related disciplines.  
  - Includes talks, presentations, and lectures.

- **Free GDC talks list**  
  - Dedicated GDC section: a separate `GDC.md` file with a list of free talks from the Game Developers Conference.  
  - Direct reference to the official GDC YouTube channel.

- **Topical sections / Table of Contents**  
  - **GDC Talks** – curated selection from GDC.  
  - **TED Talks (gaming topic)** – links via the TED gaming topic page.  
  - **Other Talks** – non-GDC, non-TED individual talks.  
  - **Panels** – multi-speaker panel discussions about game-related topics.  
  - **Conferences** – talks organized by event or conference.  
  - **Lists** – meta-lists or other curated collections of game-related talks.  
  - **Podcasts** – audio/video podcast content related to game development.  
    - **Dedicated Podcasts** – podcasts focused on game development.  
    - **Talks** – podcast episodes or talk-style content.

- **Community-driven contributions**  
  - Public contribution guidelines (`CONTRIBUTING.md`) describing how to submit new talks or edits.  
  - Maintained as an open GitHub repository to enable collaborative curation.

- **Open source licensing**  
  - Includes a `LICENSE` file specifying the open-source terms for the list itself.
