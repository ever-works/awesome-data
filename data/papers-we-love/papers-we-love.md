# Papers We Love

**Category:** Themed Directories  
**Website:** https://github.com/papers-we-love/papers-we-love#readme

Papers We Love is a community-driven, open repository of academic computer science papers. The collection is organized by topic and sometimes by institution, making it easier to discover and discuss influential and foundational CS research.

---

## Features

- **Academic CS paper directory**
  - Curated list of research papers from the computer science community.
  - Focus on papers that are influential, instructive, or commonly referenced in practice.

- **Topic-based organization**
  - Papers grouped into folders by subject area, for example:
    - `affective_computing`
    - `android`
    - `api_design`
    - `artificial_intelligence`
    - `audio_comp_sci`
    - `biocomputing`
    - `bioinformatics`
    - `brain-computer-interface`
    - `caching`
    - `combinatory_logic`
    - `comp_sci_fundamentals_and_history`
    - `computational_creativity`
    - `computer_architecture`
    - `computer_education`
  - Additional folders (not fully listed in the excerpt) likely cover many other CS subdisciplines.

- **Community-driven repository**
  - Hosted on GitHub under the `papers-we-love` organization.
  - Open to contributions via standard GitHub workflows (issues, pull requests).
  - Maintains a commit history showing ongoing updates and additions.

- **Discussion-focused**
  - Intended as a collection of “papers from the computer science community to read and discuss,” supporting reading groups, meetups, or self-study.

- **Structured as a code repository**
  - Uses GitHub folders and files for navigation.
  - Benefits from GitHub search, version control, and collaboration features.

---

## Pricing

- Access to the repository is free and open-source via GitHub.
