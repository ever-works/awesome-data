## Overview

Jetpack Compose is Android's modern toolkit for building native UI. This list provides resources for learning and mastering Compose.

## Features

### Core Components
- **Material Design 3**: Latest Material Design components
- **Layouts**: Column, Row, Box, ConstraintLayout, and more
- **State Management**: State hoisting, ViewModel integration
- **Navigation**: Type-safe navigation between screens
- **Animations**: Powerful animation APIs
- **Theming**: Dynamic color and custom themes

### Popular Libraries

#### Navigation
- Compose Destinations - Type-safe navigation
- Voyager - Multiplatform navigation library
- Compose Navigation - Official navigation component

#### UI Components
- Accompanist - Supplementary Compose libraries by Google
- Compose Material 3 - Latest Material Design components
- Compose Cupertino - iOS-style components for Compose
- Orbital - Custom animations and transitions

#### State Management
- Compose ViewModel - ViewModel integration
- MVI Kotlin - Model-View-Intent architecture
- Decompose - Lifecycle-aware components

#### Image Loading
- Coil - Image loading library for Compose
- Glide - Compose integration for Glide
- Landscapist - Image loading solutions

#### Charts and Graphs
- Vico - Light and extensible chart library
- Compose Charts - Modern charting library
- YCharts - Declarative chart library

#### Utilities
- Compose Permissions - Runtime permission handling
- Compose Rich Editor - WYSIWYG editor
- Compose Calendar - Calendar components
- Compose Markdown - Markdown rendering
