## Overview

Prompt Engineering Guide by DAIR.AI is one of the most comprehensive and authoritative resources for learning prompt engineering, featuring the latest research, practical guides, and courses.

## Learning Paths

### Beginner Level

- **Introduction to Prompting**: Basic concepts and terminology
- **Prompt Elements**: Components of effective prompts
- **General Tips**: Best practices for beginners
- **Basic Prompting Examples**: Simple use cases
- **Question Answering**: Getting factual information
- **Text Summarization**: Condensing content

### Intermediate Level

- **Few-Shot Prompting**: Learning from examples
- **Chain-of-Thought (CoT)**: Step-by-step reasoning
- **Self-Consistency**: Improving accuracy through sampling
- **Generated Knowledge**: Using LLMs to generate helpful context
- **Automatic Prompt Engineer (APE)**: Automated prompt optimization

### Advanced Techniques

- **Tree of Thoughts (ToT)**: Complex problem-solving framework
- **Retrieval Augmented Generation (RAG)**: Combining prompts with external knowledge
- **Automatic Reasoning and Tool-use (ART)**: Multi-step reasoning with tools
- **ReAct**: Reasoning and Acting framework
- **Reflexion**: Self-reflection for agents
- **Multimodal CoT**: Chain-of-thought with images and text

## Retrieval Augmented Generation (RAG)

### RAG Fundamentals

- **Vector Databases**: Storing and retrieving embeddings
- **Chunking Strategies**: Optimal document segmentation
- **Embedding Models**: Text representation
- **Similarity Search**: Finding relevant context
- **Context Window Management**: Fitting information in prompts

### Advanced RAG

- **Hybrid Search**: Combining dense and sparse retrieval
- **Re-ranking**: Improving retrieval quality
- **Query Transformation**: Enhancing search queries
- **Iterative Retrieval**: Multi-step information gathering
- **Graph RAG**: Using knowledge graphs

### RAG Tools

- **LangChain**: RAG pipelines and components
- **LlamaIndex**: Data framework for LLMs
- **Haystack**: NLP framework with RAG support
- **Weaviate**: Vector database with RAG capabilities
- **Pinecone**: Managed vector database
