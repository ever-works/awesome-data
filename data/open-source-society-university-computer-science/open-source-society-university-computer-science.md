### Open Source Society University – Computer Science

**URL:** https://github.com/ossu/computer-science#readme  
**Category:** Themed Directories  
**Tags:** computer-science, education, courses

---

#### Description
A fully open-source, self-taught computer science curriculum organized as a GitHub repository. It aggregates and sequences free online courses and resources to approximate a full university-level CS education.

---

#### Features
- **Open-source curriculum**
  - Hosted on GitHub under the MIT license
  - Public repo with issues, pull requests, and contribution guidelines (`CONTRIBUTING.md`)
  - Change history documented in `CHANGELOG.md`

- **Structured CS learning path**
  - “Path to a free self-taught education in Computer Science”
  - Follows explicit curricular guidelines (`CURRICULAR_GUIDELINES.md`)
  - Organized as an “awesome-style” list of courses and learning resources

- **Course and resource organization**
  - Dedicated `coursepages` directory for course-specific pages
  - `extras` directory for supplementary materials
  - `images` directory for visual assets embedded in the curriculum

- **Documentation & learner support materials**
  - `FAQ.md` with frequently asked questions about the curriculum
  - `HELP.md` for guidance on how to use or navigate the curriculum

- **Community & collaboration**
  - Large public community (stars and forks indicated on GitHub)
  - Open to contributions via standard GitHub workflows (issues, PRs)

- **Site / presentation structure**
  - Uses Jekyll-style directories: `_includes` and `_layouts` for building the project website
  - Custom domain configuration via `CNAME`
