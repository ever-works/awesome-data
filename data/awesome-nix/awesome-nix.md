# Awesome Nix

**Brand:** nix-community  
**Category:** Themed Directories  
**Tags:** awesome-lists, developer-resources

## Description
Awesome Nix is a curated "awesome list" of packages, tools, documentation, and learning resources for the Nix package manager and the broader Nix/NixOS ecosystem. It aggregates community-maintained links to help users discover Nix-related tools, workflows, and learning materials.

**Source:** https://github.com/nix-community/awesome-nix

## Features
- **Curated Nix ecosystem directory**
  - Focused on high-quality, community-vetted Nix-related resources.
  - Organized as an "awesome" list following the awesome.re conventions.

- **Learning resources**
  - Section dedicated to tutorials, guides, and educational content for learning Nix and NixOS ("Learning").

- **Discovery resources**
  - Section for tools and sites that help discover Nix packages, modules, and configurations ("Discovery").

- **Installation media**
  - Links to resources related to Nix/NixOS installation media.

- **Channel history**
  - Resources for viewing or working with Nix channel history.

- **Deployment tools**
  - Overview of tools supporting deployment workflows using Nix and NixOS.

- **Virtualisation**
  - Resources and tools for virtual machines and virtualization setups powered by Nix.

- **Command-line tools**
  - Highlights key CLI tools in the Nix ecosystem, such as:
    - `nix run`
    - `nix-index`
    - `nix`
    - `nixos-rebuild`
    - `home-manager`
    - `dix`
    - `nix-output-monitor`

- **Development workflows**
  - Links to development-focused Nix tooling and patterns, including:
    - `nix-shell`
    - `mkShell`, `pkgs.mkShell`
    - `flake.nix` and `.nix` configuration patterns
    - Integration examples with tools like `docker-compose`
    - Use of `.gitignore` in Nix-based projects

- **DevOps resources**
  - Section for DevOps-focused tooling and practices based on Nix (CI/CD, infra-as-code, etc.).

- **Programming language ecosystem integration**
  - Subsections for using Nix with various programming languages and their dependency formats, including:
    - **Arduino**
    - **Clojure**
    - **Crystal** (`shard.lock`)
    - **Elm** (`elm.json`)
    - **Gleam**
    - **Haskell** (e.g., `flake-parts`)
    - **Haxe**
    - **Julia**
    - **Lean** (`lake2nix`)
    - **Node.js** (`package.json`, `package-lock.json`)
    - **OCaml**
    - **PHP** (`composer.lock`)
    - **PureScript**
    - **Python** (`poetry.lock`, `uv`)
    - **Ruby**
    - **Rust** (`Cargo.lock`)
    - **Scala**
    - **Zig** (`build.zig.zon`)

- **NixOS modules**
  - Resources around creating, using, and cataloging NixOS modules.

- **NixOS configuration editors**
  - Tools for editing and managing NixOS configurations, including:
    - **Desktop apps**
    - **Web interfaces**

- **Overlays**
  - Resources related to Nix overlays (e.g., examples like `mesa_git`).

- **Distributions**
  - Links to Nix-based or NixOS-related distributions and spins.

- **Community resources**
  - Section for community hubs, chats, forums, and other social resources around Nix.

- **Contribution guidelines**
  - Documented contribution process via `CONTRIBUTING.md` for adding or updating entries.

- **Deprecated items tracking**
  - A `DEPRECATED.md` file to track entries or tools that are no longer recommended or maintained.

- **Open source licensing**
  - Licensed under an open-source license (see `LICENSE` in the repository).
