## Overview

Comprehensive collection of research on 3D generation including various representations like point clouds, voxels, and meshes.

## 3D Representations

- Point clouds
- Voxel grids
- Mesh (vertices, faces)
- Implicit functions (SDF, occupancy)
- Neural radiance fields (NeRF)

## Generation Methods

### Deep Learning Approaches

- GANs for 3D
- VAEs for shapes
- Diffusion models
- Transformers

### Input Modalities

- Text-to-3D
- Image-to-3D
- Sketch-to-3D
- Category-conditional

## Applications

- Asset generation for games
- Product design
- Virtual try-on
- Architecture visualization

## Datasets

- ShapeNet
- ModelNet
- Objaverse
- ScanNet
