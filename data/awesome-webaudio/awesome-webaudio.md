# awesome-webaudio

**Description**  
A curated, community-maintained directory of Web Audio API packages, demos, and learning resources for building web-based audio applications.

**Website**  
https://github.com/notthetup/awesome-webaudio

**Category**  
Themed directories

**Tags**  
`audio` · `javascript` · `awesome-lists`

---

## Features

- **Curated Web Audio focus**  
  - Dedicated to the Web Audio API and related web-based audio development.
  - Inspired by and structured like other “awesome” lists for easy scanning.

- **Organized package index**  
  - **Frameworks** section, including for example:
    - **Tone.js** – framework for building interactive music in the browser.
    - **Bap** – toolkit for making beats and composing sequences, inspired by MPC-style workflows.
    - **Omnitone** – ambisonic spatial audio for the web.
    - **Mach1Spatial** – vector-based panning spatial audio on the web.
  - Additional package groupings (as listed in the repo):
    - Frameworks
    - Libraries
    - Utilities
    - MIDI tools
    - Apps

- **Learning and reference resources**  
  - **Tutorials** section for step-by-step learning of the Web Audio API.
  - **Books** section collecting longer-form learning materials.
  - **Newsletters** section for ongoing Web Audio–related updates.
  - **Community** section pointing to community hubs and discussion spaces.

- **Historical and obsolete tools**  
  - **Obsolete** section to track older or no-longer-maintained Web Audio projects, helping with context and avoiding outdated choices.

- **Open contribution model**  
  - Contributions via GitHub pull requests.
  - `contributing.md` and `code-of-conduct.md` included to guide community participation.
