## Overview

A comprehensive collection of knowledge graph reasoning works, including research papers, code implementations, and datasets. Covers various approaches to reasoning over knowledge graphs for inference and prediction.

## Reasoning Tasks

### Link Prediction

- Predict missing relationships
- Entity alignment
- Relation extraction
- Completion of incomplete triples

### Entity Classification

- Type prediction
- Category assignment
- Hierarchical classification
- Multi-label classification

### Query Answering

- Complex query processing
- Multi-hop reasoning
- Aggregation queries
- Negation handling

### Logical Reasoning

- Rule-based inference
- Inductive logic programming
- Ontology reasoning
- Consistency checking

## Approaches

### Embedding-Based Methods

- **TransE** - Translational embeddings
- **DistMult** - Bilinear models
- **ComplEx** - Complex embeddings
- **RotatE** - Rotation-based embeddings
- **ConvE** - Convolutional networks

### Graph Neural Networks

- **R-GCN** - Relational Graph Convolutional Networks
- **CompGCN** - Composition-based GCN
- **KBAT** - Knowledge Base Attention
- **NBFNet** - Neural Bellman-Ford Networks

### Rule-Based Methods

- AMIE - Association Rule Mining
- RLvLR - Reinforcement Learning for Logical Reasoning
- Neural-LP - Differentiable logic programming
- DRUM - Differentiable rule mining

### Hybrid Approaches

- Combining embeddings and rules
- Neural-symbolic integration
- Knowledge-enhanced neural models
- Explainable reasoning
