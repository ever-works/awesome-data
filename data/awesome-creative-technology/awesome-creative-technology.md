# Awesome Creative Technology

**Description**

Awesome Creative Technology is a curated, open-source list of businesses, studios, companies, collectives, and groups working at the intersection of computing, design, art, and user experience. It is maintained as a GitHub repository.

## Features

- **Curated directory of organizations**
  - Lists creative technology groups, companies, studios, and collectives.
  - Focuses on entities that blend computing, design, art, and user experience.

- **Open-source repository format**
  - Hosted on GitHub as a public repository.
  - Directory data and supporting scripts stored in source files (e.g., `groups.ts`).

- **Structured data and tooling**
  - Uses TypeScript files such as `groups.ts`, `index.ts`, and `add-studio.ts` to manage and generate the list.
  - `generate-readme.ts` is used to programmatically build or update the main README listing.
  - Configuration files (`.eslintrc.json`, `.prettierrc.json`) enforce consistent code style.

- **Contribution workflow**
  - `contributing.md` outlines how to propose additions or changes to the list.
  - `add-studio.ts` provides a scripted way to add new studios or groups to the dataset.
  - `UP.md` and `CLOSED.md` provide project status and/or maintenance notes.

- **Licensing and governance**
  - A `LICENSE` file defines the open-source terms for using and contributing to the list.

- **Project assets**
  - Includes a project logo (`logo.gif`) and templates in the `templates` directory, likely for standardized entries or documentation.
