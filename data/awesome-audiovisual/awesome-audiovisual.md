# Awesome Audiovisual

**Website:** https://github.com/stingalleman/awesome-audiovisual#readme  
**Category:** Themed directories  
**Tags:** awesome-lists, audio, video

## Overview
Awesome Audiovisual is a curated, open GitHub directory of software, libraries, tools, and resources for professional and hobbyist work in lighting, audio, video, and related show-control environments (events, installations, broadcasting, etc.).

## Features
- **Curated “awesome list” format**
  - Organized list of high-quality audiovisual tools and resources.
  - Focused on professional and hobbyist use in lighting, audio, video, and show control.

- **Structured by technical domains**
  - **Lighting**
    - Lighting Software (includes both open-source and commercial / freemium tools).
    - Visualizers for simulating or previewing lighting setups.
  - **Audio**
    - Audio Software (mix of open-source, freemium, and commercial tools).
    - Audio Tools section for utilities and supporting workflows.
  - **Video**
    - References to specialized video/broadcast resources, including the `ebu/awesome-broadcasting` list.
  - **Tools**
    - General-purpose tools that support audiovisual workflows.
    - CAD tools (commercial) for planning and designing AV setups.
    - Timecode tools (open-source and commercial) for synchronization and temporal coordination.
    - Monitoring tools (primarily open-source) for observing system and signal status.
  - **Libraries and Frameworks**
    - Programming libraries (e.g., Node.js, Java) relevant to AV control and integration.
    - ArtNet libraries for lighting network protocols.
  - **Hardware**
    - Separate `HARDWARE.md` file listing AV-related hardware options.

- **License and governance**
  - Published under an open-source license (`LICENSE` file in the repository).
  - Includes a `code-of-conduct.md` defining community behavior expectations.

- **Contribution and community**
  - `contributing.md` with contribution guidelines for adding or improving entries.
  - Contributions via pull requests are encouraged.
  - General questions and discussion handled through a dedicated GitHub issue (General Discussion).

- **Metadata for each listed resource**
  - Indicators for:
    - `✓ open-source` – clearly flagged open-source projects.
    - `⚠ not free` – commercial software.
    - `⚠ freemium` – partially free, paid upgrade options.
  - Language/stack hints for libraries (e.g., `nodejs`, `java`).

## Pricing
- Awesome Audiovisual itself is a free, open GitHub repository.
- Individual tools and resources in the list may be:
  - Open-source
  - Freemium
  - Paid / commercial (marked as `⚠ not free`)
