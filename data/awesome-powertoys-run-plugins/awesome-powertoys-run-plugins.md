## Description
Awesome PowerToys Run Plugins is a curated, community‑maintained directory of plugins for **PowerToys Run**, the Windows quick launcher. It collects and organizes third‑party plugins in an "awesome‑list" format to help users discover and explore extensions that enhance and customize PowerToys Run.

## Features
- **Curated plugin list** – Focuses specifically on community plugins built for PowerToys Run.
- **Awesome‑style directory** – Structured as a classic "awesome list" with categorized entries and links.
- **Windows ecosystem focus** – Targets plugins that extend the Windows PowerToys Run launcher.
- **Community contributions** – Includes guidelines and templates (`contributing.md`, `pull_request_template.md`) for adding or updating plugin entries.
- **Quality and consistency checks** – Uses a `checklist.md` to standardize how plugins are evaluated and documented before inclusion.
- **Code of conduct** – Defines community behavior and contribution norms in `code-of-conduct.md`.
- **Repository configuration files** – Includes `.editorconfig`, `.gitattributes`, and `.vscode` settings for consistent editing and maintenance of the list.
- **Visual assets** – Provides launcher screenshots (`launcher-dark.png`, `launcher-light.png`) and branding (`powertoys-logo.png`) to illustrate PowerToys Run usage.
- **Open licensing** – Contains a `license` file clarifying reuse and contribution terms.
