## Overview

Awesome Raspberry Pi provides a curated collection of tools, projects, operating systems, and resources for Raspberry Pi development.

## Operating Systems

### Official OS

**Raspberry Pi OS**
- Official supported Raspberry Pi operating system (formerly Raspbian)
- Based on Debian Linux
- Optimized for Raspberry Pi hardware
- Desktop and Lite versions
- Pre-installed with development tools

**Ubuntu Core**
- Official minimal Ubuntu distribution for IoT
- Supports Raspberry Pi Zero 2 W and later models
- Snap-based app deployment
- Automatic updates
- Enterprise-grade security

### Alternative OS
- **Ubuntu Server** - Full Ubuntu for Pi
- **DietPi** - Lightweight Debian-based OS
- **LibreELEC** - Media center OS (Kodi)
- **RetroPie** - Retro gaming platform
- **Volumio** - Audiophile music player
- **Home Assistant OS** - Home automation

## Development Tools

### GPIO Libraries

**WiringPi**
- Gordon's Arduino wiring-like library for Raspberry Pi
- C/C++ GPIO interface
- Command-line gpio utility
- PWM, I2C, SPI support

**RPi.GPIO**
- Official Python GPIO library
- Simple pin control
- Event detection
- PWM support

**pigpio**
- Advanced GPIO library
- Hardware timed PWM
- Servo control
- Remote GPIO

### Python Tools
- **gpiozero** - Simple GPIO interface
- **picamera** - Python camera interface
- **sense-hat** - Sense HAT library
