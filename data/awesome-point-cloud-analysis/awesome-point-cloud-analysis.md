## Overview

Awesome Point Cloud Analysis collects research papers and datasets for processing 3D point cloud data, essential for autonomous vehicles, robotics, and 3D scene understanding.

## Key Tasks

### Classification
Assigning labels to entire point clouds (e.g., chair, car, tree).

### Semantic Segmentation
Labeling each point with its class (road, building, vegetation).

### Instance Segmentation
Identifying individual object instances in point clouds.

### Object Detection
Finding and localizing 3D objects with bounding boxes.

### Registration
Aligning multiple point clouds into a common coordinate system.

## Deep Learning Architectures

### PointNet
Pioneer work directly processing unordered point sets.

### PointNet++
Hierarchical feature learning with local neighborhoods.

### DGCNN
Dynamic Graph CNN for point cloud learning.

### Point Transformers
Applying attention mechanisms to points.

## Datasets

### Indoor
- ScanNet - indoor 3D scenes
- S3DIS - Stanford large-scale 3D indoor spaces

### Outdoor
- KITTI - autonomous driving
- SemanticKITTI - semantic segmentation
- nuScenes - multimodal autonomous driving
- Waymo Open Dataset

### Objects
- ModelNet40 - 3D CAD models
- ShapeNet - large-scale repository
