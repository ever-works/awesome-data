# Awesome-Diffusion-Models

**Type:** Themed directory / curated resource list  
**Category:** Themed Directories  
**Tags:** machine-learning, deep-learning, research

## Overview
Awesome-Diffusion-Models is a GitHub-based, Awesome-style collection of resources and research papers focused on diffusion models in machine learning. It serves as a specialized, regularly updated directory for learning about and working with diffusion models.

Main repository: <https://github.com/arasgungore/Awesome-Diffusion-Models>  
Full content site: <https://diff-usion.github.io/Awesome-Diffusion-Models/>

## Features
- **Curated diffusion model resources**  
  - Focused specifically on diffusion models in machine learning and deep learning.  
  - Includes links to research papers and learning materials.

- **Introductory learning materials**  
  - Section for **Introductory Posts** to help newcomers understand core diffusion theory.  
  - Example entry: *DiffusionFastForward: 01-Diffusion-Theory* by Mikolaj Czerkawski, with a linked website/notes.

- **Research paper directory**  
  - Organized collection of diffusion model research papers (details primarily available on the external site due to GitHub page constraints).

- **Supplementary documentation and site**  
  - `docs` directory for structured documentation.  
  - `website` directory powering the external page that contains the full, expanded list of resources.
