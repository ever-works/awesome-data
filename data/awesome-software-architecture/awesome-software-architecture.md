# Awesome Software Architecture

- **Tags:** architecture, software, development

## Overview
Awesome Software Architecture is a curated, community-driven list of resources focused on software architecture. It aggregates references for learning and applying architectural principles, patterns, and practices when designing and building software systems.

## Features
- **Curated resource collection** for software architecture topics.
- **Broad coverage of architectural concerns**, including:
  - **Principles** – foundational ideas and guidelines for structuring software systems.
  - **Design Patterns** – reusable architectural and design solutions.
  - **Scalability and Resilience** – patterns and resources for building scalable, fault-tolerant systems.
  - **Methodology** – approaches and processes for doing architecture in practice.
  - **Documentation** – techniques and examples for documenting software architectures.
  - **Workshop Formats** – formats and ideas for running architecture-related workshops.
  - **Modeling** – modeling techniques and notations used in architecture.
  - **Tools** – software tools that support architectural design, analysis, and documentation.
  - **Frameworks** – frameworks relevant to architectural work.
    - **Agile**
    - **Lean software development**
    - **Extreme programming**
    - **DevOps**
  - **Bonus** – additional, related resources not covered in the main sections.
- **Community contributions encouraged** via documented contribution guidelines.
- **Code of conduct** to support a respectful, collaborative community.
