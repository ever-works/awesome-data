## Overview

Awesome LaTeX is a curated directory of tools, packages, templates, and learning resources for the LaTeX / TeX typesetting system. It aggregates distributions, editors, build tools, packages, symbols, tutorials, and more into a single, organized reference.

### Distributions & Environments
- **Distributions** section for major LaTeX distributions (listed in the full README).
- **Docker Images** for ready‑to‑run LaTeX environments without local installation:
  - `reitzig/texlive-docker` – minimal TeX Live system with on‑demand package installation.
  - `Island of TeX/texlive` – Debian‑based full TeX Live system.
  - `dante-ev/docker-texlive` – Debian‑based full TeX Live system with extra tools (pandoc, Perl, Python pre‑installed).

### Engines & Web Formulas
- **Engines** section covering LaTeX engines (e.g., pdfTeX, XeTeX, LuaTeX and related tools in the full list).
- **LaTeX formulas on the web** subsection for tools and libraries to render LaTeX math in browsers.

### Editors & IDEs
- **Overview**: links to curated community list of LaTeX editors/IDEs:
  - “Big list of LaTeX Editors/IDEs” on TeX StackExchange.
  - “List of popular LaTeX editors” with screenshots and short descriptions.
- **LaTeX‑focused editors**: dedicated LaTeX IDEs (full enumeration in the original list).
- **General purpose text editors**: editors that support LaTeX among many languages, such as:
  - Atom – listed with reference to the Awesome Atom collection and FOSS badge.
- **Online editors**: browser‑based LaTeX editors and collaboration platforms.

### Bibliography & Citation Tools
- **Bibliography tools** section for managing references and bibliographies (BibTeX, biblatex tools, GUIs, etc., in the full list).

### Build & Automation
- **Build Tools** for compiling LaTeX documents (CLI tools, wrappers, automation helpers).
- **GitHub Actions** subsection for CI/CD workflows to automatically build LaTeX documents using GitHub Actions.

### Miscellaneous & GUI Tools
- **Misc. Tools** for assorted LaTeX‑related utilities (e.g., log analyzers, checkers, converters).
- **LaTeX‑compatible GUI tools** for creating diagrams, graphics, or content that exports to LaTeX‑friendly formats.

### Packages
Curated LaTeX packages grouped by purpose:
- **References** – cross‑referencing, citations, clever referencing helpers.
- **Tables** – advanced table formatting and layout.
- **Graphics** – image and graphics integration, including:
  - **PSTricks** – PostScript‑based drawing tools.
  - **TikZ** – powerful vector graphics and diagramming in LaTeX.
- **Source Code** – syntax highlighting and code listings.
- **Typography** – fonts, microtypography, and layout refinement.
- **Presentations / Slides** – slide and presentation frameworks (e.g., beamer and others in the full list).

### Templates
- Collection of LaTeX templates:
  - Articles, reports, theses, CVs, letters.
  - Presentation templates.
  - Publisher / journal / conference templates (as listed in the repository).

### Symbols
- **Symbols** section referencing symbol catalogs and tools for finding math and special symbols.

### Resources, Showcases & Community
- **Resources** – general information hubs, documentation, and reference sites.
- **Showcases** – examples of documents or projects created with LaTeX.
- **Tutorials** – step‑by‑step guides for various aspects of LaTeX.
- **Books** – recommended books for learning and mastering LaTeX.
- **Blogs** – blogs covering LaTeX tips, news, and workflows.
- **Social media** – links to LaTeX‑related social media accounts and communities.
- **Meta Awesome‑LaTeX** – information about contributing, tagging, and list maintenance.
