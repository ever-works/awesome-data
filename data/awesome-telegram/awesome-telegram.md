## Overview

Awesome Telegram is a curated collection of great Telegram groups, channels, bots, and libraries for Telegram development.

## Bot Development Frameworks

- **grammY** - TypeScript framework for Telegram Bot API
- **TediCross** - Bot which bridges Telegram chats with Discord channels
- **OnioAIagent-V2** - Open-source framework for AI agents with Telegram support

## Bot Templates

- EdJoPaTo/telegram-bot-boilerplate - For scalable JavaScript bots
- Node.js templates for quick bot development
- Python bot starters

## Libraries

- python-telegram-bot - Python wrapper for Telegram Bot API
- Telethon - Pure Python 3 MTProto API Telegram client library
- node-telegram-bot-api - Node.js wrapper
