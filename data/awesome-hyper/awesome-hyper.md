# Awesome Hyper

**Description**  
Awesome Hyper is a curated, community-maintained list of plugins, themes, and resources for the Hyper terminal application.

**Website**  
<https://github.com/bnb/awesome-hyper#readme>

**Category**  
- Themed directories

**Tags**  
- awesome-lists  
- terminal  
- javascript

**Platform / Brand**  
- Hosted on GitHub

---

## Features

- **Curated list of Hyper packages**  
  - Collection of Hyper plugins (extensions) that enhance terminal functionality.  
  - Organized and maintained as part of an “awesome” style list.

- **Curated list of Hyper themes**  
  - Themes to customize the appearance of the Hyper terminal.  
  - Links to external theme repositories and resources.

- **Resources for Hyper users**  
  - Additional Hyper-related documentation, tools, and learning resources.  
  - References to related awesome lists (e.g., awesome-node, awesome-npm) that include CLI tools usable within Hyper.

- **Open-source repository structure**  
  - Public GitHub repository including:
    - `README.md` with the main curated lists and links.
    - `CONTRIBUTING.md` with contribution guidelines.  
    - `code-of-conduct.md` defining community behavior expectations.  
    - `LICENSE` specifying usage rights.  
    - Project configuration files (e.g., `.travis.yml`, `.gitignore`).

- **Community contributions**  
  - Accepts community-submitted additions and improvements via GitHub pull requests.  
  - Uses documented contribution guidelines for consistent curation.

- **Integration context**  
  - References the official Hyper website (hyper.is) for the terminal itself.  
  - Connects to the broader “awesome” ecosystem (e.g., awesome-nodejs, awesome-npm) for complementary tools.
