# Awesome Search

**Website:** https://awesomelists.top  
**Category:** Meta Directories  
**Tags:** awesome-lists, meta-directory, searchable  
**Featured:** Yes

## Description
Awesome Search is a dedicated search engine and web interface for exploring, searching, and navigating GitHub “Awesome” lists from a single page. It focuses on the lists curated in the sindresorhus/awesome repository and helps users quickly find relevant resources by keyword and category.

## Features
- **Unified search across Awesome lists**  
  - Search every Awesome list included in the [sindresorhus/awesome](https://github.com/sindresorhus/awesome) repository from one interface.
- **Single-page access to lists**  
  - Browse and filter all supported Awesome lists within a single-page web application.
- **Direct list URLs**  
  - Access any supported Awesome list with a direct URL pattern, e.g.:
    - `https://awesomelists.top/#/sindresorhus/awesome-nodejs`  
  - Suitable for bookmarking favorite lists for quick return access.
- **Table-of-contents navigation**  
  - When an Awesome list includes a table of contents, Awesome Search exposes it as a “content” section for smooth navigation to specific sections within that list.
- **Category-based exploration**  
  - Browse lists through high-level categories such as:
    - Platforms
    - Programming Languages
    - Front-End Development
    - Back-End Development
    - Computer Science
    - Big Data
    - Theory
    - Books
    - Editors
    - Gaming
    - Development Environment
    - Entertainment
    - Databases
    - Media
    - Learn
    - Security
    - Content Management Systems
    - Hardware
    - Business
    - Work
    - Networking
    - Decentralized Systems
    - Health and Social Science
    - Events
    - Testing
    - Miscellaneous
    - Related
- **Example entry point**  
  - Predefined example link: [Awesome Node.js](https://awesomelists.top/#/sindresorhus/awesome-nodejs) to quickly see how a list is presented.
- **Legacy version access**  
  - Optional access to an older version of the interface at: https://legacy.awesomelists.top/#/

## Pricing
No pricing or paid plans are mentioned; Awesome Search appears to be free to use.
