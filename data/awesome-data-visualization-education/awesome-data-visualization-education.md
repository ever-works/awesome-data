## Overview

Awesome Data Visualization Education provides curated open educational resources for learning data visualization principles, techniques, and tools.

## Core Topics

### Fundamentals
- Visual encoding principles
- Color theory for visualization
- Chart types and when to use them
- Data-to-visual mapping
- Perceptual principles

### Theory
- Information visualization theory
- Visual analytics
- Storytelling with data
- Dashboard design

### Tools & Technologies
- D3.js for custom visualizations
- Tableau for business intelligence
- Python (Matplotlib, Seaborn, Plotly)
- R (ggplot2, Shiny)
- Observable notebooks
