# Awesome Public Datasets – CyberSecurity Category

## Overview
Awesome Public Datasets – CyberSecurity Category is a curated subsection of the broader Awesome Public Datasets (APD) meta-directory. It focuses specifically on cybersecurity-related datasets, aggregating high-quality, publicly available data useful for security research, experimentation, and education.

Source: <https://github.com/awesomedata/apd-core/tree/master/core//CyberSecurity>

## Category
- Themed directories / Meta-directory
- Part of the larger "Awesome"-style collection of topic-organized resources

## Features
- **Cybersecurity-focused collection**  
  Dedicated to datasets relevant to cybersecurity, network defense, and threat analysis.

- **Curated entries**  
  Includes handpicked, vetted datasets rather than an unfiltered list, aiming for quality and relevance.

- **Example datasets mentioned**  
  - **CCCS-CIC-AndMal-2020** – Android malware–related dataset.  
  - **Traffic and Log Data Captured During a Cyber Defense Exercise** – Network traffic and log data from simulated defense scenarios.

- **Research-oriented resources**  
  Datasets are suitable for academic and industrial research, including malware analysis, intrusion detection, traffic analysis, and security tooling evaluation.

- **Part of APD meta-directory**  
  Integrates with the wider Awesome Public Datasets project, allowing users to discover related datasets in other domains (e.g., machine learning, networking, etc.).

- **Open and public datasets**  
  Focuses on datasets that are publicly accessible for learning, experimentation, and reproducible research.

## Use Cases
- Building and evaluating intrusion detection systems
- Malware classification and behavior analysis
- Network traffic and log analysis for security
- Training and benchmarking security ML/AI models
- Academic coursework and labs in cybersecurity
