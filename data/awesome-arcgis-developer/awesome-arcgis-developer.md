# Awesome ArcGIS Developer

**Category:** Themed Directories  
**Tags:** gis, developer-tools, maps  
**Source:** https://github.com/Esri/awesome-arcgis-developer#readme

## Overview
Awesome ArcGIS Developer is a curated, open-source directory of resources for developers working with ArcGIS products. It aggregates links to SDKs, APIs, tools, samples, and learning materials for building mapping and location-aware applications.

## Features
- **Curated resource list** for ArcGIS development and related location services
- **Focus on ArcGIS products** and how to develop with them
- **Coverage of APIs and SDKs** used for mapping and geospatial applications
- **Developer tools and utilities** relevant to GIS and location-aware solutions
- **Code samples and examples** to illustrate practical ArcGIS use cases
- **Learning resources** such as tutorials and documentation links
- **GitHub-based project** with standard files:
  - `README.md` for main resource list and usage
  - `CONTRIBUTING.md` for contribution guidelines
  - `CODE_OF_CONDUCT.md` for community standards
  - `LICENSE` specifying usage terms
- **Project configuration and automation files** (e.g., `.gitattributes`, `.gitignore`, GitHub workflows, `package.json`, `lychee.toml` for link checking)
