# OSSU Computer Science

**Website:** https://github.com/ossu/computer-science  
**Category:** Themed Directories  
**Tags:** computer-science, education, courses

## Overview
OSSU Computer Science is a community-maintained, open-source curriculum that curates a complete path to a free, self-taught computer science education. It aggregates and organizes high-quality open courses and learning resources into a structured, university-level program.

## Features
- **Complete CS Curriculum**  
  - Covers core topics expected from a traditional computer science degree.  
  - Structured as a full, multi-year program rather than a single course.

- **Free and Open Resources**  
  - Built entirely from freely accessible online courses and materials.  
  - Emphasizes open educational resources from various platforms.

- **Community-Maintained**  
  - Open-source repository hosted on GitHub.  
  - Curriculum and documentation updated via community contributions and pull requests.

- **Curricular Guidelines**  
  - Dedicated `CURRICULAR_GUIDELINES.md` file describing goals, structure, and standards for included courses.  
  - Focus on aligning with university-level CS education expectations.

- **Documentation and Help**  
  - `FAQ.md` answering common questions about how to follow and use the curriculum.  
  - `HELP.md` offering guidance on getting started with the program and repository.  
  - `CONTRIBUTING.md` explaining how to propose changes, add courses, or improve materials.

- **Course Organization**  
  - `coursepages/` directory with pages for individual courses and learning units.  
  - `extras/` directory for supplementary materials beyond the main curriculum.  
  - Uses Jekyll layouts (`_includes`, `_layouts`) for structured presentation on the web.

- **Versioning and Change History**  
  - `CHANGELOG.md` tracking curriculum and repository updates over time.  
  - Git history with commits documenting modifications to the learning path.

- **Open License**  
  - Released under the MIT License, allowing reuse, modification, and redistribution with minimal restrictions.

- **Web Presence**  
  - Custom domain configured via `CNAME`.  
  - Rendered as a static site from the GitHub repository for easier browsing of the curriculum.

## Pricing
- The curriculum is composed of **free** online resources.  
- There are **no subscription fees** or paid plans listed in the repository content provided.
