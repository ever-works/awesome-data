## Overview

Awesome TypeScript Projects showcases high-quality open source projects written in TypeScript, demonstrating real-world applications and best practices.

## IDEs & Editors

### VS Code
Microsoft's popular editor, itself written in TypeScript, with excellent TypeScript support.

### Monaco Editor
The browser-based code editor powering VS Code, TypeScript-based.

## Frameworks

### Angular
Google's comprehensive framework, TypeScript-first from the ground up.

### NestJS
Node.js framework for building efficient server-side applications with TypeScript.

### Remix
Full-stack React framework with strong TypeScript support.

## Libraries

### RxJS
Reactive programming library with comprehensive TypeScript types.

### TypeORM
ORM for TypeScript and JavaScript supporting multiple databases.

### Prisma
Next-generation ORM with excellent TypeScript integration.

## Development Tools

### ESLint
Linting with TypeScript-specific rules via @typescript-eslint.

### Prettier
Code formatting with TypeScript support.

### ts-node
TypeScript execution environment for Node.js.

### TypeDoc
Documentation generator for TypeScript projects.

## Build Tools

### Vite
Fast build tool with first-class TypeScript support.

### esbuild
Extremely fast bundler written in Go, handles TypeScript natively.

### swc
Rust-based compiler, much faster than tsc.
