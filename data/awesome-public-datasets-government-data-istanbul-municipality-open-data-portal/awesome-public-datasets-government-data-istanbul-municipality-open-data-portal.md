# Awesome Public Datasets – Government Data: Istanbul Municipality Open Data Portal

- **Website**: https://data.ibb.gov.tr  
- **Provider**: Istanbul Metropolitan Municipality (İstanbul Büyükşehir Belediyesi – İBB)  
- **Category**: Themed directories / Government open data  
- **Tags**: open-data, city, datasets

## Overview
The Istanbul Metropolitan Municipality (İBB) Open Data Portal is a city-level open data platform that provides access to datasets from various municipal and urban domains. It aggregates data from different city information systems and makes them available for exploration, download, and use via APIs.

## Features

### Portal Usage & Access
- Central web portal for Istanbul’s municipal and urban datasets.
- Access via “İBB Giriş” (İBB login) user system.
- Existing users continue with their registered email address through the updated login infrastructure.
- Detailed usage guidance available via a dedicated "Açık Veri Portalı Nasıl Kullanılır?" (How to Use the Open Data Portal) help page.
- Ability to submit data requests for datasets that are not yet available through an online request form.

### Data Discovery
- **Search**:
  - Search bar for keyword-based discovery of datasets.
  - Browsing datasets through topic-based lists and categories.
- **Explore**:
  - View dataset information and metadata.
  - Preview data for an initial understanding before download or integration.

### Data Access & Use
- Download datasets for offline use.
- Access data through APIs for integration into applications, analyses, or projects.
- Suitable for use in research, civic tech projects, and urban analytics.

### Data Categories
Datasets are tagged and organized into multiple thematic categories:
- **Bilgi ve İletişim Teknolojileri (Information & Communication Technologies)**
- **Enerji (Energy)**
- **Ekonomi (Economy)**
- **Güvenlik (Security)**
- **Mobilite (Mobility / Transportation)**
- **Çevre (Environment)**
- **İnsan (People / Social Services)**
- **Yönetişim (Governance)**
- **Yaşam (Life / Urban Life)**

These categories help users locate relevant datasets related to technology, energy, economy, security, transport, environment, social services, governance, and everyday urban life in Istanbul.

## Pricing
- The portal is a government open data service; the provided content does not mention any pricing, suggesting public access without listed fees.
