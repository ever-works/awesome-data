## Overview

Awesome UI by kevindeasis is a comprehensive curated list covering everything related to UI & UX design, including theory, practical tutorials, books, libraries, and design resources.

## Features

- **Design Theory**: Fundamentals of UI/UX principles and best practices
- **Tutorials**: Step-by-step guides for learning design
- **Books**: Recommended reading for designers
- **Component Libraries**: Pre-built UI components and design systems
- **Tools**: Design software and prototyping tools
- **Inspiration**: Galleries and showcases of great design
- **Accessibility**: Resources for designing inclusive interfaces
- **Typography**: Font resources and typography guides
- **Color**: Color theory and palette generators
- **Icons**: Icon libraries and resources
