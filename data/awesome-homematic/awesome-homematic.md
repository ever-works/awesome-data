# Awesome Homematic

**URL:** https://github.com/homematic-community/awesome-homematic#readme  
**Category:** Themed Directories  
**Tags:** iot, home-automation, awesome-lists

## Overview
Awesome Homematic is a curated, community-maintained collection of links related to Homematic smart home devices from eQ-3. It focuses on software, documentation, community resources, and tools that support Homematic-based home automation, with a strong emphasis on German-language resources.

## Features
- **Curated link collection**
  - Organized list of Homematic-related resources from across the web
  - Emphasis on quality and relevance rather than exhaustive listing

- **Coverage of Homematic ecosystem**
  - Focused on Homematic smart home devices from eQ-3
  - Includes resources popular particularly in the German-speaking community

- **Structured by topic areas** (as sections in the list)
  - Community Resources (mostly German language)
  - Documentation
  - Mobile Apps
  - CCU Alternatives
  - Alternative Sensors, Actuators, and Hardware Modifications
  - CCU Addons
  - Interfacing Software
  - Misc Software
  - Software Modules
  - Smart Home Software supporting Homematic
  - Miscellaneous

- **Open contribution model**
  - Public contribution guidelines for adding or improving entries
  - Community-driven maintenance via GitHub

- **Open licensing**
  - Published under CC0 (Public Domain), allowing free use, reuse, and adaptation of the list contents

## Use Cases
- Discover software and tools that work with Homematic devices.
- Find alternative CCU solutions and add-ons for extended functionality.
- Explore documentation and community resources (especially in German) for setup, integration, and troubleshooting.
- Identify mobile apps and smart home platforms that support Homematic.

## Pricing
- Access to the Awesome Homematic list is **free**.
- Content is released under a **Public Domain CC0 license**, with no associated usage fees.
