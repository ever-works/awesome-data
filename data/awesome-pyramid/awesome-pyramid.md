# Awesome Pyramid

**Website:** https://github.com/uralbash/awesome-pyramid#readme  
**Category:** Themed Directories  
**Tags:** awesome-lists, python, framework

## Description
Awesome Pyramid is a curated, community-maintained list of applications, packages, tutorials, and other resources for the Pyramid Python web framework. It is inspired by and based on the broader awesome-python collection.

## Features
- **Curated resource collection**
  - Focused specifically on the Pyramid web framework and its ecosystem
  - Includes apps, projects, extensions, and learning materials
- **Thematic categorization of packages and tools**
  - **Admin Interface** – Extensions and tools that add or enhance admin UIs for Pyramid-based projects.
  - **Asset Management** – Packages for handling static files and other front-end assets.
  - **Async** – Resources related to asynchronous patterns and support in Pyramid.
  - **Authentication** – Packages that extend or improve authentication mechanisms.
  - **Authorization** – Tools and libraries dealing with permissions and authorization policies.
  - **Caching & Session** – Helpers for server-side caching and session management.
  - **Debugging** – Tools to inspect, trace, and debug Pyramid applications.
  - **Email** – Utilities for sending and managing email from Pyramid projects.
  - **Forms** – Libraries that extend form handling and introduce additional form types.
  - **Media Management** – Tools for handling media files (uploading, processing, serving).
  - **RESTful API** – Packages to build REST-style APIs with Pyramid.
  - **Search** – Integrations and libraries providing search capabilities.
  - **Security** – Add-ons that strengthen application security.
  - **Services** – Service-layer helpers and related infrastructure.
  - **Settings** – Utilities to manage configuration and application settings.
  - **Storage** – Backends and extensions for data and file storage.
  - **Task Queue** – Libraries that support background jobs and task queues.
  - **Templates** – Templating engines and related integrations.
  - **Testing** – Tools to test Pyramid code and generate test data.
- **Community contributions**
  - CONTRIBUTING guidelines provided for adding and maintaining entries.
- **Open repository structure**
  - Public GitHub repository with organized sections and continuous updates.
