## Overview

Awesome Vector Search is a curated collection of libraries, services, and research papers related to vector search, embeddings, and high-dimensional similarity search.

## Vector Databases

- **Qdrant** - High-performance vector similarity search engine with extended filtering support
- **Weaviate** - Open-source vector database that stores both objects and vectors
- **Milvus** - Cloud-native vector database built for scalable similarity search
- **Chroma** - AI-native open-source embedding database
- **Pinecone** - Managed vector database for production-grade applications

## Search Libraries

- **Faiss** - Facebook's library for efficient similarity search and clustering of dense vectors
- **Annoy** - Approximate Nearest Neighbors in C++/Python optimized for memory usage
- **NMSLIB** - Non-Metric Space Library for efficient similarity search
- **USearch** - Smaller and faster single-file similarity search library
- **Hora** - Approximate nearest neighbor search library

## Features

- Efficient similarity search at scale
- Support for various distance metrics (cosine, Euclidean, dot product)
- Approximate nearest neighbor (ANN) algorithms
- Integration with machine learning frameworks
- Production-ready vector databases
- Research papers and academic resources

## Use Cases

- Semantic search and RAG (Retrieval Augmented Generation) applications
- Recommendation systems
- Image and video similarity search
- Duplicate detection
- Clustering and classification
- Natural language processing applications

## Pricing

Varies by solution - many open-source libraries are free, while managed services like Pinecone and Qdrant Cloud offer free tiers with paid scaling options.
