## Overview
Awesome Public Datasets – Government (Meta: Fredericton, NB, Canada) is an entry in the Awesome Public Datasets list that points to Fredericton’s municipal open data catalogue. It serves as a thematic directory reference for civic/government datasets related to the City of Fredericton in New Brunswick, Canada.

## Features
- **Meta-directory entry** within the Awesome Public Datasets GitHub repository.
- **Government / civic focus**, specifically for:
  - Municipal datasets from Fredericton, New Brunswick, Canada.
- **Open data orientation**:
  - Highlights datasets that are published as open data by the municipality.
- **Catalogue-level scope**:
  - Points to Fredericton’s overall open data catalogue rather than a single dataset.
- **Part of a curated collection**:
  - Included in a large, topic-centric list of high-quality public datasets maintained in the Awesome Public Datasets project.
- **GitHub-based reference**:
  - Hosted as an item inside a GitHub README-driven directory, benefiting from community curation and updates.
