# Awesome Seed RS

A curated directory of resources for the Seed framework, focused on building Rust-powered WebAssembly web applications.

- **Website:** https://github.com/seed-rs/awesome-seed-rs#readme
- **Category:** Themed directories
- **Tags:** awesome-lists, rust, webassembly
- **Brand:** seed-rs

## Overview
Awesome Seed RS is a community-maintained "awesome" list that aggregates learning materials, example projects, and libraries related to the Seed framework. Seed is a Rust framework for creating WebAssembly front-end web apps; this list helps developers discover tools and references in that ecosystem.

## Features
- **Curated resource collection**
  - Organized list of Seed-related links and projects.
  - Focused specifically on Rust + WebAssembly front-end development with Seed.

- **Framework-specific coverage**
  - Resources tailored to the Seed framework (not generic Rust or WebAssembly only).
  - Includes examples and libraries that demonstrate idiomatic Seed usage.

- **Examples and sample apps**
  - Example projects for learning how to structure Seed applications.
  - Code samples that showcase core patterns in Rust/Seed WebAssembly apps.

- **Libraries and ecosystem tools**
  - Third-party libraries built for or commonly used with Seed.
  - Utilities that integrate Rust/Seed apps into broader web stacks.

- **Learning materials**
  - Tutorials and guides to help developers get started with Seed.
  - Reference materials for building and deploying Seed-based WebAssembly apps.

- **Open, community-driven list**
  - Hosted on GitHub for easy browsing and contribution.
  - Follows the common "awesome list" style and conventions.
