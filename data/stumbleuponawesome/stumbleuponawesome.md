# StumbleUponAwesome

A browser-extension "discovery button" that opens a random website from curated Awesome lists, aimed at developers and tech/science enthusiasts.

**Website/Source:** https://github.com/basharovV/StumbleUponAwesome

## Features
- Browser extension interface for quick access from the toolbar
- Opens a random site from curated Awesome lists (GitHub Awesome collections)
- Focused on content for developers, technology, and science enthusiasts
- Uses a scraper component to gather and update links from Awesome datasets
- Includes animation assets (e.g., rabbit-hole and header animations) for visual feedback when stumbling
- Configuration via project files (e.g., jsconfig.json) to support development and customization
- Open-source codebase with changelog and version history
- Licensed under an open-source license (see `LICENSE` in the repository)
