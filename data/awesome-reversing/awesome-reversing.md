## Overview
awesome-reversing is an open-source, Awesome-style curated directory of software reverse-engineering resources. It aggregates tools, learning materials, and references for analyzing binaries, bytecode, and malware across various platforms.

## Features
- **Curated structure**
  - Organized as an Awesome list focused on reverse engineering.
  - Emphasis on educational and practical resources.

- **Books**
  - Collection of books dedicated to reverse engineering concepts and techniques.

- **Courses**
  - Reverse engineering courses for structured learning.

- **Channels**
  - Binary analysisâ€“oriented channels (e.g., video or streaming channels) for ongoing learning and demonstrations.

- **Practice resources**
  - Practice environments and challenges for reverse engineering.
  - Includes warnings to be careful when working with malware samples.

- **Hex editors**
  - Tools for viewing and editing binary files at the byte level.

- **Binary format tools**
  - Utilities for inspecting and working with binary file formats.

- **Disassemblers / Decompilers**
  - Tools for converting machine code or bytecode back into assembly or higher-level representations.

- **Binary analysis**
  - Resources and tools for static and dynamic analysis of binaries.

- **Bytecode analysis**
  - Specialized tools for analyzing bytecode (e.g., from virtual machines or managed runtimes).

- **Import reconstruction**
  - Tools for reconstructing import tables and related metadata in binaries.

- **Dynamic analysis**
  - Tools to observe program behavior at runtime (e.g., sandboxing, tracing, instrumentation).

- **Debugging**
  - Debuggers and related utilities for stepping through and inspecting executing programs.

- **Mac decrypt tools**
  - Tools focused on decrypting or unpacking binaries on macOS.

- **Document analysis**
  - Tools for analyzing potentially malicious or obfuscated documents.

- **Scripting**
  - Scripting resources and tools to automate or extend reverse-engineering workflows.

- **Android**
  - Android-specific reverse engineering tools and resources.

- **YARA resources**
  - Rules and tools related to YARA for pattern matching in malware and binaries.

- **Open-source GitHub repository**
  - Publicly accessible code and list.
  - Community contributions via pull requests and issues.
  - Contributor list and basic repository statistics (stars, watchers, forks) available on GitHub.
