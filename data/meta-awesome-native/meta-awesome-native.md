**Tags:** meta-directory, native, awesome-lists

### Overview
Meta Awesome – Native is a curated meta-directory that aggregates “awesome” lists focused on native development environments and tooling. It serves as a central index pointing to multiple community-maintained lists related to native software development.

### Features
- Aggregates multiple awesome lists related to native development.  
- Focus on native development environments (e.g., platform-specific SDKs, compilers, toolchains).  
- Coverage of native tooling, such as build systems, debuggers, profilers, and related utilities.  
- Centralized entry point to discover specialized awesome lists instead of searching for each individually.  
- GitHub-based structure using Markdown, making it easy to browse, search, and contribute.  
- Community-driven: new awesome lists can be added via pull requests on GitHub.  
- Part of a broader “meta-awesome” collection that groups awesome lists by domain/topic.

### Pricing
- Free to access and use (public GitHub repository).
