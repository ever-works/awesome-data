# Awesome Terraform

**Website:** https://github.com/shuaibiyy/awesome-terraform#readme  
**Category:** Themed Directories  
**Tags:** devops, infrastructure, terraform

## Overview
Awesome Terraform is a curated, community-maintained directory of resources related to HashiCorp Terraform and OpenTofu. It aggregates links to modules, tools, and educational materials for working with infrastructure as code.

## Features
- **Curated resource list** focused on Terraform and OpenTofu.
- **Terraform modules**: references to reusable modules for common infrastructure patterns.
- **Tools ecosystem**: links to CLIs, linters, testing tools, and utilities that support Terraform workflows.
- **Educational resources**: guides, articles, and documentation to learn and improve Terraform usage.
- **Open source repository**: maintained on GitHub for transparency and collaboration.
- **Contribution guidelines**: documented process for adding or updating entries via pull requests.
- **Automated link checking**: GitHub Actions workflows to check for broken links, misspellings, and missing resources.
- **License file**: clear licensing for the list’s content.

## Pricing
- Free, open-source resource list (no paid plans).
