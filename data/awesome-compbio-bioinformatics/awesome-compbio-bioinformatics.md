## Overview

Curated list of FREE courses available online from top universities on Computational Biology and Bioinformatics.

## Introductory Courses

- **Introduction to Bioinformatics**: Basic concepts and tools
- **Computational Biology Fundamentals**: Core principles
- **Genomics Data Analysis**: Working with sequencing data

## Advanced Topics

- **Systems Biology**: Modeling biological systems
- **Structural Bioinformatics**: Protein structure analysis
- **Comparative Genomics**: Cross-species analysis
- **Network Biology**: Biological network analysis

## Tools and Software

- **BLAST**: Sequence similarity search
- **Bioconductor**: R packages for bioinformatics
- **BioPython**: Python libraries
- **Galaxy**: Web-based platform

## Data Analysis

- **RNA-Seq Analysis**: Transcriptomics
- **ChIP-Seq**: Epigenomics
- **Variant Calling**: Genomic variants
- **Phylogenetics**: Evolutionary analysis

## Universities Offering Free Courses

- **MIT**: MITx courses on edX
- **Harvard**: HarvardX bioinformatics
- **UCSD**: Coursera specialization
- **Stanford**: Online courses
