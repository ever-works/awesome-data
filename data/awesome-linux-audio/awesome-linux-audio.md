## Overview

Awesome Linux Audio is a curated list focused on professional audio, video, lighting, and live events applications for Linux-based production.

## Digital Audio Workstations (DAWs)

- Ardour - Professional multi-track recording and editing
- REAPER - Complete DAW for multitrack audio production
- Qtractor - Audio/MIDI multi-track sequencer
- Audacity - Multi-track audio editor and recorder
- Rosegarden - Music composition and editing environment

## Audio Plugins & Effects

- LADSPA plugins
- LV2 plugins
- VST plugins (via bridges)
- DSSI instruments

## Synthesis & Sound Design

- Hydrogen - Drum machine
- ZynAddSubFX - Realtime software synthesizer
- Bristol - Synthesizer emulations
- Yoshimi - Software synthesizer

## Audio Processing

- JACK Audio Connection Kit - Professional sound server
- PulseAudio - Sound server for POSIX systems
- PipeWire - Multimedia processing server

## MIDI Tools

- MIDI sequencers
- Virtual MIDI keyboards
- MIDI controllers
- MIDI routing tools

## Live Performance

- SuperCollider - Platform for audio synthesis
- Sonic Pi - Live coding synth
- Pure Data - Visual programming for multimedia
- Non Mixer - Modular digital audio mixer

## Video Production

- Kdenlive - Video editing
- Shotcut - Cross-platform video editor
- Blender - 3D creation suite with video editing
- OBS Studio - Video recording and live streaming

## Lighting Control

- QLC+ - DMX lighting control
- OLA (Open Lighting Architecture)

## Audio Analysis

- Spectrum analyzers
- Oscilloscopes
- Level meters
- Tuners

## Professional Features

- Multi-channel audio
- Low-latency performance
- Professional audio interfaces support
- Automation and scripting
- Session management
