# Awesome Plone

**Description**  
Awesome Plone is a community‑curated, “awesome‑style” list of add-ons, tools, and documentation for Plone, an open‑source Python CMS. It helps users discover well‑maintained, commonly used Plone extensions and related resources without having to manually sift through thousands of packages on PyPI and GitHub.

## Features

- **Curated list of Plone add-ons**  
  Focuses on widely used, well-regarded add-ons for the Plone CMS.

- **Community-driven curation**  
  Maintained collaboratively by the Plone community to share knowledge about common products and techniques.

- **Focus on current Plone and Python versions**  
  - Includes add-ons that work with the latest major Plone versions (5.2 and 6 at the time of writing).  
  - Restricts to add-ons that support Python 3.

- **Helps navigate a large ecosystem**  
  Addresses the difficulty of choosing among:  
  - 3000+ Plone-related add-ons on PyPI.  
  - 1500+ repositories in the `collective` GitHub organization.

- **Complements external discovery tools**  
  - Mentions and complements a filterable index of Plone-related PyPI packages at `pag.derico.tech`.

- **Volto / REST API awareness**  
  - Acknowledges Plone 6’s Volto React frontend and `plone.restapi`.  
  - Points users to a dedicated `awesome-volto` list for Volto-specific add-ons.
