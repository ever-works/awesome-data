# Gapminder World Demographic Databases

Healthcare-relevant demographic and public health datasets curated by Gapminder, referenced from the Awesome Public Datasets healthcare directory.

---

## Overview

Gapminder’s World demographic databases provide hundreds of statistical indicators related to health, demographics, and development. The data is accessible as bulk downloads via GitHub repositories or as individual indicators that can be previewed and exported in common spreadsheet formats.

---

## Features

### Data Access & Formats
- Bulk download of complete datasets containing hundreds of indicators.
- Data organized as loose CSV files.
- Compatible with standard spreadsheet software (e.g., Excel, Google Sheets, LibreOffice).
- Option to download individual indicators as CSV or XLSX.

### Repositories & Indicator Collections
- **Systema Globalis** (`ddf--gapminder--systema_globalis`)
  - Indicators inherited from Gapminder World.
  - Many indicators are still regularly updated.
- **Fast Track** (`ddf--gapminder--fasttrack`)
  - Indicators compiled manually by Gapminder.
- **World Development Indicators** (`ddf--open_numbers--world_development_indicators`)
  - Direct copy of World Bank’s World Development Indicators.

### Data Model & Developer Support
- Uses the **DDF (Data Description Format) data model** for more efficient work with full datasets.
- DDF model documentation available for developers who want to integrate or process the data programmatically.

### Indicator-Level Access
- Menu-based listing of all indicators available in Gapminder Tools.
- Ability to select an indicator, preview its data, then download.

### Documentation
- Dedicated **data documentation** section describing the datasets and their structure.

### Licensing & Attribution
- Data can be **reused freely**.
- Requires attribution to the **original data source** (where applicable) and to **Gapminder**.
- Example attribution: 
  - “FREE DATA FROM WORLD BANK VIA GAPMINDER.ORG, CC-BY LICENSE”.
- Further license and usage details provided in Gapminder’s free-materials page.
