## Overview
Awesome EVM Security is a curated “awesome list” focused on security for the Ethereum Virtual Machine (EVM). It aggregates tools, published audits, research resources, and educational materials related to EVM and smart contract security.

## Features
- **Curated tools for EVM security**  
  Collection of security tools used to analyze, test, and harden EVM-based smart contracts and applications.

- **Audit references**  
  Links to audits and audit-related resources for learning from real-world EVM security reviews.

- **Research resources**  
  Academic and industry research materials covering vulnerabilities, exploit techniques, and defenses in EVM environments.

- **Learning materials**  
  Educational content to understand EVM internals, common pitfalls, and best practices for secure smart contract development.

- **Awesome-list style organization**  
  Structured, link-based directory format typical of GitHub awesome lists, making it easy to browse relevant resources.
