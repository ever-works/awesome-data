## Overview

Awesome pnpm provides resources for pnpm, the fast and disk-efficient package manager.

## What is pnpm?

Fast, disk space efficient package manager that uses hard links and a content-addressable store, saving gigabytes of disk space.

## Key Features

- **Disk Efficiency** - One version of a package on disk
- **Fast** - Up to 2x faster than npm
- **Strict** - Non-flat node_modules
- **Monorepo Support** - Built-in workspace support
- **Security** - Prevents phantom dependencies

## Workspace Features

- Monorepo management

- Filtering and selection
- Parallel execution
- Shared dependencies
