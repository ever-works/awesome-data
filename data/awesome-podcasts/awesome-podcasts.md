# awesome-podcasts

**Website:** https://github.com/rShetty/awesome-podcasts  
**Category:** Themed Directories  
**Tags:** media, curated-lists, awesome-lists

## Overview
A community-maintained "Awesome"-style directory of high-quality podcasts, primarily focused on topics useful for software engineers and programmers. Hosted on GitHub and curated via open-source contributions.

## Features
- **Curated podcast list for software engineers/programmers**
  - Focus on podcasts that are educational or valuable for developers.
  - Follows the conventions and structure of other "Awesome" lists.

- **Community-maintained repository**
  - Open for contributions via pull requests.
  - Includes a `contributing.md` guide for adding or updating entries.

- **Multi-language support**
  - Separate markdown files for different languages, including:
    - Burmese (`MY.md`)
    - English (`README.md`)
    - German (`DE.md`)
    - Spanish (`ES.md`)
    - French (`FR.md`)
    - Portuguese – Brazil (`PT.md`)
    - Portuguese – Portugal (`pt_PT.md`)
    - Russian (`RU.md`)
    - Turkish (`TR.md`)

- **GitHub-native structure**
  - Organized as a standard GitHub repo for easy browsing.
  - Includes a logo (`logo.svg`).
  - Versioned via commits, branches, and tags.

## Content Structure
- Main list of important podcasts for software engineers (in `README.md`).
- Translated versions of the list in multiple language-specific `.md` files.
- Contribution guidelines in `contributing.md`.

## Pricing
- Free, open-source GitHub repository (no paid plans mentioned).
