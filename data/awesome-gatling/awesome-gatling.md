# Awesome Gatling

A curated directory of resources, tools, and articles for Gatling, the Scala-based load and performance testing framework.

**Website:** https://github.com/aliesbelik/awesome-gatling#readme

---

## Description

Awesome Gatling is an open, community-maintained “awesome list” that aggregates learning materials, examples, tools, and ecosystem projects related to Gatling. It is aimed at developers, QA engineers, and performance testers who use Gatling for load and performance testing.

---

## Features

- **Curated resource collection**
  - Organized list of links and references specifically focused on Gatling.
  - Includes documentation, tutorials, articles, and blog posts.

- **Coverage of multiple aspects of Gatling**
  - General introductions and getting-started guides.
  - Advanced usage patterns and best practices for performance testing.
  - Examples and sample projects demonstrating Gatling in practice.

- **Tooling and integrations**
  - Links to tools that extend or complement Gatling-based testing workflows.
  - References to libraries and plugins in the Gatling ecosystem.

- **Community and contribution guidelines**
  - Public GitHub repository structure with contribution instructions (CONTRIBUTING.md).
  - Code of conduct (CODE-OF-CONDUCT.md) for contributors.

- **Open-source licensing**
  - Licensed under an open-source license (see LICENSE.md in the repository) for reuse and modification subject to license terms.

- **Part of the Awesome ecosystem**
  - Conforms to the Awesome List standards (badged via awesome.re).
