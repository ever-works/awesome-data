## Description
Awesome jQuery is a curated directory of jQuery plugins, resources, and tools focused on DOM manipulation and front‑end web development. It organizes community-maintained links and references into a single, browsable list.

## Features
- **Curated plugin directory** for jQuery-based components and utilities.
- **Resources collection** including tutorials, guides, and learning materials related to jQuery and front‑end development.
- **Tools listing** covering helpers for DOM manipulation, interaction, and workflow around jQuery projects.
- **Community-maintained repository** with contribution guidelines (`CONTRIBUTING.md`) for adding and updating entries.
- **Code of Conduct** (`CODE_OF_CONDUCT.md`) governing community participation and contributions.
- **Awesome list compliant** (part of the broader “awesome” list ecosystem and standards).
- **Open structure** via `README.md` and supporting docs that can be cloned and extended in other projects.
