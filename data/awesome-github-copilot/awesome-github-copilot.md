## Overview

Awesome GitHub Copilot is the official repository with community-contributed customizations for GitHub Copilot. Now features a dedicated website, Learning Hub, and plugin system.

## Repository Contents

- **175+ agents**: Specialized AI assistants for specific tasks
- **208+ skills**: Reusable capabilities
- **176+ instructions**: Custom prompts and configurations
- **48+ plugins**: Bundled collections for specific domains
- **7 agentic workflows**: Complex multi-step processes
- **3 hooks**: Event-driven automations

## Website Features

### Learning Hub
Curated articles, walkthroughs, and reference material covering:
- Core concepts (agents, skills, instructions)
- Hands-on guides for hooks and workflows
- MCP servers integration
- Copilot coding agent tutorials

### Plugin System
Themed collections for specific domains:
- Frontend development
- Python projects
- Azure cloud
- Team-specific workflows

Plugins bundle related agents, skills, and commands into single installable packages.

## Customization Types

### Agents
Specialized AI assistants for tasks like:
- Code review
- Test generation
- Documentation writing
- Refactoring

### Skills
Reusable capabilities:
- Language-specific expertise
- Framework knowledge
- Best practices
- Design patterns

### Instructions
Custom prompts that:
- Guide behavior
- Set context
- Define output format
- Specify constraints
