# Awesome ChatGPT

**URL:** https://github.com/sindresorhus/awesome-chatgpt#readme  
**Category:** Themed Directories  
**Brand:** OpenAI

## Overview
Awesome ChatGPT is a curated GitHub directory of tools, prompts, browser extensions, and other resources built around ChatGPT. It organizes community-built projects and references into a single, browsable list focused on ChatGPT-related use cases.

## Features
- **Curated resource list** of ChatGPT-related projects and references.
- **Focus on ChatGPT ecosystem**, including:
  - Tools and utilities that integrate with or extend ChatGPT.
  - Collections of prompts and prompt libraries.
  - Browser extensions and plugins.
  - Other AI resources specifically centered on ChatGPT.
- **GitHub-based directory** structure, making it easy to browse, search, and contribute.
- **Community contributions** via pull requests and issues (standard GitHub workflow).
- **Open, public access** to the list content (subject to GitHub repository visibility).

## Pricing
- Accessing and using the Awesome ChatGPT list on GitHub is **free**.
- Individual tools and resources listed in the directory may have their own separate pricing, which is managed outside of the Awesome ChatGPT repository.
