## Overview

A curated list of reasonably mature or actively developed open source embedded scripting languages to use in your applications, including Datalog implementations.

## Features

- **Language Diversity**: Scripts for C++, Rust, Python, and other host languages
- **License Information**: Copyleft licenses clearly marked for informed decision-making
- **Mature Projects**: Focus on production-ready scripting languages
- **Multiple Paradigms**: Functional, imperative, logic programming options
- **Benchmark Data**: Performance comparisons available via related projects

## Notable Implementations

- **Rhai**: Embedded scripting language for Rust with JavaScript-like syntax and dynamic typing
- **ChaiScript**: Header-only embedded scripting for C++ with modern C++ features
- **Lua**: Lightweight, embeddable scripting language (widely used)
- **JavaScript engines**: V8, QuickJS, and others for embedding

## Use Cases

- Game modding and scripting
- Application configuration and automation
- Plugin systems
- Domain-specific languages
- Runtime extensibility
