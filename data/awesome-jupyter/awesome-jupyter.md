# Awesome Jupyter

**Description**  
Awesome Jupyter is a curated directory of projects, libraries, tools, extensions, notebooks, and educational resources across the Jupyter ecosystem. It focuses on interactive computing and data science workflows using Jupyter.

**Website**  
<https://github.com/markusschanta/awesome-jupyter#readme>

## Features

- **Curated list format**  
  - Maintained as an “Awesome” style list on GitHub.  
  - Organizes links to high-quality Jupyter-related tools and resources.

- **Coverage of the Jupyter ecosystem**  
  - Focuses on projects and libraries built around Jupyter.  
  - Includes items for working with live code, equations, visualizations, and narrative text within notebooks.

- **Resource types included**  
  - Tools and extensions for Jupyter.  
  - Notebooks and example projects.  
  - Libraries that enhance or integrate with Jupyter.  
  - General resources for learning and using Jupyter in data science workflows.

- **Open and community-driven**  
  - Hosted on GitHub with a CONTRIBUTING guide, enabling community submissions and updates.  
  - Licensed under CC-BY-SA-4.0 for sharing and adaptation with attribution.

- **Ecosystem context**  
  - Built around Jupyter, an open-source web application for creating and sharing documents containing live code, equations, visualizations, and narrative text.
