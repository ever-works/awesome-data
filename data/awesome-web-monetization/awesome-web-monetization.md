## Awesome Web Monetization

**Category:** Themed Directories  
**Website:** https://github.com/thomasbnt/awesome-web-monetization#readme

### Description
Awesome Web Monetization is a curated "awesome list" directory focused on the Web Monetization standard. It aggregates tools, packages, articles, documentation links, and other resources related to enabling direct, browser-based payments via the Web Monetization JavaScript API.

### Features
- Curated collection of resources specifically about Web Monetization
- Includes:
  - Packages and modules for implementing Web Monetization
  - Tools and utilities that support Web Monetization use cases
  - Articles and learning materials
  - Documentation links and reference material
- Organized as an open-source GitHub repository
- Community-driven contributions via standard open-source workflows (issues, pull requests)
- Includes supporting project files:
  - `code-of-conduct.md` for community conduct guidelines
  - `contributing.md` for contribution instructions
  - `LICENSE` defining usage terms
  - `package.json` and `yarn.lock` for project/dependency metadata
- Part of the wider Awesome ecosystem (compliant with Awesome list conventions)

### Technology / Standard Highlighted
- Focuses on the **Web Monetization** JavaScript browser API, a web service that enables sending money directly from within the browser.
