## Overview

Awesome Fish is a collection of resources for fish shell, designed to be user-friendly with sensible defaults. Fish features syntax highlighting, auto-suggestions, and tab completions that work out of the box without configuration.

## Features

- Official fish shell documentation
- Fisher plugin manager and plugins
- Oh My Fish framework
- Themes and prompt customizations (Tide, Starship)
- Auto-suggestion configurations
- Command completion functions
- Abbreviations and aliases
- Functions and event handling
- Universal variables and settings
- Integration with other tools (fzf, ripgrep)
- Vi mode support
- Configuration examples
- Migration from bash/zsh
- Plugin development guides
- Performance tips
- Web-based configuration interface
- Community resources
- Video tutorials
