# Awesome Public Datasets – Government Data: Israel Open Data Portal

**URL:** https://data.gov.il  
**Provider / Brand:** Government of Israel  
**Category:** Themed Directories  
**Tags:** datasets, government, open-data

## Overview
The Israel Open Data Portal (data.gov.il) is the central governmental repository for public datasets from Israeli government ministries and public bodies. It aggregates government data and makes it available for public use, with tools for browsing, searching, and programmatic access via APIs.

## Features

### Data Catalog & Browsing
- Central portal for **government data repositories** from all Israeli ministries and government bodies.
- **Datasets directory** with search and browse functionality.
- **Organizations directory** to view datasets by originating government body.
- **Popular datasets** section highlighting frequently used or important datasets.
- **Popular tags** (thematic categories) such as:
  - Transportation (תחבורה)
  - Finance / Treasury (אוצר)
  - Population (אוכלסייה)
  - Agriculture (חקלאות)
  - Tourism (תיירות)
  - Education (חינוך)
- Example datasets presented on the homepage, e.g.:
  - **Master Code Lists (מילוני מאסטר / Code Lists)** – standardized value and classification lists.
  - **Vehicles with Disabled Parking Permits** – list of car numbers authorized to carry a disabled parking permit by the Ministry of Transport.

### API Access & Examples
The portal supports programmatic access to datasets with documented query patterns and examples, including:

- **Paginated queries (שאילתה עם דיפדוף)**
  - Control the number of returned records using `limit`.
  - Choose the starting row using `offset`.

- **Field-based filtering (שאילתה לפי שדות)**
  - Use `filter` parameters in API calls to retrieve records matching specific field values.

- **Listing datasets (הצגת רשימת המאגרים)**
  - API example for retrieving the list of available datasets on the site.

- **Listing files within a dataset (הצגת הקבצים במאגר)**
  - API example to retrieve the files/resources contained in a dataset by dataset name.

- **Data consumption via crawling (צריכת נתונים על ידי crawling)**
  - Option to use or define a `User-Agent` for crawling-based access to data.

- **Filtering values by column / page (שאילתה סינון מספר ערכים לפי עמוד)**
  - API example for filtering a specified number of values by a given column.

- Additional API examples are accessible directly through the portal ("הצג שאילתה" / "show query").

### Documentation & Guides

- **About page** explaining the purpose and framework of the government data repository.
