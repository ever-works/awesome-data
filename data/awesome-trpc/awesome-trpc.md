## Overview

Awesome tRPC provides resources for tRPC, enabling end-to-end typesafe APIs without code generation.

## What is tRPC?

tRPC allows you to build fully typesafe APIs without schemas or code generation, with TypeScript inference across client and server.

## Key Features

- **Type Safety** - Full TypeScript inference
- **No Code Generation** - Types inferred automatically
- **Framework Agnostic** - Works with any framework
- **Real-time** - WebSocket support
- **Batching** - Request batching
- **Caching** - Built-in caching support

## Framework Integration

- **Next.js** - Official Next.js integration
- **React** - React Query integration
- **Express** - Express adapter
- **Fastify** - Fastify plugin
- **SvelteKit** - Svelte integration
