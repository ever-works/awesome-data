# awesome-vscode

**Description**  
A curated "awesome list" of Visual Studio Code extensions, packages, and learning resources. It serves as a thematic directory within the wider awesome-list ecosystem, focused entirely on enhancing and customizing VS Code.

**Website**  
https://github.com/viatsko/awesome-vscode

**Category**  
- Themed directories

**Tags**

## Features

### Scope and Structure
- Curated list of Visual Studio Code extensions, packages, and resources.  
- Organized with a detailed table of contents for easy navigation.  
- Focuses on practical, developer-oriented tooling and learning material.

### Main Sections
- **Official**  
  Links to official VS Code resources (editor downloads, docs, marketplace, etc.).

- **Syntax**  
  Resources and extensions improving syntax highlighting and language support.

- **Migrating from Other Editors**  
  Guides and tools to help users transition from:
  - Vim  
  - Atom  
  - Sublime Text  
  - Visual Studio  
  - IntelliJ IDEA  
  - Camel Humps-style navigation

- **Using VS Code with Particular Technologies**  
  Extensions and setup notes for using VS Code with specific frameworks, languages, and platforms.

- **Lint and IntelliSense**  
  Language-specific linting, IntelliSense, and tooling collections, including:
  - 1C  
  - AutoHotkey (e.g., AutoHotkey Plus)  
  - Bash (e.g., Bash IDE, Bash Debug, Shellman)  
  - C++ (plus a “More” subsection)  
  - C#, ASP .NET and .NET Core  
  - Clojure (e.g., Calva)  
  - CSS (e.g., CSS Peek)  
  - Go  
  - Groovy  
  - Haskell  
  - HLSL  
  - Shell  
  - Java  
  - JavaScript, including:
    - General linters  
    - Framework-specific tooling  
    - Debugger for Chrome  
    - Facebook Flow  
    - TypeScript  
  - MATLAB  
  - Markdown (e.g., markdownlint, Markdown All in One, Markdown Emoji)  
  - PHP (e.g., PHP Tools, IntelliSense helpers, Laravel, Twig, Smarty, other extensions, further reading)  
  - POV-Ray  
  - Python (including TensorFlow-related tooling)  
  - ReasonML  
  - Rust  
  - Terraform

- **GitHub Integration**  
  Extensions and tools for integrating GitHub workflows into VS Code, including:
  - Core GitHub integration extensions  
  - GitHub Pull Requests and Issues  
  - GistPad  
  - GitHub Actions  
  - GitHub Repositories  
  - GitHub Pull Request Monitor

- **Productivity**  
  Extensions focused on workflow and productivity enhancements, including:
  - ARM Template Viewer  
  - Azure Cosmos DB tools  
  - Azure IoT Toolkit  
  - Bookmarks  
  - Browser Preview (deprecated)  
  - Color Tabs  
  - Create tests helpers  
  - Dendron (note-taking / knowledge management)  
  - Deploy-related tools  
  - Duplicate Action utilities  
  - Error Lens  
  - Toggle utilities  
  - ES7 React/Redux/GraphQL/React-Native snippets (and related snippets like `Gi`)  
  - Git History  
  - Git Project Manager  
  - GitLink  
  - GitLens  
  - Git Indicators  
  - GitLab Workflow  
  - Gradle Tasks  
  - Icon Fonts  
  - Import Cost  
  - Jira and Bitbucket integration  
  - JS Parameter Annotations  
  - Jumpy  
  - Kanban  
  - Live Server
