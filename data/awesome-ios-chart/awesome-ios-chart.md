## Description
A curated directory of iOS chart libraries, collecting charting components for iOS apps written in Objective-C and Swift.

## Features
- Curated “awesome list” of iOS chart libraries.
- Focus on data visualization and charting components for iOS applications.
- Includes libraries implemented in **Objective-C** (listed in `Objective-C.md`).
- Includes libraries implemented in **Swift** (listed in `Swift.md`).
- Organized as a GitHub repository with separate markdown files per language.
- Licensed open source (see `LICENSE` file in the repository).
