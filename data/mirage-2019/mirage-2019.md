# MIRAGE-2019

**Category:** Datasets  
**Slug:** `mirage-2019`

## Overview
MIRAGE-2019 is a human-generated mobile traffic analysis dataset from the COMICS group at the University of Napoli. It provides labeled mobile network traffic traces to support research in traffic classification and user/application behavior analysis. The dataset is collected using the MIRAGE architecture, a reproducible framework for mobile-app traffic capture and ground-truth creation.

**Source URL:** http://traffic.comics.unina.it/mirage/

## Features
- Human-generated mobile network traffic traces
- Labeled data suitable for classification tasks
- Focus on mobile traffic behavior analysis
- Collected using the MIRAGE reproducible architecture for:
  - Mobile-app traffic capture
  - Ground-truth creation
- Part of a family of MIRAGE datasets (including MIRAGE-AppAct-2024, MIRAGE-COVID-CCMA-2022, MIRAGE-Video)
- Curated and referenced in networking and traffic-analysis dataset directories
