# Medicare Data Engine

**Brand:** Centers for Medicare & Medicaid Services (CMS)  
**Category:** Themed Directories  
**Type:** Open data portal / dataset directory  
**Website:** https://data.medicare.gov

## Overview

Medicare Data Engine is the official Medicare Open Data portal operated by CMS. It provides public access to a wide range of Medicare-related datasets, cataloged as part of the broader "Awesome Public Datasets" healthcare directory. The portal is intended for research, analysis, and data science or machine learning projects involving U.S. Medicare data.
