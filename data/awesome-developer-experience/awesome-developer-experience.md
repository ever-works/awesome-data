## Overview

Awesome Developer Experience curates resources that help improve the day-to-day experience of developers, from onboarding to deployment, focusing on reducing friction and increasing productivity.

## Features

- Developer onboarding best practices and templates
- Documentation tools and frameworks (Docusaurus, MkDocs, GitBook)
- Local development environment setup and standardization
- CLI tools and terminal productivity enhancers
- Code review and collaboration tools
- Testing and debugging utilities
- API design and developer portals
- Error tracking and monitoring for developers
- Developer advocacy and community building resources
- Metrics for measuring developer productivity
