## Overview

Awesome Firebase Genkit is a curated "awesome list" of community plugins, tools, examples, and learning resources around Firebase Genkit, an open‑source framework for building AI‑powered applications and features.

The repository aggregates links and references from across the Genkit ecosystem to help developers discover useful components and educational materials.

## Features

- **Curated ecosystem directory**  
  - Centralized list of resources related to Firebase Genkit and the broader Genkit ecosystem.  
  - Organized as a standard GitHub "awesome list" for easy browsing.

- **Plugins and tools**  
  - Links to third‑party and community plugins that extend Genkit functionality.  
  - References to auxiliary tools that integrate with or support Genkit‑based workflows.

- **Examples and sample projects**  
  - Collection of example implementations demonstrating how to build AI‑powered apps and features with Genkit.  
  - Code samples intended to illustrate real‑world usage patterns.

- **Talks, articles, and learning materials**  
  - List of conference talks, videos, and presentations about Genkit.  
  - Articles, blog posts, and written guides explaining concepts, patterns, and best practices.

- **Community contributions**  
  - Public contribution workflow documented in `CONTRIBUTING.md`.  
  - Code of Conduct for contributors and participants.  
  - Tracked by external awesome‑list indexers for visibility and quality.

- **Open source governance**  
  - Licensed under an open source license (see `LICENSE` file in the repository).  
  - Repository health and formatting enforced via linting GitHub Actions.
