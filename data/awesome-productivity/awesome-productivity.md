# Awesome Productivity

**URL:** https://github.com/jyguyomarch/awesome-productivity#readme  
**Category:** Themed Directories  
**Tags:** productivity, awesome-lists

## Overview
Awesome Productivity is a curated, GitHub-hosted list of tools, techniques, resources, and articles focused on improving personal and professional productivity. It serves as a thematic directory, organizing productivity-related content from across the web into a single, structured resource.

## Features
- **Curated Directory Format**  
  - Organized as an "awesome list" on GitHub.  
  - Focuses specifically on productivity-related material.

- **Broad Productivity Scope**  
  - Covers both **personal** productivity (habits, time management, focus) and **professional** productivity (workflows, collaboration, efficiency at work).

- **Resource Types**  
  - Tools and apps that support productivity.  
  - Techniques and methods (e.g., workflows, frameworks, approaches).  
  - Articles, guides, and written resources.  
  - General reference links related to productivity.

- **Centralized Access**  
  - Single GitHub README entry point that links out to the full collection.  
  - Publicly accessible via the repository URL.
