## Features
- **Long-run time coverage**
  - Historical macroeconomic data spanning multiple centuries.
- **Cross-country comparability**
  - Data structured to compare economic performance across many countries.
- **Key economic indicators**
  - GDP per capita.
  - Population.
  - Aggregated measures suitable for macroeconomic and growth analysis.
- **Multiple documented releases**
  - 2013 Maddison Project Database release.
  - 2018 Maddison Project Database release.
  - 2020 Maddison Project Database release.
  - 2023 Maddison Project Database release.
- **Legacy Maddison database version**
  - Access to the last version of the original Maddison database from 2010.
- **Versioned datasets and documentation**
  - Each release is provided as a separate version with associated documentation and methodology (implied by versioned release structure).
- **Academic and research orientation**
  - Designed for use in economic history, macroeconomics, and comparative development research.
