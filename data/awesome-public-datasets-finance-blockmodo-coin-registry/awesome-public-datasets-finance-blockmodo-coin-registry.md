# Awesome Public Datasets – Finance / Blockmodo Coin Registry

## Overview
Blockmodo Coin Registry is a GitHub-hosted, publicly accessible collection of JSON-formatted metadata for over 1,500 cryptocurrency tokens. It is part of the Awesome Public Datasets (finance category) and is used by websites, exchanges, trading terminals, DEXs, developers, and news publications to display accurate reference data about coins.

- **Type:** Open dataset / metadata registry
- **Category:** Themed directories → Finance, Blockchain
- **Format:** JSON files (one or more per coin)
- **Source:** https://github.com/Blockmodo/coin_registry
- **License:** MIT

## Features
- **Global coin registry**
  - Central collection of JSON files describing 1,500+ cryptocurrency tokens.
  - Organized under a `coins` directory in the repository.

- **Structured JSON metadata per coin**
  - Each coin has a JSON “payload” describing its properties.
  - Designed for programmatic consumption by apps and services.

- **Rich coin information fields** (as described in the repository summary)
  - Chat rooms and community links.
  - Social and community channels.
  - Block explorers.
  - Contact or project information.
  - Other reference/auxiliary metadata used for display and integration.

- **Developer-focused design**
  - Intended for:
    - Website and app developers needing up-to-date coin info.
    - Exchange and DEX operators integrating token metadata.
    - Terminal and dashboard developers.
    - News and data publications displaying coin details.
  - JSON format simplifies integration into frontends, backends, and tools.

- **Open source repository structure**
  - `coins/` directory containing the registry files.
  - `README.md` with introduction and usage information.
  - `LICENSE` documenting MIT license.
  - `CODE_OF_CONDUCT.md` for community contributions.
  - `.gitignore` for repository housekeeping.

- **Ecosystem usage**
  - Used directly by Blockmodo (https://blockmodo.com).
  - Referenced by DEXs, developers, and exchanges as a common registry of token metadata.
