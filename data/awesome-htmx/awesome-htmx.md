## Overview

Awesome HTMX provides resources for HTMX, a library that gives you access to AJAX, CSS Transitions, WebSockets and Server Sent Events directly in HTML through attributes. HTMX enables building modern UIs with the simplicity of hypertext.

## Features

- Official HTMX documentation and essays
- Server-side framework integrations (Django, Laravel, Rails, FastAPI)
- HTMX extensions (SSE, WebSocket, client-side-templates)
- Hypermedia-driven architecture patterns
- Forms and validation with HTMX
- Infinite scroll and lazy loading implementations
- Modal dialogs and dynamic content
- Real-time updates with Server-Sent Events
- WebSocket integration for bidirectional communication
- Search and filtering patterns
- Optimistic UI updates
- Error handling strategies
- Authentication flows with HTMX
- Testing HTMX applications
- Performance considerations
- Accessibility best practices
- Real-world application examples
- Migration from SPA frameworks
