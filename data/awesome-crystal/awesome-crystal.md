# Awesome Crystal

**Website:** https://github.com/veelenga/awesome-crystal#readme  
**Category:** Themed Directories  
**Tags:** awesome-lists, crystal, developer-tools

## Overview
Awesome Crystal is a curated directory of Crystal language libraries, frameworks, tools, and related software. It focuses on stable and useful Crystal projects and resources, following the style and standards of the broader Awesome list ecosystem.

## Features
- **Curated list of Crystal resources**
  - Collects Crystal libraries, frameworks, tools, and software in one place.
  - Emphasis on projects that are stable and broadly useful to the Crystal community.

- **Language-specific focus (Crystal)**
  - Dedicated to the Crystal programming language and its ecosystem.
  - Helps developers quickly discover shards (Crystal packages) and related tools.

- **Organized by categories**
  - Includes sections such as:
    - Algorithms and Data Structures
    - Blockchain
    - C bindings
  - Additional categories are available in the full README to cover a wide range of Crystal use cases.

- **Integration with shard search**
  - Directs users to [shards.info](https://shards.info) to search for Crystal shards/packages.

- **Community-driven curation**
  - Open to contributions from the community via pull requests.
  - Maintains [contribution guidelines](https://github.com/veelenga/awesome-crystal/blob/master/.github/CONTRIBUTING.md) for consistent quality and format.

- **Alignment with Awesome ecosystem**
  - Inspired by and compliant with the [Awesome](https://github.com/sindresorhus/awesome) list conventions.
  - Uses the Awesome badge to indicate adherence to Awesome list standards.

- **Repository structure and metadata**
  - Contains standard project files such as:
    - `README.md` – main index of Crystal resources.
    - `LICENSE` – license information for the list.
    - `CODE_OF_CONDUCT.md` – community behavior guidelines.
    - `.github` configuration (including contributing documentation).
    - `shard.yml` and `shard.lock` for repository tooling/tests related to the list.
    - `logo/` assets for Awesome Crystal branding.

## Pricing
- Not applicable. Awesome Crystal is an open, GitHub-hosted curated list and is free to use and contribute to.
