# Learn to Program

**Category:** Themed Directories  
**Creator / Curator:** karlhorky  
**Type:** Curated GitHub directory of learning resources

## Overview

Learn to Program is an "awesome-style" curated list of educational resources for building a foundation in web development and programming. It focuses on collecting high‑quality, primarily beginner‑friendly materials, highlighting whether each resource is free or paid and the main topics it covers.

## Features

- **Curated directory format**
  - Organized as a GitHub README list.
  - Focuses on foundation in web development and programming.
  - Emphasizes quality and discoverability of learning resources.

- **Beginner-focused resources**
  - Collection targeted at beginners and early‑stage learners, with some intermediate content.

- **Platform & format variety**
  - Interactive platforms, full courses, bootcamp prep, games, tutorials, and guides.

- **Price-type labels for each resource**
  - Each entry indicates whether the platform is Free, Freemium, or Paid.

- **Topical coverage highlights**
  - Each resource lists the main technologies or concepts taught.

- **Representative resources in the list (Beginner section)**
  - **Codecademy** – Freemium, interactive coding platform.  
    _Topics: HTML, CSS, JavaScript, jQuery, Python, Ruby, Rails, PHP, C++, Java._
  - **Khan Academy's Hour of Code** – Free, 1‑hour interactive introductions.  
    _Topics: HTML, CSS, JavaScript, SQL._
  - **UpLeveled Bootcamp Prep Course** – Freemium, prep for web dev bootcamps.  
    _Topics: HTML, CSS, JavaScript, Node.js, Git, GitHub._
  - **Treehouse** – Paid, structured courses on building websites and apps.  
    _Topics: Web Design, Front End Web Development, Rails, iOS, Android, PHP._
  - **Learn CSS Layout** – Free, focused tutorial on CSS layout.  
    _Topics: CSS._
  - **Udemy (Programming / Development)** – Freemium course marketplace (quality varies by course).  
    _Topics: HTML, CSS, JavaScript, Ruby, Rails, Python, iOS, Android._
  - **Code Avengers** – Freemium, basic web and app development courses.  
    _Topics: HTML, CSS, JavaScript._
  - **Shay Howe's Learn to Code HTML & CSS** – Free, beginner–intermediate guides.  
    _Topics: HTML, CSS, JavaScript._
  - **HTML Dog** – Free, beginner and intermediate web development guides.  
    _Topics: HTML, CSS, JavaScript._
  - **freeCodeCamp** – Free, project‑based curriculum; learn to code while contributing to nonprofits.  
    _Topics: HTML, CSS, JavaScript, Databases, Git & GitHub, Node.js, React.js, D3.js, Python._
  - **Vertabelo Academy** – Free, interactive SQL courses with exercises and quizzes.  
    _Topics: SQL, database concepts._
  - **The Odin Project** – Free, beginner–intermediate full‑stack curriculum with Ruby and JavaScript paths.  
    _Topics: HTML, CSS, JavaScript, Ruby, Rails._
  - **MDN Learning Area** – Free, official Mozilla guides on web fundamentals and modern tooling.  
    _Topics: HTML, CSS, JavaScript, accessibility, performance, React, Ember, Vue, Svelte, Angular, Git, GitHub._
  - **GitHub Skills** – Free, self‑paced, interactive GitHub projects.  
    _Topics: Git, GitHub._
  - **Grid Garden** – Free, browser game teaching CSS Grid.  
    _Topics: CSS Grid (CSS layout)._ 

*(The full directory may contain additional sections and resources beyond those listed here.)*

## Pricing

Learn to Program itself is a free, open GitHub directory.  
Individual resources in the list have varying pricing models, which are noted per item:

- **Free** – e.g., Khan Academy's Hour of Code, Learn CSS Layout, freeCodeCamp, Vertabelo Academy, The Odin Project, MDN Learning Area, GitHub Skills, Grid Garden, HTML Dog, Shay Howe's guides.
- **Freemium** – e.g., Codecademy, UpLeveled Bootcamp Prep Course, Udemy, Code Avengers.
- **Paid** – e.g., Treehouse.
