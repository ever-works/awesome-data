# Awesome Esports

**URL:** https://github.com/strift/awesome-esports#readme  
**Category:** Themed Directories  
**Tags:** awesome-lists, games, media

## Overview
Awesome Esports is a curated, open-source directory of projects related to esports and competitive gaming. It organizes tools, libraries, documentation, and applications used around esports games and tournaments.

## Features
- **Curated open-source list**
  - Focused specifically on esports-related projects.
  - Includes active projects, with a separate archive for unmaintained ones.

- **Clear categorization of projects**
  - **Libraries** (programming libraries organized by use case):
    - APIs & Web Services
    - Authentication
    - Data parsing and analysis
    - Team management
    - Tournaments management
  - **Developer tools**
    - Tooling aimed at developers working with esports data, platforms, or integrations.
  - **Unofficial documentation**
    - Community-maintained documentation of editor APIs related to esports ecosystems.
  - **Applications & Tools**
    - General-purpose tooling and applications for esports.
    - Game-specific sections, including:
      - Counter-Strike: Global Offensive
      - Dota 2
      - League of Legends
  - **Education**
    - Learning resources around esports technologies or development (where provided in the list).

- **Archive of unmaintained projects**
  - Separate `ARCHIVE.md` for projects that are no longer maintained but remain useful for reference or educational purposes.

- **Community standards & contributions**
  - `CODE-OF-CONDUCT.md` defining community behavior.
  - `CONTRIBUTING.md` with contribution guidelines for adding or updating resources.
  - Open to community contributions via GitHub.

- **Open license**
  - Distributed under an open-source license (see `LICENSE` file in the repository).

## Typical Use Cases
- Discovering APIs, tools, and libraries for building esports-related applications.
- Finding data analysis and tournament management tooling for esports.
- Exploring game-specific utilities for CS:GO, Dota 2, and League of Legends.
- Using archived projects as examples or references for learning.

## Pricing
- Awesome Esports is a free, open-source resource directory. There are no paid plans or pricing tiers.
