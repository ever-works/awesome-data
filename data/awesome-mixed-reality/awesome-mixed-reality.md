## Overview

Awesome Mixed Reality provides comprehensive resources for developing MR applications that seamlessly blend virtual content with the physical environment, supporting devices like Meta Quest, HoloLens, and Magic Leap.

## Features

- Unity XR and Meta XR SDK documentation and tutorials
- Spatial anchors and persistent content solutions
- Hand tracking and eye tracking implementations
- Passthrough and scene understanding APIs
- Physics interactions between virtual and real objects
- Multi-user and shared experiences frameworks
- Performance optimization for MR applications
- Accessibility considerations for MR experiences
- MR design patterns and best practices
- Sample projects and open-source applications
