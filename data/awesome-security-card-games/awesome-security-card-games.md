# Awesome Security Card Games

**Website:** https://github.com/Karneades/awesome-security-card-games  
**Category:** Themed Directories  
**Tags:** security, games

## Overview
Awesome Security Card Games is a curated directory of security-focused card games (often used as tabletop exercises). These games are designed to help people practice and discuss different areas of security in an engaging, interactive way.

## Features
- **Curated list of security card games**
  - Focused specifically on games related to information and cyber security.
  - Includes games that can be used as tabletop exercises.
- **Covers multiple security domains**
  - Application Security
  - Cryptography
  - Data Privacy
  - Incident Response
  - Threat Modeling
- **Training and discussion focus**
  - Games intended to help train security skills.
  - Facilitates structured discussion around security topics.
- **Additional resources section**
  - "Various Resources" category for related learning materials or references.
- **Open contribution model**
  - Public contribution guidelines for adding or updating games.
  - Code of conduct for contributors.
- **Open-source licensing**
  - Repository includes a LICENSE file specifying usage terms.

## Use Cases
- Finding security-related card games for workshops, meetups, or training sessions.
- Planning tabletop exercises for incident response, threat modeling, or application security.
- Discovering engaging formats to teach or discuss security and privacy topics.
