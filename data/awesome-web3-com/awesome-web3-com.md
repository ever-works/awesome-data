## Overview

Awesome Web3 is a curated directory of resources, libraries, tools and more for the Web3 ecosystem, last updated February 2026.

## Coverage

- **Blockchain Platforms**: Resources for Ethereum, Polygon, Solana, and other chains
- **Smart Contracts**: Solidity libraries, frameworks, and best practices
- **Web3 Libraries**: JavaScript/TypeScript libraries for Web3 integration
- **Development Tools**: IDEs, testing frameworks, deployment tools
- **DeFi Projects**: Decentralized finance protocols and applications
- **NFT Resources**: Tools and platforms for NFT development
- **Learning Materials**: Tutorials, courses, and documentation
