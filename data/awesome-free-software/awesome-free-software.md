## Overview

Awesome Free Software is a curated "awesome list" of software projects and related resources that are **free as in freedom**. Every project listed is distributed under licenses that allow anyone to use, modify, and share the software for any purpose (e.g., MIT, GPL, and other FOSS licenses).

## Features

- **Focus on software freedom**
  - Only includes projects under licenses that grant the right to use, modify, and redistribute for any purpose.
  - Highlights common free/open-source licenses (e.g., MIT, GNU GPL) and links to broader license information.

- **Curated software directory**
  - Organized by practical categories to help discover free software alternatives:
    - Audio
    - Command Line Tools
    - Communication
    - File Management
    - File Sharing
    - Graphics
    - IDEs
    - Office
    - Operating Systems
    - Other
    - Payments
    - Scientific Work
    - Screencasting
    - Text Editors
    - Utilities
    - Video
    - Virtualization
    - Web Applications
    - Web Servers

- **Hardware-related sections**
  - Companies working with or producing open/free hardware.
  - Hardware resources relevant to free/open ecosystems.

- **Resource collections beyond software**
  - Articles about free software and related topics.
  - Blogs covering free software and open-source.
  - Businesses involved with free software.
  - Discovery tools and services for finding free/open software.
  - Events related to free and open-source software.
  - Licensing information and references.
  - Organizations active in the free software movement.
  - Other curated lists related to free/open software.
  - People profiles and notable contributors.
  - Talks and presentations.
  - Documentaries.

- **Community-driven curation**
  - Open GitHub repository with contribution guidelines (`CONTRIBUTING.md`).
  - Multiple contributors maintaining and expanding the list.

- **Open content licensing**
  - List content is licensed under **CC0 1.0**, dedicating it to the public domain for maximum reuse.
