# Machine Learning Tutorials (Awesome)

**Website:** https://github.com/ujjwalkarn/Machine-Learning-Tutorials#readme  
**Category:** Themed Directories  
**Tags:** machine-learning, education, resources

## Overview
Machine Learning Tutorials (Awesome) is a curated, topic-wise directory of machine learning and deep learning tutorials, articles, and educational resources. It follows the "awesome list" style and aggregates learning materials from courses, videos, books, and community lists.

## Features
- **Topic-wise curation**
  - Resources organized by machine learning and deep learning topics.
  - Focus on both conceptual understanding and practical application.

- **Introduction and foundational learning resources**
  - Links to major introductory ML courses and collections, including:
    - Machine Learning Course by Andrew Ng (Stanford / Coursera).
    - AI/ML YouTube courses (dair-ai curated list).
    - Curated lists of machine learning resources from third-party sites (e.g., Hackr.io).
    - In-depth introduction to ML through expert video series (e.g., 15-hour video collections).
    - "An Introduction to Statistical Learning" as a foundational statistics/ML text.
    - Lists of university machine learning courses.
    - Guides like "Machine Learning for Software Engineers" and "Dive into Machine Learning".

- **Related awesome and curated lists**
  - Links out to broader or adjacent collections, including:
    - Awesome Machine Learning (frameworks, libraries, software).
    - Awesome data visualization libraries and resources.
    - Awesome Data Science repositories focused on real-world application.
    - The Open Source Data Science Masters curriculum.

- **Language-specific companion lists**
  - Separate curated lists maintained by the same author:
    - R tutorials for Data Science, NLP, and Machine Learning.
    - Python tutorials for Data Science, NLP, and Machine Learning.

- **Community contributions**
  - Public contributing guidelines (`contributing.md`) for adding or improving resources.
  - Maintained as an open GitHub repository with version history and licensing information.

- **Open license**
  - Repository includes a `LICENSE` file (open-source licensing for reuse/contribution).
