# Awesome Digital Humanities

**Category:** Themed Directories  
**Tags:** digital-humanities, research, data-science

## Overview
Awesome Digital Humanities is a curated, GitHub-hosted directory of software and resources for humanities scholars who use quantitative, computational, and digital methods in their research and teaching.

## Features
- Curated “awesome list” focused specifically on digital humanities.
- Emphasis on tools and resources suited for quantitative and computational approaches in the humanities.
- Organizes links to software and resources useful for:
  - Humanities research projects
  - Data-driven and computational analysis
  - Digital scholarship workflows
- Open repository structure with standard project files:
  - README, LICENSE, CONTRIBUTING, CODE_OF_CONDUCT, SECURITY
  - Assets and layout files enabling a browsable site version of the list.
- Community-driven maintenance via GitHub (issues and pull requests).
