## Meta Awesome - PHP Installation

**Category:** Meta Directories  
**Brand:** GitHub  
**Source:** <https://github.com/jepemo/meta-awesome/blob/master/doc/php-installation.md>

A curated "meta" Awesome-style page that aggregates guides, how-tos, and lists related to installing PHP. It acts as a central reference to other Awesome lists and documentation resources focused on PHP installation.

### Features
- Curated as part of the broader **Meta Awesome** collection.
- Focused specifically on **PHP installation** topics.
- Aggregates **Awesome-style guides and lists** rather than hosting the guides itself.
- Acts as a **meta resource**, pointing to:
  - PHP installation tutorials and walkthroughs.
  - Tool collections relevant to installing and managing PHP.
  - Documentation resources related to PHP setup.
- Hosted on **GitHub**, making it easy to track changes and contributions.
