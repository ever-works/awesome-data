## Overview

Awesome Eleventy provides resources for Eleventy (11ty), the simpler static site generator.

## What is Eleventy?

A simpler static site generator with flexible templating and zero client-side JavaScript by default.

## Key Features

- **Template Agnostic** - Use any templating language
- **Zero Config** - Works out of the box
- **Data Driven** - Flexible data cascade
- **Fast Builds** - Incrementalbuilds
- **No Client JS** - Optional JavaScript

## Supported Templates
