# Search Engine Optimization (SEO)

**Category:** Themed Directories  
**Tags:** SEO, marketing, web  
**Source:** https://github.com/marcobiedermann/search-engine-optimization#readme

A curated, checklist-style collection of Search Engine Optimization (SEO) tips, techniques, and tools to help improve website visibility in search engines.

---

## Features

### 1. Checklist-Based SEO Guidelines
- Organized as a practical checklist covering key on-page and technical SEO areas.
- Focuses on elements that directly impact search engine crawling, indexing, and ranking.

### 2. URL & Technical Structure
- Guidance on handling:
  - `404 FILE_NOT_FOUND` pages
  - Proper use of `divs` and `tables` in layout and content
  - `.htaccess` configuration
  - Pagination tags: `rel="next"` and `rel="prev"`
  - `robots.txt` setup
  - `<meta name="robots" content="">` configuration

### 3. Meta Information
- Recommendations for:
  - `<meta name="description" content="">`
  - `<title>Website Title</title>`
- Focus on writing effective titles and descriptions for search visibility and click-through.

### 4. Keywords
- Use of:
  - `<meta name="keywords" content="">`
- Historic/legacy keyword meta tag considerations (primarily informational for older systems).

### 5. Content Structure
- Heading hierarchy and emphasis:
  - Proper usage of `H1` through `H6`
  - Using `strong` for emphasizing important text
- Emphasis on semantic, well-structured HTML for better indexing and accessibility.

### 6. Images
- Recommendations around image markup:
  - Explicit `width=""` and `height=""` attributes
- Indirect impact on performance and layout stability, which affects SEO.

### 7. Videos
- Handling video content via:
  - `<video>` elements
- Ensuring multimedia content is accessible and indexable.

### 8. Links & Internationalization
- Link-related practices including:
  - `rel="nofollow"` for controlling link equity
  - Alternate language/region URLs:
    - `<link rel="alternate" href="example.com/fr/" hreflang="fr-fr" />`
- Helps with international SEO and controlling crawler behavior.

### 9. Mobile Optimization
- Mobile-friendly meta configuration:
  - `<meta name="viewport" content="width=device-width, initial-scale=1">`
- Supports responsive design and mobile-first indexing.

### 10. Sitemap & Social Media
- Sections dedicated to:
  - Sitemaps (XML/HTML) for better crawl coverage
  - Social media integration for improved discovery and rich snippets

### 11. Tools & Services Directory
A categorized list of third-party tools and services to support SEO workflows:

- **Webmasters** – Tools similar to search console platforms for monitoring site health and indexing.
- **Analytics** – Services for tracking traffic, user behavior, and performance metrics.
- **Optimization** – Tools for improving site speed, technical health, and on-page factors.
- **Keywords** – Keyword research and analysis tools.
- **Links** – Backlink analysis and link management tools.
- **Structured Data** – Validators and helpers for schema.org and rich snippet markup.
- **Bookmarklets** – Quick-access tools for in-browser SEO checks.
- **Browser Extensions** – SEO add-ons for Chrome/Firefox and other browsers.
- **Jekyll Plugins** – SEO-related plugins for Jekyll-based static sites.
- **TYPO3 Extensions** – SEO extensions for TYPO3 CMS.
- **WordPress Plugins** – SEO plugins for WordPress-based sites.

### 12. Learning Resources
- **Books** – Curated reading list on SEO concepts, tactics, and strategy.
- **Courses** – Educational resources and training material for learning SEO.
