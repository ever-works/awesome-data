# Awesome Generative AI

**URL:** https://github.com/steven2358/awesome-generative-ai#readme  
**Category:** Themed Directories  
**Tags:** ai, machine-learning, awesome-lists

## Description
Awesome Generative AI is a curated, community-maintained directory of modern generative AI projects and services. It focuses on tools, models, libraries, and learning resources for text, image, audio, video, coding, agents, and other modalities.

## Features
- **Curated Awesome List**
  - Organized collection of high-quality generative AI projects and services.
  - Focus on modern tools, models, and applications across multiple domains.

- **Multi‑Modal Coverage**
  - **Text**
    - Models
    - Chatbots
    - Custom interfaces
    - Search engines
    - Local search engines
    - Writing assistants
    - ChatGPT extensions
    - Productivity tools
    - Meeting assistants
    - Academic tools
    - Leaderboards
    - Other text generators
  - **Coding**
    - Coding assistants
    - Developer tools
    - Playgrounds
    - Local LLM deployment tools
  - **Agents**
    - Autonomous agents
    - Custom assistants
  - **Image**
    - Image generation models
    - Image generation services
    - Graphic design tools
    - Image libraries
    - Model libraries
    - Stable Diffusion resources
  - **Video**
    - Video-related generative AI tools and services (section present in the list’s structure).

- **Learning & Discovery Resources**
  - Recommended reading section for foundational and advanced materials.
  - Milestones section highlighting key developments in generative AI.
  - Separate **Discoveries List (DISCOVERIES.md)** with up‑and‑coming and experimental projects.

- **Open Contribution Model**
  - Public contribution guidelines (CONTRIBUTING.md) for adding or updating entries.
  - Contributions via GitHub pull requests or issues.
  - ARCHIVE.md used to manage or deprecate older/retired entries.

- **Licensing & Openness**
  - Repository includes a LICENSE file specifying usage rights for the list content.

## Use Cases
- Discovering generative AI tools for text, image, video, and audio.
- Finding coding assistants and developer tools for working with LLMs and generative models.
- Exploring autonomous agent frameworks and custom assistant tools.
- Researching state of the art and historical milestones in generative AI.
- Identifying up‑and‑coming projects through the Discoveries list.

## Pricing
- The GitHub list itself is **free to access and use** as an open directory.
- Individual tools/services linked from the list may have their own separate pricing, which is not defined in this repository.
