## Overview

Awesome Generative AI is a comprehensive collection of modern generative AI tools, models, and resources spanning text, images, audio, video, and code generation.

## Large Language Models (LLMs)

### Commercial Models
- GPT-4/GPT-5 (OpenAI) - Advanced language understanding and generation
- Claude (Anthropic) - AI assistant with extended context
- Gemini (Google) - Multimodal AI model
- Llama (Meta) - Open-source foundation models

### Text Generation Tools
- ChatGPT - Conversational AI interface
- Perplexity - AI-powered search and answers
- Character.AI - Conversational AI characters
- Claude - Long-form content generation

## Image Generation

### Diffusion Models
- Stable Diffusion - Open-source text-to-image generation
- DALL-E 3 - OpenAI's image generation model
- Midjourney - AI art generation platform
- Adobe Firefly - Creative AI tools

### Image Editing
- Runway ML - Video and image editing with AI
- PhotoRoom - AI background removal
- Remove.bg - Background removal tool

## Code Generation

- GitHub Copilot - AI pair programmer
- Cursor - AI code editor
- Tabnine - AI code completion
- Amazon CodeWhisperer - AWS code generation

## Audio & Music

- Eleven Labs - AI voice generation
- Mubert - AI music generation
- Suno - Music creation from text
- Riffusion - Music generation via diffusion

## Video Generation

- Runway Gen-2 - Text and image-to-video
- Pika Labs - Video generation platform
- HeyGen - AI avatar videos
- Synthesia - AI video creation

## Multimodal AI

- GPT-4V - Vision-enabled GPT-4
- Gemini - Native multimodal understanding
- Claude with vision - Image understanding

## Features

- Text generation and completion
- Image synthesis and editing
- Code generation and assistance
- Audio and music creation
- Video generation and editing
- Multimodal understanding

## Use Cases

- Content creation and marketing
- Software development
- Creative arts and design
- Education and tutoring
- Research and analysis
- Productivity and automation

## Pricing

Varies widely:
- Free tiers: ChatGPT, Claude, Stable Diffusion
- Subscription: ChatGPT Plus ($20/month), Midjourney ($10-60/month)
- Pay-per-use: OpenAI API, Anthropic API
- Open source: Stable Diffusion, Llama models
