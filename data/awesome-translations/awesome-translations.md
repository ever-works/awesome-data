# Awesome Translations

**Website:** https://github.com/mbiesiad/awesome-translations#readme  
**Category:** Themed directories  
**Tags:** localization, developer-tools, awesome-lists

## Overview
Awesome Translations is a curated, open-source list of tools, libraries, platforms, and educational resources focused on translation and localization workflows, including translation (t9n), localization (l10n), globalization (g11n), multilingualization (m17n), and internationalization (i18n).

## Features
- **Curated translation ecosystem directory**  
  - Organized collection of tools, books, articles, blogs, courses, and other resources related to translations and localization.

- **Coverage of multiple domains in language technology**  
  - Translation (t9n): resources specifically for text translation.  
  - Localization (l10n): tools and guides for adapting products to different languages and regions.  
  - Globalization (g11n): resources for global-ready product design and implementation.  
  - Multilingualization (m17n): handling content and products in multiple languages.  
  - Internationalization (i18n): preparing software and content for easy localization.

- **Structured content sections**  
  - **Translations** – general resources focused on translation practices and workflows.  
  - **Platforms**  
    - Localization and translation platforms.  
    - Project management tools related to translation/localization.  
    - Platform-specific tools for particular ecosystems or technologies.
  - **Apps**  
    - Automated translation applications.  
    - Translation software for translators and teams.
  - **Tools**  
    - Automatic translation tools for end users.  
    - Libraries and tools for developers integrating translation or localization.  
    - Machine-translation tools/APIs for developers.  
    - Other supporting tools in the translation/i18n workflow.
  - **Articles**  
    - Introductions to internationalization concepts and best practices.  
    - Developer-focused articles, including topics like `hreflang` usage.  
    - Content on understanding computer-assisted translation (CAT).  
    - Articles on machine translation and its applications.
  - **Blogs** – curated blogs discussing translation, localization, and internationalization.  
  - **Online Courses** – learning materials and structured courses on translation/i18n topics.  
  - **Books** – recommended books covering translation, localization, and related workflows.  
  - **Other** – additional relevant resources that don’t fit into the main sections.

- **Open-source and community-driven**  
  - Public GitHub repository with versioned updates and history.  
  - Clear **Contributing** guidelines for adding or improving resources.  
  - **Code of Conduct** for community participation.  
  - **License** file specifying reuse and contribution terms.

## Intended Audience
- Developers implementing i18n/l10n in applications or services.  
- Localization engineers and translation workflow designers.  
- Translators and language professionals exploring tools and platforms.  
- Learners seeking articles, books, blogs, and courses on translation and localization.
