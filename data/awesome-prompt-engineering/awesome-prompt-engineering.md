## Overview

Awesome Prompt Engineering is a comprehensive repository containing hand-curated resources for mastering the art and science of prompt engineering with large language models. The list focuses on Generative Pre-trained Transformers (GPT), ChatGPT, Claude, and other modern LLMs.

## Key Resources

- **Fundamentals**: Guides covering prompt design principles, meta-prompting, and context engineering
- **Advanced Techniques**: Chain-of-thought prompting, few-shot learning, retrieval-augmented generation (RAG)
- **Tools and Platforms**: Prompt optimization tools, testing frameworks, and management systems
- **Research Papers**: Academic papers on prompt optimization, evaluation, and tuning methods
- **Community Resources**: Learn Prompting community with 40,000+ members, courses, and hackathons
- **Security**: Coverage of prompt injection, jailbreaking, and defensive techniques
- **Agent Frameworks**: DSPy, LangGraph, and other frameworks for building AI agents

## Featured Topics

- Google Prompting Essentials covering 5-step prompt design
- GPT-5 and latest model capabilities
- System prompts and instruction tuning
- Multi-modal prompting for vision and audio
- Automatic prompt optimization
- HackAPrompt competitions and challenges

## Practical Applications

The repository includes real-world examples for code generation, content creation, data analysis, customer support automation, and more.

Available in 13 languages, making it accessible to a global audience of practitioners and researchers.
