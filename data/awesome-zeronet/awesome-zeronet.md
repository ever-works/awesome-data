# Awesome ZeroNet

A curated, community-maintained directory of implementations, plugins, tools, and “zites” (ZeroNet sites) for the ZeroNet decentralized, peer‑to‑peer web-like network.

- **Website/Source**: https://github.com/zolagonano/awesome-zeronet#readme
- **Platform**: ZeroNet ecosystem
- **Category**: Themed directories / Awesome lists

---

## Overview

Awesome ZeroNet is a structured index of useful resources in the ZeroNet ecosystem. It focuses on listing and categorizing:

- ZeroNet sites (“zites”) by purpose and type
- Client and protocol implementations
- Plugins that extend ZeroNet functionality
- Guides, documentation, and development resources
- Public proxy instances and community hubs

It is maintained as a GitHub Awesome list and linked to the ZeroNet project.

### 4. Guides and Documentation

Organized documentation resources for learning and using ZeroNet:

- **Installation guides**  
  Resources for setting up and running ZeroNet.

- **Site development guides**  
  Tutorials and references on how to build and maintain zites.

These help both new and experienced users deploy and develop on the network.

### 5. Proxies (Instances)

Section listing public ZeroNet proxy instances, allowing access to some ZeroNet content from a regular web browser (without running a client). Each entry includes the proxy URL/endpoint and a short description in the full list.

Directory of community spaces related to ZeroNet, such as:

- Other social/community zites

This helps users discover active places to discuss ZeroNet and its ecosystem.
