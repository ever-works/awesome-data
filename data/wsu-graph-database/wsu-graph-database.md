# WSU Graph Database

**Category:** Themed Directories  
**Tags:** datasets, graph, research  
**Website:** http://www.eecs.wsu.edu/mgd/gdb.html

## Description
The WSU Graph Database is a collection of graph datasets maintained by Washington State University. It provides a centralized directory of network and graph instances commonly used in research areas such as data mining, pattern recognition, and graph algorithms. The database is intended as a reusable resource for researchers who need standardized graph data for experimentation, benchmarking, and comparative studies.

## Features
- **Collection of graph datasets**: Hosts multiple graph instances for use in research and experimentation.
- **Focus on research applications**: Designed for studies in data mining, pattern recognition, and graph algorithm development and evaluation.
- **Centralized directory**: Acts as a single access point to a variety of reusable graph data resources.
- **Academic maintenance**: Curated and maintained by Washington State University, providing an academic context for the datasets.
