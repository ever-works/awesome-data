## Overview

Awesome TUIs showcases terminal user interface applications that provide rich, interactive experiences in the command line.

## Productivity Tools

### Time Management

**pomo**
- Minimal Pomodoro timer for the terminal with productivity statistics
- Session tracking and history
- Customizable work/break intervals
- Sound notifications
- Simple keyboard controls
- Progress visualization

### Note Taking

**Toney**
- Terminal-based note-taking app for developers
- Markdown support
- Code syntax highlighting
- Tag-based organization
- Search functionality
- Vim-style keybindings
