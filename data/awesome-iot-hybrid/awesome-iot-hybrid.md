# Awesome IoT & Hybrid

**URL:** https://github.com/weblancaster/awesome-IoT-hybrid#readme  
**Category:** Themed directories  
**Tags:** iot, web, framework

## Overview
Awesome IoT & Hybrid is a curated, GitHub-hosted directory of tools, frameworks, operating systems, and learning resources for building Internet of Things (IoT) and hybrid applications, with an emphasis on web technologies.

## Features
- **Curated "awesome list" format**
  - Follows the popular "Awesome" list convention (inspired by sindresorhus/awesome).
  - Organizes links and resources into clear sections for easier discovery.

- **IoT-focused content**
  - **OS (Operating Systems)**: Section dedicated to IoT-related operating systems and platforms.
  - **Frameworks & Tools**: Collection of libraries, frameworks, and development tools for IoT solutions.
  - **Resources / Websites / Projects**: Links to tutorials, reference projects, documentation sites, and other educational material relevant to IoT.

- **Industrial IoT (IIoT)**
  - Dedicated section for Industrial IoT resources, tools, and platforms.

- **Hybrid application resources**
  - **Hybrid Desktop**: Items related to building desktop applications using web or hybrid technologies.
  - **Hybrid Mobile**: Resources for creating mobile apps with hybrid/web-based approaches.
  - **Tools & Plugins**: Supporting plugins, utilities, and tooling for hybrid app development.

- **Miscellaneous section**
  - Catch-all area for related items that do not fit into the main categories but are relevant to IoT or hybrid development.

- **Open source and collaborative**
  - Hosted on GitHub with visible history and commits.
  - Contributions are encouraged via standard Git workflow:
    - `git checkout -b my-new-branch`
    - `git commit -am 'fix stuff'`
    - `git push origin my-new-branch`
