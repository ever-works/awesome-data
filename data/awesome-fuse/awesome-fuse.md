# Awesome Fuse

**Website:** https://github.com/fuse-compound/awesome-fuse#readme  
**Category:** Themed Directories  
**Tags:** awesome-lists, mobile, developer-tools

## Overview
Awesome Fuse is a community-maintained, curated directory of resources for building mobile applications with the Fuse platform. It organizes links to Fuse-related applications, articles, community hubs, development resources, and plugins in one place.

## Features
- **Curated resource list**
  - Focused specifically on the Fuse mobile app development platform.
  - Includes hand-picked links rather than auto-generated listings.

- **Coverage of key resource types**
  - **Applications** built with or demonstrating Fuse.
  - **Articles** about Fuse, including tutorials, guides, and write-ups.
  - **Community** links such as forums, groups, and communication channels.
  - **Development** resources, tools, and libraries helpful for building Fuse apps.
  - **Open Source** Fuse projects and example code.
  - **Plugins** that extend Fuse functionality.

- **Open contribution model**
  - Public contribution guidelines available via `CONTRIBUTING.md`.
  - Suggestions and additions accepted through GitHub.

- **License and reuse**
  - Content released under CC0 (public domain dedication).
  - Rights to the list are waived to the extent possible under law, allowing free reuse.

- **Maintenance status transparency**
  - Repository explicitly notes that the list is currently unmaintained.
  - Open call for a new maintainer via a linked GitHub issue.

## Pricing
- **Free**
  - Open GitHub repository.
  - No paid plans or commercial pricing indicated.

## Notes
- The list is currently unmaintained; resource freshness may vary.
- Best suited for developers working with or exploring the Fuse mobile app development platform who want a centralized index of tools, examples, and learning materials.
