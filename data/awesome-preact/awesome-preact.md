# Awesome Preact

**Category:** Themed Directories  
**Tags:** awesome-lists, javascript, web  
**Source:** https://github.com/preactjs/awesome-preact#readme

## Overview
Awesome Preact is a curated “awesome list” of libraries, components, and learning resources for **Preact**, a fast ~3kB alternative to React that uses a similar modern API. The list aggregates tools and references that help developers build web applications with Preact.

## Features
- Curated list format focused specifically on the **Preact** ecosystem.
- Includes **libraries** that extend or complement core Preact functionality.
- Covers **UI components** and component libraries built for or compatible with Preact.
- Links to **learning resources** such as tutorials, guides, and reference material about Preact.
- Highlights Preact as a **lightweight (~3kB) React alternative** using a **modern React-like API**.
- Organized as a GitHub README, making it easy to browse, search, and contribute via pull requests.

## Pricing
- Access to the Awesome Preact list is **free** (public GitHub repository).
