# Awesome Laravel Education

**Category:** Themed Directories  
**Slug:** awesome-laravel-education  
**Tags:** laravel, education, web-development

## Overview
Awesome Laravel Education is a curated, community-maintained collection of learning resources for the Laravel PHP framework. It aggregates tutorials, screencasts, guides, articles, and tools suitable for beginner, intermediate, and advanced Laravel developers.

## Features

### Multilingual Resource Lists
- Curated lists available in multiple languages:
  - Chinese
  - English
  - Portuguese

### Structured Learning Index (English Version)
Organized sections to help you navigate Laravel learning materials:

- **New to Laravel**
  - Introductory screencast series:
    - Laracasts: Laravel From Scratch
    - Laracasts: Laravel 5 Fundamentals
  - Environment setup resources:
    - Laravel Homestead installation guides for Windows, Linux, and macOS
    - Laracasts: Say Hello to Laravel Homestead 2.0
    - Setup Laravel executable environment on AWS EC2 quickly (automation script/repo)
  - Official tutorial-based learning:
    - Laravel Documentation: Basic Task List (quickstart)
    - Laravel Documentation: Intermediate Task List (quickstart)

- **Quick References**
  - Centralized links to Laravel reference material (cheatsheets, docs, and similar quick-look resources).

- **Learn From Video / Posts**
  Categorized learning materials (videos, blog posts, and articles) for core Laravel topics:
  - Composer / Homestead / Docker
  - Routing / Controllers / Requests / Responses
  - Database
  - Models / Eloquent
  - Views / Blade / Language
  - Forms / Validation
  - Architecture Foundations / IoC
  - Auth / Security
  - Middleware
  - Front End
  - Session
  - Filesystem / Cloud Storage
  - Cache
  - Mail
  - Testing and Coding Standard
  - Case Study

- **Tutorials**
  - Curated step-by-step Laravel tutorials beyond the official quickstarts.

- **Blogs**
  - List of Laravel-related blogs for ongoing learning and best practices.

- **Newsletters**
  - Laravel and PHP ecosystem newsletters for staying updated on new tools and techniques.

- **Open Source Laravel**
  - Notable open source Laravel projects for code reading, learning, and reuse.

- **Coding Standard**
  - Resources on coding style, standards, and conventions (Laravel and broader PHP).

- **Dependency Management**
  - Guidance and tools around Composer and package management in Laravel/PHP.

- **Develop Tools**
  - Development tools relevant to Laravel projects (IDEs, debuggers, helpers, etc.).

- **PHP Book List for Developer**
  - Recommended PHP books for improving language and ecosystem understanding.

- **Interview**
  - Interview-focused resources for Laravel/PHP roles (questions, prep material, etc.).

- **Misc**
  - Additional Laravel- or PHP-related resources that don't fit other categories.

### Community & Contributions
- Open contribution model via GitHub pull requests.
- Guidelines:
  - One pull request per suggestion.
  - New categories or improvements to existing categorization are encouraged.
- Acknowledgment of community contributors (content, translations, improvements).

## Pricing
- Not applicable. Awesome Laravel Education is an open, GitHub-hosted curated list of resources and is free to access and contribute to.
