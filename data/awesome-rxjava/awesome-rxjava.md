# Awesome RxJava

**Website:** https://github.com/eleventigers/awesome-rxjava#readme  
**Category:** Themed Directories  
**Tags:** awesome-lists, java, developer-tools

## Overview
Awesome RxJava is a curated “awesome list” of useful resources for working with [RxJava](https://github.com/ReactiveX/RxJava). It aggregates links to libraries, tools, guides, and community channels in the RxJava ecosystem, following the general Awesome List conventions.

## Features

- **Curated RxJava Ecosystem List**  
  - Focused specifically on the RxJava library and related tooling.  
  - Inspired by the broader [Awesome](https://github.com/sindresorhus/awesome) lists format.

- **Bindings**  
  - Section for RxJava bindings to other components and frameworks.  
  - Explicitly mentions bindings around `SharedPreferences`.

- **Utilities**  
  - Dedicated section for utility libraries and helper components built on top of RxJava.

- **Testing**  
  - Section for testing-related tools, patterns, and libraries for RxJava code.

- **Guides**  
  - Collection of guides and learning resources specific to RxJava usage and patterns.

- **Articles**  
  - Articles and posts explaining RxJava concepts, including:  
    - `repeatWhen` operator  
    - `retryWhen` operator  
    - `Subject` and related patterns

- **Tools**  
  - Section for development and debugging tools that support or integrate with RxJava.

- **Community**  
  - Pointers to community and project channels, including:  
    - Twitter handle: `@RxJava`  
    - GitHub project: `ReactiveX/RxJava`

- **Open Contribution Model**  
  - `CONTRIBUTING.md` present for guidelines on how to propose additions and changes to the list.
