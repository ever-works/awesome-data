## Overview

Awesome shadcn-vue provides resources for shadcn-vue, the Vue.js port of the popular shadcn/ui component library.

## What is shadcn-vue?

A Vue port of shadcn/ui, providing beautifully designed, accessible, and customizable components built with Radix Vue and Tailwind CSS.

## Component Collections

- Form components
- Data display components
- Navigation components
- Feedback components
- Overlay components
- Layout utilities

## Templates & Starters

- Admin dashboards
- SaaS templates
- Landing pages
- E-commerce starters
- Documentation sites

## Integration

- Nuxt.js integration
- Vue 3 Composition API
- TypeScript support
- Tailwind CSS configuration
- Dark mode support

## Key Features

- Copy-paste components
- Full accessibility (ARIA)
- Customizable with Tailwind
- Keyboard navigation
- Screen reader support

## Use Cases

- Vue 3 applications
- Nuxt.js projects
- Design systems
- Rapid prototyping
