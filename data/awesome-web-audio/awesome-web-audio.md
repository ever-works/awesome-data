# Awesome Web Audio

A curated directory of Web Audio API resources, including packages, demos, and learning materials for web‑based audio development.

**Website:** https://github.com/notthetup/awesome-webaudio#readme

---

## Overview

Awesome Web Audio is an open, community-maintained list of tools and resources for working with the Web Audio API. It aggregates frameworks, libraries, utilities, example apps, and educational materials relevant to browser-based audio programming.

The list is published as a GitHub repository and follows the general "awesome list" format.

---

## Features

### Scope & Structure
- Curated list focused specifically on Web Audio API and web-based audio development.
- Organized into clear sections for easier navigation:
  - **Packages**
    - Frameworks
    - Libraries
    - Utilities
    - MIDI
    - Apps
  - **Resources**
    - Tutorials
    - Books
    - Newsletters
    - Community
  - **Obsolete**
    - Projects without activity since January 2019 or officially discontinued.

### Packages
- **Frameworks** – Higher-level structures and toolkits built on top of the Web Audio API to simplify audio application development.
- **Libraries** – Reusable components, modules, and APIs for audio processing, synthesis, visualization, and related tasks.
- **Utilities** – Helper tools and small utilities for workflow, debugging, analysis, or integration of Web Audio features.
- **MIDI** – Tools and libraries for handling MIDI input/output in the browser and integrating MIDI workflows with Web Audio.
- **Apps** – Example and production web applications that showcase or rely heavily on the Web Audio API.

### Learning & Community Resources
- **Tutorials** – Guides, walkthroughs, and how‑to articles for learning the Web Audio API and related techniques.
- **Books** – Longer-form learning materials and references about web audio and audio programming in the browser.
- **Newsletters** – Periodic updates and curated news related to web audio, libraries, and ecosystem developments.
- **Community** – Links to community spaces (such as forums, groups, or chats) focused on Web Audio and audio development on the web.

### Maintenance & Quality
- **Curated and community-driven** – Contributions are accepted via GitHub pull requests.
- **Obsolete section** – Explicit separation of unmaintained or inactive projects (no activity since January 2019 or marked as dead), helping users find currently active tools.

### Governance & Contribution
- **Code of Conduct** – A documented code of conduct (`code-of-conduct.md`) to guide community behavior.
- **Contributing Guidelines** – Instructions in `contributing.md` describing how to propose additions or changes to the list.

### Licensing
- **CC0 Public Domain Dedication** – The list content is released under CC0, waiving copyright and related rights to the greatest extent possible under law.
