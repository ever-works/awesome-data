# Awesome Textpattern

A community-curated, "awesome-style" directory of plugins, themes, and resources for the lightweight PHP-based Textpattern CMS.

**Website:** https://github.com/drmonkeyninja/awesome-textpattern#readme  
**Platform:** Textpattern CMS (PHP)  
**Category:** Themed directories / Awesome lists

---

## Description
Awesome Textpattern is a structured collection of plugins, themes, and learning resources for Textpattern CMS, focused on the latest Textpattern versions (v4.7.x, with a separate branch for v4.6.x). It helps Textpattern users discover tools to extend and enhance their sites, and provides references for development and best practices.

---

## Features

### Scope & Versioning
- Curated list specifically for **Textpattern CMS**.
- Primary focus on **Textpattern v4.7.x**.
- Separate branch for **Textpattern v4.6.x** resources.

### Plugin Directory
Plugins are categorized to help locate functionality quickly:
- **Admin** – Plugins that extend or improve the Textpattern admin interface.
- **Comments** – Tools for managing and enhancing comment functionality.
- **Content** – Plugins for content management, formatting, and display.
- **Embedding** – Solutions for embedding external content or services.
- **Forms** – Enhancements and helpers for Textpattern forms.
- **Images** – Image handling, galleries, and media-related plugins.
- **Navigation** – Menus, sitemaps, and navigation-related tools.
- **Performance** – Caching, optimization, and performance-improving plugins.
- **SEO** – Search engine optimization utilities.
- **Social** – Plugins that add social sharing or social network integrations.
- **Miscellaneous** – Additional plugins that don’t fit other categories.

### Themes
Curated themes for different parts of Textpattern:
- **Back-end themes** – Admin UI / control panel themes.
- **Front-end themes** – Site/front-facing templates and layouts.

### Plugin Development
- Dedicated section for **plugin development** resources and references, supporting developers who want to build or extend Textpattern plugins.

### General Resources
Collection of materials to improve Textpattern skills and knowledge:
- **Official resources** – Links to official Textpattern documentation and related materials.
- **Migration** – Resources on migrating to or between Textpattern versions.
- **Web development software integration** – Guides or tools for integrating Textpattern with development tools and workflows.

### Community
Directory of community-related links:
- **Core Team** – Information and references related to the Textpattern core team.
- **Plugin developers** – List of plugin authors, ordered alphabetically by author prefix.
- **Blogs** – Blogs and sites publishing Textpattern-related content.

### Contribution Workflow
- Open, community-driven contribution model.
- Contribution guidelines provided in `CONTRIBUTING.textile`.
