## Overview

Awesome Cloudflare Workers provides resources for Cloudflare Workers, serverless functions running on the edge.

## What are Cloudflare Workers?

Serverless execution environment that allows you to create entirely new applications or augment existing ones without configuring or maintaining infrastructure.

## Key Features

- **Edge Computing** - Run in 200+ cities worldwide
- **V8 Isolates** - Fast cold starts
- **Standards-based** - Web APIs (fetch, streams, etc.)
- **KV Storage** - Eventually consistent key-value storage
- **Durable Objects** - Strongly consistent coordination
- **R2 Storage** - S3-compatible object storage

## Frameworks

- **Hono** - Ultrafast web framework
- **Remix** - Full-stack web framework
- **Qwik City** - Resumable framework
- **SvelteKit** - Svelte meta-framework
- **Astro** - Content-focused framework
