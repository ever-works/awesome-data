# Typedb Awesome

**Description**

Typedb Awesome is a curated, Awesome-style directory of frameworks, libraries, software, and learning resources for TypeDB, a polymorphic database with a conceptual data model and a type-theoretic query language (TypeQL).

**Category:** Themed directories  
**Tags:** database, graph, developer-resources

---

## Features

- **Curated TypeDB ecosystem list**  
  - Collects and organizes community and official projects around TypeDB in one place.

- **Open source projects using TypeDB**  
  Examples of tools and applications built on TypeDB, including:
  - TypeDB Studio – IDE for TypeDB
  - Blitz-ORM
  - TypeDB-ML
  - TypeDB Loader
  - TypeDB OSI – Cyber Threat Intelligence
  - TypeDB OSI – Biomedical Knowledge Graph
  - Source++
  - Rasa
  - TypeBridge

- **Documentation links**  
  - Section dedicated to official and community documentation resources for learning and using TypeDB.

- **Example projects**  
  - TypeDB Driver Examples and other sample projects demonstrating practical usage patterns.

- **Learning materials**  
  - **Blog posts**: Curated articles about TypeDB concepts, use cases, and best practices.
  - **Videos**: Talks, tutorials, and other video content related to TypeDB.

- **Client drivers**  
  - **Official drivers**: Rust, Python, Java, Node.js, and C.
  - **Community driver**: Julia.

- **IDE and editor integrations**  
  - Emacs – TypeQL Plugin
  - Visual Studio Code – TypeQL Plugin
  - JetBrains / IntelliJ – TypeQL Plugin

- **Conferences**  
  - Section for conference talks and sessions focused on TypeDB.

- **Community & contribution**  
  - Links to the TypeDB community (e.g., Discord) for discussion and collaboration.
  - Contribution guidelines (via `contributing.md`) for adding or updating resources.
  - Code of conduct for community participation.

- **Repository organization**  
  - Maintained as a GitHub repository for transparency and community-driven updates.
  - Includes licensing information (via `LICENSE`).
