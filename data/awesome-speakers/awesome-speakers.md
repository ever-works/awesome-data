# Awesome Speakers

A curated, community-driven directory of conference and meetup speakers from the programming and design worlds, with their topics and contact details to help organizers discover and invite diverse speakers.

- **Website/Source**: https://github.com/karlhorky/awesome-speakers#readme
- **Category**: Themed directories
- **Tags**: awesome-lists, community, events

## Overview
Awesome Speakers is an open list of speakers in software development and design, organized primarily by geography. Each entry typically includes the speaker’s name, location, topics they speak about, and a way to contact or learn more about them. The project aims to make it easier for event organizers to find relevant, high-quality, and diverse speakers for meetups and conferences.

## Features

- **Curated speaker list**
  - Focused on programming and design communities
  - Includes speakers suitable for meetups, conferences, and other tech events

- **Geographic organization**
  - Speakers grouped by region/country/city (e.g., Africa → Nigeria → Lagos)
  - Helps organizers find speakers near a specific location or region

- **Speaker profiles**
  - Name and avatar/image (stored under an `avatars` directory)
  - Topics/areas of expertise (e.g., JavaScript, Node.js, TypeScript, Testing, Community)
  - Contact or social links (e.g., Twitter profiles) when provided

- **Topic-based discovery**
  - Each speaker lists key topics they speak on, allowing organizers to filter informally by technology, discipline, or theme

- **Open contribution model**
  - `CONTRIBUTING.md` guides community members on how to add or update speaker entries
  - Pull request template (`PULL_REQUEST_TEMPLATE`) standardizes contributions
  - Encourages broad community participation to expand and maintain the list

- **Open-source repository structure**
  - Managed as a GitHub repository (`karlhorky/awesome-speakers`)
  - Uses configuration and tooling files (`.eslintrc.cjs`, `prettier.config.cjs`, `tsconfig.json`, etc.) to keep data and code consistent and maintainable
  - `scripts` directory for automation and maintenance tasks related to the list

- **Part of the Awesome ecosystem**
  - Conforms to the general conventions of “Awesome” lists (https://awesome.re)
  - Badged as an Awesome list, making it discoverable among other curated resources

## Use Cases

- Event organizers sourcing speakers for meetups, conferences, and workshops
- Community groups seeking diverse voices in programming and design
- Speakers wishing to be discoverable by organizers through an open, community-maintained directory
