# Awesome Construct

**Website:** https://github.com/ConstructCommunity/awesome-construct#readme  
**Category:** Themed Directories  
**Tags:** games, engine  
**Platform / Brand:** Construct (Construct 2 & Construct 3)

## Overview
Awesome Construct is a curated, community-maintained directory of resources for the Construct 2 and Construct 3 game engines. It aggregates tools, tutorials, examples, and other materials to help developers build games with Construct.

## Features
- **Curated resource list** for Construct 2 and Construct 3 game engines.
- **Focus on multiple resource types**, including:
  - Tools and utilities for Construct game development.
  - Tutorials and learning materials.
  - Example projects and sample files.
  - Additional resources relevant to Construct users.
- **Covers both Construct 2 and Construct 3**, with separate comparison/content file (`c2vsc3.md`).
- **Community guidelines and contribution workflow** via `CODE_OF_CONDUCT.md` and `CONTRIBUTING.md`.
- **Versioned and structured as a GitHub repository**, making it easy to browse, fork, and contribute.
- **Licensed content** (see `LICENSE` in the repository) suitable for open collaboration.

## Pricing
- Access to the Awesome Construct list and repository is **free** (open GitHub repository; no pricing or paid plans stated).

## Resource Type
- **Type:** Curated directory / awesome list
- **Intended users:** Construct 2 & Construct 3 developers looking for plugins, behaviors, templates, tools, and learning resources.
