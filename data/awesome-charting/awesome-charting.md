## Description
Awesome Charting is a curated, alphabetically ordered directory of charting and data‑visualization resources for the web. It focuses on relevant, currently active JavaScript charting libraries and related tools that developers can use for different visualization use cases.

## Features
- Curated list of chart and data‑visualization resources
- Emphasis on JavaScript charting libraries for the web
- Focus on libraries and tools that are currently active and maintained
- Organized by categories to cover different visualization use cases
- Items within each category are ordered alphabetically
- Inspired by and follows the style of the broader "Awesome" list ecosystem

## Type
- Open-source curated directory / awesome list
