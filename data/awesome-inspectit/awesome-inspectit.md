# Awesome inspectIT

**Source:** https://github.com/inspectit-labs/awesome-inspectit#readme

## Overview
Awesome inspectIT is a curated, community-driven list of documentation and resources related to **inspectIT**, an open source Application Performance Management (APM) tool for analyzing Java (EE) applications.

## Features
- Focused on **inspectIT** and its ecosystem
- Curated links to:
  - **Official Resources** for inspectIT
  - **Tutorials** on using and configuring inspectIT
  - **Videos** demonstrating features, workflows, or presentations
  - **Blog Posts** discussing inspectIT usage, best practices, and case studies
  - **Profiles and Sensor Configurations** for instrumenting Java (EE) applications with inspectIT
  - **Uncategorized** items covering additional related materials
- Community contribution guidelines via `CONTRIBUTING.md`
- Licensed under **CC0 1.0** (public domain dedication) for the list contents
