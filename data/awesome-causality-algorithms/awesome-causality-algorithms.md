## Overview

Awesome Causality Algorithms indexes computational methods for discovering and estimating causal relationships from data.

## Causal Discovery

### Constraint-Based Methods
- PC Algorithm
- FCI (Fast Causal Inference)
- CCD (Cyclic Causal Discovery)

### Score-Based Methods
- GES (Greedy Equivalence Search)
- FGES (Fast GES)
- GIES (Greedy Interventional Equivalence Search)

### Functional Causal Models
- LiNGAM (Linear Non-Gaussian Acyclic Model)
- ANM (Additive Noise Models)
- PNL (Post-Nonlinear models)

## Causal Effect Estimation

### Treatment Effect Estimation
- Propensity score matching
- Inverse probability weighting
- Doubly robust estimation
- Causal forests

### Instrumental Variables
- Two-stage least squares
- GMM estimation

## Software Implementations

- **CausalNex**: Python library for causal reasoning
- **DoWhy**: Microsoft's library for causal inference
- **EconML**: Econometric methods
- **CausalML**: Uber's package

## Applications

- Policy evaluation
- Marketing attribution
- Medical treatment effects
- Root cause analysis
