## Overview

Awesome Cloud Native is an extensive collection inspired by the CNCF landscape, featuring the best cloud-native tools and applications.

## Features

- **CNCF Projects**: Graduated, incubating, and sandbox projects
- **Serverless Platforms**: Knative, OpenFaaS, Fission, Kubeless
- **Cloud Providers**: AWS, GCP, Azure cloud-native services
- **Container Tools**: Docker alternatives and complementary tools
- **Orchestration**: Kubernetes distributions and management tools
- **Service Mesh**: Istio, Linkerd, Consul mesh solutions
- **Observability**: Monitoring, logging, and tracing stacks
- **CI/CD**: Cloud-native continuous delivery platforms
