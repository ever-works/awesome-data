# Meta Awesome – Editors

**Website:** https://github.com/jepemo/meta-awesome/blob/master/doc/editors.md  
**Category:** Meta Directories  
**Tags:** meta-directory, editor, awesome-lists

## Overview
Meta Awesome – Editors is a curated meta-list focused on text editors. Instead of listing editor resources directly, it links to multiple "awesome" collections about text editors, covering plugins, configurations, and practical usage tips across different editors and ecosystems.

## Features
- **Meta collection of editor lists**  
  Aggregates multiple "awesome"-style lists related to text editors into one place.

- **Focus on text editors**  
  Targets tools used for code and text editing (e.g., general-purpose text editors and IDE-like editors).

- **Links to specialized awesome lists**  
  Includes references to editor-specific awesome collections (such as those focused on extensions, themes, language integrations, and workflows).

- **Configuration resources**  
  Points to lists and resources that cover configuration setups, dotfiles, and customization patterns for different editors.

- **Usage tips and best practices**  
  Curates lists that offer guides, tips, and tricks for effective editor usage.

- **Community-maintained GitHub document**  
  Hosted as a markdown document in a GitHub repository, allowing contributions through pull requests and issues.

- **Part of a broader meta-awesome project**  
  Belongs to the `meta-awesome` collection, which organizes meta-lists for various domains beyond editors.
