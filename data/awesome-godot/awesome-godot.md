# Awesome Godot

**Category:** Themed Directories  
**Slug:** `awesome-godot`  
**Website:** https://github.com/godotengine/awesome-godot#readme

## Overview
Awesome Godot is a curated, community-maintained directory of free and open source games, plugins, add-ons, scripts, and learning resources for the Godot game engine. It organizes Godot-related resources into structured sections to help developers and learners quickly find examples, tools, and reference material.

## Features

### Structured Resource Categories
- **Games** – Open source games built with Godot, organized by:
  - **2D**
    - By engine version: Godot 4, Godot 3, Godot 2
    - Example Godot 4 entries:
      - *99Managers Futsal Edition* – Free/libre futsal team-management game.
      - *A Dark Forest* – Minimal incremental game inspired by “A Dark Room”.
      - *Librerama* – Fast-paced arcade collection of mini-games.
      - *Poder Solar* – Simple resource management game.
      - *Unknown Horizons (Godot port)* – Work-in-progress reimplementation of Unknown Horizons.
    - Example Godot 3 entries:
      - *Blipshift* – Top-down moving platforms game.
      - *Breakable* – Breakout-like game.
      - *Circle Jump* – Mobile arcade game with an accompanying “how it’s made” tutorial.
      - *Everplast* – 2D platformer focused on speed and challenge.
      - *Falling Square* – Simple reflex-based mobile game.
      - *Flappy Race* – 2D online multiplayer game inspired by Flappy Bird and Mario Kart (desktop & HTML5).
      - *Haldric* – Work-in-progress reimplementation of Battle for Wesnoth.
      - *Memory Game* – Simple memory-matching game.
      - *Pocket Broomball* – Broomball sports game for iOS, Android, and HTML5.
      - *Portal 2D* – 2D remake of the Portal series.
      - *ROTA* – Gravity-bending puzzle platformer.
      - *spindle of serendipity* – Customizable word-guessing game.
      - *Starship Olympics* – Local party game with starship arena battles.
      - *TuxBuilder* – Work-in-progress reimplementation of SuperTux.
    - Example Godot 2 entries (partial from provided content):
      - *BlockPop* – Breakout clone.
      - *Bombs* – Simple mobile game.
  - **3D** – (Listed in the full directory; not fully included in the provided excerpt.)
  - **XR** – Extended reality projects (VR/AR) using Godot.

- **Projects** – Open source Godot-based projects and codebases that can serve as references or starting points.

- **Templates** – Ready-made Godot project templates and starter kits to accelerate new game or app creation.

- **Demos** – Demonstration projects showing specific engine features, patterns, or techniques.

- **Tutorials** – Learning resources for Godot, including guides, tutorials, and educational material.

- **Plugins and Scripts** – Extensible tools and code snippets to enhance Godot’s capabilities (editor add-ons, gameplay scripts, utilities, etc.).

- **Modules** – Engine modules and low-level extensions for Godot.

- **GDScript/C# Editor Support** – Resources that improve language support in external editors or IDEs for GDScript and C#.

- **Themes**
  - **Engine Themes** – Visual themes for the Godot editor.
  - **Syntax Themes** – Color schemes for code highlighting when working with Godot-related languages.

- **Unofficial Godot Builds** – Community-maintained builds of the Godot engine with custom patches or features.

- **Bash Scripts** – Command-line scripts to automate common Godot workflows and project tasks.

- **Websites** – External sites related to Godot (documentation, community hubs, learning portals, etc.).

- **Other** – Miscellaneous Godot-related tools and resources that don’t fit in the above categories.
