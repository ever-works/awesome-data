# Awesome Game Boy Development

**URL:** https://github.com/gbdev/awesome-gbdev#readme  
**Category:** Themed Directories  
**Tags:** games, development, hardware

## Overview
Awesome Game Boy Development is a curated, open-source directory of resources for developing on the Nintendo Game Boy and Game Boy Color. It aggregates tools, documentation, emulators, related projects, and open-source ROMs, and is maintained as a GitHub repository with a companion web version.

## Features
- **Curated resource list**
  - Focused specifically on Game Boy and Game Boy Color development.
  - Includes links to tools, documentation, emulators, related projects, and open-source ROMs.
  - Presented as an "awesome"-style list, following the common GitHub awesome-list format.

- **Multiple resource formats**
  - **Tools** for building, testing, and debugging Game Boy/Game Boy Color software.
  - **Documentation** for hardware and software aspects of the platforms.
  - **Emulators** catalogued separately (via `EMULATORS.md`).
  - **Related projects** that complement or extend Game Boy development.
  - **Open-source ROMs** to study and learn from existing code.

- **Structured repository contents**
  - `README.md` as the main entry point to the list.
  - `EMULATORS.md` for dedicated emulator listings.
  - `MORE.md` for additional or extended resources beyond the main list.
  - `CartridgeList.csv` containing data related to Game Boy cartridges (e.g., for reference or tooling).
  - `CODE_OF_CONDUCT.md` and `CONTRIBUTING.md` for community participation guidelines.

- **Web version of the list**
  - A separate, more user-friendly web presentation of the same resources: https://gbdev.github.io/resources

- **Community & collaboration**
  - Hosted on GitHub under the `gbdev` organization, allowing issues and pull requests for updates.
  - Discord community available for discussion and collaboration on Game Boy development (link from README).

- **Platform scope clarification**
  - Targets original Game Boy and Game Boy Color.
  - Explicitly distinguishes Game Boy Advance development, which is covered in a separate `awesome-gbadev` project.

## Pricing
- The directory and all its listed resources in the repository are free to access.
