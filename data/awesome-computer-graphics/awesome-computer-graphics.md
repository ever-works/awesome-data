## Overview

Awesome Computer Graphics provides comprehensive resources for learning computer graphics, from fundamentals to advanced rendering techniques and real-time graphics.

## Essential Books

### Physically Based Rendering
THE definitive book for offline rendering (3rd edition), covering theory and implementation of photorealistic rendering.

### Fundamentals of Computer Graphics
Comprehensive introduction by Steve Marschner covering all CG basics.

### Ray Tracing from the Ground Up
Hands-on approach to building a ray tracer by Kevin Suffern.

### Real-Time Rendering
The bible of real-time graphics, covering modern GPU techniques.
