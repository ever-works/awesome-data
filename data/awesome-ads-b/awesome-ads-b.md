**Tags:** awesome-lists, GIS, hardware

## Overview
Awesome ADS-B is a curated, community-maintained directory of tools, projects, Docker images, and other resources focused on ADS-B (Automatic Dependent Surveillance–Broadcast), the technology used for broadcasting aircraft data.

## Features
- Curated list of ADS-B related resources
- Focus on ADS-B tools and software projects
- Collection of Docker images for ADS-B setups
- Includes various related resources (guides, references, and other materials) around ADS-B
- Discoverable via the short URL `adsb.cool`
- Hosted as an open GitHub repository with contributors and stars visible publicly
