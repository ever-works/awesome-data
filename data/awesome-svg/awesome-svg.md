# Awesome SVG

**Website:** https://github.com/willianjusten/awesome-svg  
**Category:** Themed Directories  
**Tags:** awesome-lists, web-development, curated-lists

## Description
Awesome SVG is a curated "awesome list" focused on SVG (Scalable Vector Graphics) resources. It aggregates tools, libraries, and learning materials to help developers and designers work more effectively with SVG on the web.

## Features
- Curated collection of SVG-related resources
- Focus on scalable vector graphics for the web
- Organized into topical subdirectories (via a `topics` folder) for easier navigation
- Includes:
  - Tools for creating, editing, and optimizing SVGs
  - Libraries for working with SVG in web development
  - Learning resources and educational materials about SVG
- Hosted as an open GitHub repository, allowing community contributions (via `contributing.md`)
- Includes configuration for code style and environment (`.editorconfig`, `.gitignore`)
- CI setup via Travis (`.travis.yml` and `ci_script.sh`) to help maintain list quality and formatting

- Open, curated list / directory of SVG resources
