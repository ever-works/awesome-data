# Awesome J2ME

**Category:** Themed Directories  
**Website:** https://github.com/hstsethi/awesome-j2me#readme

## Summary
Awesome J2ME is a curated directory of resources related to Java Platform Micro Edition (J2ME), the Java specification used for older mobile devices, keypad phones, Symbian devices, and PDAs. It aggregates documentation, tools, learning materials, and software relevant to J2ME and MIDP-based MIDlets (`.jad` / `.jar`).

## Features
- **Comprehensive J2ME Focus**
  - Dedicated to Java Platform Micro Edition (J2ME) and related technologies.
  - Covers MIDP (built on CLDC) and MIDlet applications for legacy mobile platforms.

- **Documentation & Learning Resources**
  - Links to general J2ME documentation.
  - Academic articles and papers about J2ME.
  - Tutorials for getting started and working with J2ME.
  - Blog post explaining the introduction, setup process, and history of the Awesome J2ME project.

- **Communities**
  - Section for J2ME-related community links (forums, groups, or community hubs).

- **Development Tools**
  - **IDEs:** Curated list of integrated development environments suitable for J2ME development.
  - **SDKs:** References to Java ME / J2ME SDKs (including versions that support MIDP up to Java ME SDK 3.4).
  - Example `sample-midlet.java` included in the repository for reference.

- **Emulators**
  - Directory of emulators for running and testing J2ME/MIDP applications.

- **Hardware**
  - Resources related to hardware that can run J2ME (e.g., old keypad phones, Symbian devices, PDAs).

- **Native Software Catalog**
  - **Apps:** Curated list of notable J2ME applications.
  - **Video Games:** Collection of J2ME games and related resources.

- **Reverse Engineering & Tools**
  - Section on reverse engineering J2ME software.
  - **Decompilers:** Links to decompilers and similar tools for inspecting MIDlets and J2ME code.

- **Related Projects**
  - References to other relevant curated lists and projects, including related ecosystems like Symbian.

- **Meta & Contribution Information**
  - Clear contents/structure for navigation (Communities, Development, Emulators, Hardware, Native Software, Related Projects, Reverse Engineering, Tutorials, Academic Articles, etc.).
  - `contributing.md` describing how to contribute new links and improvements.
  - Open-source license (LICENSE file in the repository).
