# Awesome WebAssembly

**Description**

Awesome WebAssembly is a curated, community-driven list of WebAssembly (Wasm) resources, projects, and communities. It focuses on the WebAssembly binary instruction format as a portable compilation target for running code efficiently across client and server platforms.

**Website**

- GitHub: https://github.com/idematos/awesome-webassembly#readme

## Features

- **Curated WebAssembly resources**  
  - Organized collection of links and references related to WebAssembly (Wasm).

- **Learning resources**  
  - *Basics*: Introductory materials explaining what WebAssembly is and how it works.  
  - *Articles*: Blog posts, technical write-ups, and deep dives into Wasm concepts and use cases.  
  - *Books*: Longer-form publications and books focused on WebAssembly.  
  - *Videos*: Talks, tutorials, and conference presentations about WebAssembly.

- **Project directories**  
  - *Compilers*: Projects that compile various programming languages to WebAssembly.  
  - *Runtimes*: WebAssembly runtimes and virtual machines for running Wasm modules in different environments.  
  - *Libraries*: Reusable libraries and modules targeting or leveraging WebAssembly.  
  - *Tools*: Developer tools, utilities, and build tooling for working with WebAssembly.  
  - *Frameworks*: Frameworks that integrate or are built around WebAssembly.

- **Community links**  
  - Section dedicated to WebAssembly-related communities (forums, chat groups, organizations, and similar).

- **Open contribution model**  
  - Public contribution guidelines for adding or updating entries.  
  - Encourages community participation to keep the list up to date.

  - Distributed with an explicit LICENSE file in the repository (open-source terms available on GitHub).
