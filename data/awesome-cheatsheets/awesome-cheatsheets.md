## Overview

Awesome Cheatsheets provides single-file references containing everything you should know about programming languages, frameworks, and tools.

## Features

- **Programming Languages**: JavaScript, Python, Go, PHP, and more
- **Frameworks**: React, Vue, Django, Express, and others
- **Databases**: MySQL, PostgreSQL, MongoDB, Redis
- **DevOps Tools**: Docker, Kubernetes, Git, Bash
- **Frontend**: HTML, CSS, Sass, Tailwind CSS
- **Backend**: Node.js, Express, Django, Laravel
- **Tools**: Vim, Tmux, regular expressions
- **All-in-One**: Everything in single, comprehensive files
