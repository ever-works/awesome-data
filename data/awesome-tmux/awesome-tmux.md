## Overview

Awesome Tmux is a collection of resources for tmux, the powerful terminal multiplexer. Tmux allows users to switch between several programs in one terminal, detach and reattach sessions, and manage multiple terminal windows and panes.

## Features

- Official tmux documentation and man pages
- Tmux Plugin Manager (TPM) and popular plugins
- Configuration examples and dotfiles
- Themes and status bar customizations
- Session management tools and automation
- Vim and Neovim integration
- Clipboard management across tmux
- Mouse support and configuration
- Pane and window navigation strategies
- Resurrection and continuum plugins for session persistence
- Copy mode and buffer management
- SSH and remote server workflows
- Tmux with tmuxinator and tmuxp
- Performance optimization tips
- Troubleshooting common issues
- Cheat sheets and quick references
- Video tutorials and screencasts
- Books and long-form guides
