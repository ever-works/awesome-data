## Overview

Awesome Edge Computing is a comprehensive collection of production-ready frameworks, services, and repositories focused on edge computing and IoT. Edge computing brings computation and data storage closer to the sources of data to improve response times and save bandwidth.

## Key Frameworks and Platforms

### Kubernetes-Based Solutions
- **K3s**: Certified lightweight Kubernetes distribution that's production ready, perfect for IoT and edge computing where resource constraints matter
- **KubeEdge**: Open platform for edge computing that provides core infrastructure support for networking, application deployment, and metadata synchronization between cloud and edge
- **microk8s**: Lightweight Kubernetes for developers, edge, and IoT
- **Akri**: Kubernetes Resource Interface for exposing heterogeneous leaf devices as resources in a Kubernetes cluster

### Edge Platforms
- **Baetyl**: Extends cloud computing to edge devices, enabling cloud-edge collaboration
- **Eliot**: Open source system for managing containerized applications in IoT devices
- **Azure IoT Edge**: Microsoft's edge computing platform for deploying cloud workloads to run on IoT devices

### Networking Solutions
- **Cilium**: eBPF-based networking, security, and observability
- **Calico**: Cloud native networking and network security
- **Flannel**: Simple overlay network for Kubernetes

## Features

- Low latency processing by computing data near the source
- Reduced bandwidth usage by processing data locally
- Improved reliability with distributed architecture
- Enhanced privacy and security
- Support for resource-constrained devices
- MQTT protocol support for IoT communication
