---

## Overview

Awesome Steam Deck is a community-maintained directory of tools, guides, and configurations tailored for the Steam Deck. It focuses on improving gaming, system management, and everyday use on Valve’s Linux-based handheld.

The repository is hosted on GitHub and organized as a traditional Awesome list, aggregating links rather than shipping software itself.
