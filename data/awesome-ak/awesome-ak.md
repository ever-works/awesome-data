## Overview
awesome-ak is an "Awesome"-style curated list of notable websites collected from the creatorâ€™s personal bookmarks. It functions as a lightweight web directory and also provides the entire bookmark collection as a downloadable HTML file for import into browsers.

## Features
- **Curated website directory**
  - Collection of notable and useful websites organized as an Awesome list.
  - Focuses on links originally gathered from personal bookmarks.
- **Downloadable bookmark file**
  - Provides an HTML `bookmark.html` file containing the full bookmark set.
  - Direct download link exposed via GitHub Pages: `https://anzenkodo.github.io/awesome/bookmark.html`.
- **Browser import compatibility**
  - Designed to be imported as standard bookmarks into major browsers.
  - Linked instructions for importing HTML bookmarks in:
    - Chromium-based browsers (Chrome, Brave, Edge, etc.)
    - Firefox
    - Safari
- **Open-source repository**
  - Hosted on GitHub under the `AnzenKodo/awesome-ak` repository.
  - Includes source files such as:
    - `README.md` for documentation
    - `bookmark.html` for the bookmark export
    - `run.js` (likely for automation or generation of the list)
    - `LICENSE` (MIT license)
- **Awesome list compliance**
  - Uses the official Awesome badge and format conventions.
- **Social & follow-resource directory**
  - Links to social follow lists maintained by the author:
    - Twitter Lists
    - YouTube subscriptions (channels list)
    - GitHub following
  - Additional sections for community resources such as Discord servers (e.g., language-focused communities like V, Deno, Nix/NixOS).

## Usage
- Browse the curated list directly in the GitHub README.
- Download the `bookmark.html` file and import it into a browser using built-in bookmark import tools.
