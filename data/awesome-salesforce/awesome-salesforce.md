# Awesome Salesforce

**Website:** https://github.com/mailtoharshit/awesome-salesforce#readme  
**Category:** Themed Directories  
**Tags:** saas, crm, awesome-lists

## Overview
Awesome Salesforce is a community-maintained, curated list of libraries, tools, and resources for developers building on the Salesforce platform. It aggregates useful links and components to help with Salesforce app development and related workflows.

## Features
- Curated collection of Salesforce-related resources
- Focus on tools and libraries for Salesforce app development
- Includes various categories of Salesforce platform resources (e.g., code, integrations, utilities)
- GitHub-based list, open to community contributions via pull requests
- Organized README and index files to navigate content
- Issue templates and contribution guidelines for maintainers and contributors

## Type
- Open, curated resource list (not a commercial product or SaaS)

## Pricing
- Free to access and use (open GitHub repository)
