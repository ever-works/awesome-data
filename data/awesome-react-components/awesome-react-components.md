## Overview

Awesome React Components is a comprehensive collection of React components and libraries, carefully curated to maintain high quality by removing non-awesome entries when adding new ones.

## UI Component Libraries

### Design Systems
- **Material-UI (MUI)** - Google's Material Design in React
- **Ant Design** - Enterprise-class UI design system
- **Chakra UI** - Accessible component library
- **Mantine** - Full-featured React components
- **Radix UI** - Unstyled, accessible primitives
- **shadcn/ui** - Re-usable components built with Radix

### Utility-First
- **Tailwind UI** - Premium components for Tailwind CSS
- **daisyUI** - Component library for Tailwind
- **Headless UI** - Unstyled components from Tailwind Labs

## Data Display

### Tables & Grids
- **TanStack Table** - Headless table library
- **AG Grid** - Feature-rich data grid
- **React Data Grid** - Excel-like grid component
- **Material React Table** - Built on TanStack Table

### Charts & Visualization
- **Recharts** - Composable charting library
- **Victory** - Modular charting components
- **Nivo** - Rich data visualization components
- **Chart.js with React** - react-chartjs-2

## Form Libraries

- **React Hook Form** - Performant forms with validation
- **Formik** - Build forms in React
- **React Final Form** - Subscription-based form state
- **TanStack Form** - Headless form management

## State Management

- **Zustand** - Small, fast state management
- **Jotai** - Primitive and flexible state
- **Redux Toolkit** - Official Redux toolset
- **MobX** - Simple, scalable state management
- **Recoil** - State management from Facebook

## UI Utilities

### Animation
- **Framer Motion** - Production-ready animations
- **React Spring** - Spring-physics animations
- **React Transition Group** - Component transitions

### Drag & Drop
- **dnd kit** - Modern drag and drop toolkit
- **React DnD** - Drag and drop for React
- **React Beautiful DnD** - Beautiful drag and drop

## Specialized Components

### Date & Time
- **React DatePicker** - Date picker component
- **React Day Picker** - Flexible date picker
- **Date-fns** with React components

### File Upload
- **React Dropzone** - Drag and drop file uploads
- **Uppy** - Modular file uploader

### Rich Text Editors
- **Slate** - Customizable framework for rich text
- **Draft.js** - Rich text editor framework
- **TipTap** - Headless editor framework
- **Lexical** - Extensible text editor from Facebook

## Notifications & Modals

- **React Hot Toast** - Lightweight notifications
- **React Toastify** - Notification library
- **Radix Dialog** - Accessible modal dialogs
