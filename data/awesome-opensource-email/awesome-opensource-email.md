## Overview

Awesome Opensource Email is a curated list of open-source email resources, servers, and tools for self-hosted email infrastructure and marketing.

## SMTP Servers & Mail Transfer Agents

- **KumoMTA** - High-performance MTA
- **Haraka** - Modern, high-performance, pluggable SMTP server
- **Zone-MTA** - Modern outbound SMTP relay
- **Postal** - Complete mail server for websites & web servers
- **Maddy** - Composable all-in-one mail server
- **Chasquid** - SMTP server written in Go

## Newsletter & Marketing Platforms

- **listmonk** - High-performance, self-hosted newsletter and mailing list manager
- **Mailtrain** - Self-hosted newsletter application built on Nodemailer
- **SendPortal** - Open-source self-hosted email marketing solution
- **Mautic** - Open-source marketing automation platform

## Development & Testing Tools

- **SMTP4dev** - Fake SMTP server for development
- **Maildev** - SMTP server for testing and development
- **MailHog** - Email testing tool for developers
- **MailTrap** - Fake SMTP server to test emails

## SMTP Client & Server Libraries

- Go SMTP libraries
- Python email libraries
- Node.js email packages
- PHP mailer libraries

## Features to Consider

- Deliverability and reputation management
- Bounce and complaint handling
- DKIM, SPF, and DMARC support
- Queue management
- Webhooks and APIs
- Analytics and reporting

## Self-Hosting Benefits

- Complete control over data
- No monthly fees
- Privacy and compliance
- Customization freedom
- No sending limits (beyond your infrastructure)

## Use Cases

- Transactional emails
- Marketing campaigns
- Newsletter delivery
- Automated notifications
- Internal communications
