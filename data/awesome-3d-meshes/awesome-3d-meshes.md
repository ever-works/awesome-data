## Overview

A curated collection of famous 3D meshes used in computer graphics research, testing, and applications. Includes classic test meshes, production-quality models, and mesh processing tools.

## Features

- Classic test meshes (Stanford Bunny, Utah Teapot, Dragon, etc.)
- Production-quality 3D models
- Mesh processing libraries and tools
- Mesh simplification algorithms
- Mesh generation techniques
- File format converters

## Use Cases

- Computer graphics research
- Algorithm testing and benchmarking
- 3D printing preparation
- Game development
- Visual effects production
- Academic projects

## Common Mesh Formats

- OBJ - Widely supported text-based format
- STL - Common for 3D printing
- PLY - Polygon file format with vertex colors
- FBX - Industry standard for animation
- GLTF - Modern web-friendly format

## Related Tools

- MeshLab - Open source mesh processing system
- Blender - Complete 3D creation suite
- CloudCompare - 3D point cloud and mesh processing
